# <a name="credit-check-status-request"></a>Αίτηση κατάστασης του πιστωτικού ελέγχου

Η παραγγελία σας είναι σε εκκρεμότητα πιστωτικό επαλήθευσης. Αυτό ολοκληρώνεται συνήθως σε 24 ώρες, αλλά μπορεί να διαρκέσει έως και 48 ώρες. Όταν ολοκληρωθεί η εγκατάσταση, την κατάσταση της συνδρομής σας θα αλλάξει από σε εκκρεμότητα σε ενεργή και υπηρεσίες θα είναι διαθέσιμες για χρήση.
