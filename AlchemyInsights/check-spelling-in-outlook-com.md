# <a name="use-your-browser-to-check-spelling"></a>Χρησιμοποιήστε το πρόγραμμα περιήγησης για να κάνετε ορθογραφικό έλεγχο

Υπάρχει δυνατότητα ορθογραφικού ελέγχου σε Outlook.com. Για να ελέγξετε την ορθογραφία, πρέπει να χρησιμοποιήσετε το πρόγραμμα περιήγησης web. Ορθογραφικός έλεγχος είναι διαθέσιμη στο Microsoft άκρη, Internet Explorer 10 και νεότερες εκδόσεις και τρέχουσες εκδόσεις του Firefox, Chrome και Safari. Ελέγξτε τις επιλογές για το πρόγραμμα περιήγησης web για να μάθετε περισσότερα σχετικά με τον έλεγχο ορθογραφίας.
