# <a name="change-how-often-you-pay"></a>Αλλαγή της συχνότητας πληρωμής

 **Θέλετε να αλλάξετε τη συχνότητα πληρωμής σας;** Αν θέλετε να αλλάξετε τη συχνότητα πληρωμής για τη συνδρομή microsoft 365 για επαγγελματική συνδρομή, είτε από ετήσια σε μηνιαία ή μηνιαία σε ετήσια, μπορούμε να σας βοηθήσουμε με αυτό. Θα πρέπει να αγοράσετε την ίδια συνδρομή με τη νέα συχνότητα πληρωμής και να ακυρώσετε την παλιά συνδρομή. Παρακαλούμε χρησιμοποιήστε την επιλογή "Ας σας καλέσουμε" για να μιλήσετε με έναν μηχανικό υποστήριξης.
