# <a name="microsoft-bookings-app"></a>Εφαρμογή Κρατήσεις της Microsoft

Η Microsoft Bookings είναι μια εφαρμογή προγραμματισμού συναντήσεων που σας βοηθά να παρακολουθείτε τις κρατήσεις σας, το προσωπικό σας και τους πελάτες σας. Για πληροφορίες ή αντιμετώπιση προβλημάτων στην εφαρμογή Κρατήσεις, ανατρέξτε στα [άρθρα κρατήσεων](https://support.office.com/article/b9c9295c-c654-4b10-b5cc-f739825fc092).
