# <a name="cant-preview-all-of-an-attachment"></a>Δεν είναι δυνατό να κάνετε προεπισκόπηση όλων των συνημμένο

Είστε ενήμεροι για ένα ζήτημα όπου οι ορισμένοι πελάτες δεν είναι δυνατό να κάνετε προεπισκόπηση όλων των συνημμένο. Ως εναλλακτικός τρόπος αντιμετώπισης, ανοίξτε το συνημμένο αρχείο σε ένα νέο παράθυρο.
