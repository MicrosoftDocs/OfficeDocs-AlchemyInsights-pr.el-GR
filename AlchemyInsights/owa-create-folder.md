# <a name="create-a-folder"></a>Δημιουργία φακέλου

1. Στο κάτω μέρος του παραθύρου φακέλων του Outlook στο Web, επιλέξτε **Δημιουργία φακέλου**. Ίσως χρειαστεί να κάνετε κύλιση προς τα κάτω στο παράθυρο φακέλων για να δείτε το κουμπί " **Νέος φάκελος** ".
1. Στο πλαίσιο κειμένου, πληκτρολογήστε ένα όνομα για το φάκελο και, στη συνέχεια, πατήστε το πλήκτρο ENTER.

Για να μάθετε περισσότερα, ανατρέξτε στο [θέμα εργασία με φακέλους μηνυμάτων στο Outlook στο Web](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
