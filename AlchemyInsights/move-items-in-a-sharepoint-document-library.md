    <span data-ttu-id="20be2-108">Για να δημιουργήσετε ένα νέο φάκελο για να μετακινήσετε τα αρχεία, επιλέξτε μια θέση στην ιεραρχία φακέλων, κάντε κλικ στην επιλογή **Νέος φάκελος**, πληκτρολογήστε ένα όνομα για το φάκελο και κάντε κλικ στο σημάδι ελέγχου για να αποθηκεύσετε το όνομα.</span><span class="sxs-lookup"><span data-stu-id="20be2-108">To create a new folder to move the files to, select a location in the folder hierarchy, click **New folder**, enter a name for the folder, and click the check mark to save the name.</span></span>

 <span data-ttu-id="20be2-110">Όταν ένα αρχείο μετακινείται, θα εξακολουθεί να εμφανίζεται στον κατάλογο προέλευσης μέχρι να μετακινηθεί πλήρως στον προορισμό και, στη συνέχεια, θα διαγραφεί.</span><span class="sxs-lookup"><span data-stu-id="20be2-110">When a file is moving, it will still appear in the source directory until its fully moved to the destination, and then it will be deleted.</span></span> <span data-ttu-id="20be2-111">Το αρχείο θα παραμείνει στον κάδο ανακύκλωσης τοποθεσιών προέλευσης μετά την ολοκλήρωση της μετακίνησης και θα υπόκειται στο κανονικό χρονοδιάγραμμα ανακύκλωσης, εκτός εάν ένας χρήστης το ανακτήσει από τον κάδο ανακύκλωσης.</span><span class="sxs-lookup"><span data-stu-id="20be2-111">The file will remain in the source sites recycle bin after the move is complete and be subject to the normal recycle schedule unless a user recovers it from the recycle bin.</span></span>
