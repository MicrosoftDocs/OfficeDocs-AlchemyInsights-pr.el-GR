# <a name="how-to-change-your-theme"></a>Πώς μπορείτε να αλλάξετε το θέμα

Για να αλλάξετε το θέμα στο Outlook.com ή το Outlook στο web, μεταβείτε στις **Ρυθμίσεις** και επιλέξτε ένα θέμα στο πλαίσιο **θέμα**. Για να δείτε όλα τα διαθέσιμα θέματα, επιλέξτε **Προβολή όλων** στο πλαίσιο **θέμα**.
