# <a name="cant-add-default-2010-approval-workflow"></a>Δεν είναι δυνατή η προσθήκη προεπιλεγμένης ροής εργασίας έγκρισης 2010

Σε μια συλλογή τοποθεσιών του Microsoft SharePoint, δεν μπορείτε να προσθέσετε μια καθολικά επαναχρησιμοποιήσιμη ροή εργασίας (όπως "Έγκριση - SharePoint 2010") σε μια λίστα ή βιβλιοθήκη. Η ροή εργασίας έγκρισης ενδέχεται να λείπει. Για να επιλύσετε αυτό το ζήτημα, ανατρέξτε στο θέμα [Δεν είναι δυνατή η προσθήκη μιας καθολικά επαναχρησιμοποιήσιμης ροής εργασίας σε μια λίστα ή βιβλιοθήκη του SharePoint](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Για περισσότερες πληροφορίες σχετικά με τις εγκρίσεις ροής εργασίας, ανατρέξτε στο θέμα [Όλες οι ροές εργασίας έγκρισης](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Μάθετε περισσότερα σχετικά με [τις σύγχρονες εγκρίσεις στη ροή](https://flow.microsoft.com/blog/introducing-modern-approvals).
