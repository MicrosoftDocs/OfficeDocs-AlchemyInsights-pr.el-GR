# <a name="turn-off-focused-inbox"></a>Απενεργοποιήστε την επιλογή "Εισερχόμενα" εστιάζει

1. Επιλέξτε τις **Ρυθμίσεις** \> **εστιάζει στο φάκελο "Εισερχόμενα"**.  
    
2. Στην περιοχή **κατά τη λήψη ηλεκτρονικού ταχυδρομείου**, επιλέξτε **δεν ταξινόμηση μηνυμάτων**.
    
3. Επιλέξτε " **ΟΚ"**.
    
Για να μάθετε περισσότερα, ανατρέξτε στην ενότητα [Εστιάζει στο φάκελο "Εισερχόμενα" για το Outlook](https://go.microsoft.com/fwlink/p/?linkid=873108).
