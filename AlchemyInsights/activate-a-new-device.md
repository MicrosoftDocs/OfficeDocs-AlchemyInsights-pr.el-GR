# <a name="activating-a-new-device-running-windows-10"></a>Ενεργοποίηση νέας συσκευής που εκτελεί Windows 10

Εάν αγοράσατε μια νέα συσκευή που λειτουργεί με Windows 10, θα ενεργοποιηθεί αυτόματα όταν συνδέσετε τη συσκευή σας στο Internet. Εάν οι διακομιστές ενεργοποίησης είναι απασχολημένοι, ίσως χρειαστεί να περιμένετε λίγο για να ενεργοποιηθούν τα Windows.

Για να επιλέξετε την κατάσταση ενεργοποίησης, επιλέξτε **Έναρξη** **Settings**  >  **ενημέρωσης ρυθμίσεων &**  >  **Ενεργοποίηση**ασφαλείας.

**Σημείωση:** Σε ορισμένες περιπτώσεις, ίσως χρειαστεί να εισαγάγετε τον αριθμό-κλειδί προϊόντος των Windows 10 που συνόδευε τη συσκευασία της συσκευής ή να τον συμπεριλάβετε ως κάρτα ή στο πιστοποιητικό αυθεντικότητας (COA) που είναι προσαρτημένο στη συσκευή.

[Μάθετε περισσότερα σχετικά με την ενεργοποίηση στα Windows 10](https://support.microsoft.com/help/12440)
