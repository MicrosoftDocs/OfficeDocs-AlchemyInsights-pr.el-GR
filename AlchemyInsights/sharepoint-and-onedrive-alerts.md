# <a name="not-receiving-sharepoint-and-onedrive-alerts"></a>Δεν λαμβάνετε ειδοποιήσεις του SharePoint και του OneDrive

Πρώτα ελέγξτε το φάκελο ανεπιθύμητη αλληλογραφία ή spam στο email σας.

Στη συνέχεια, προσδιορίστε εάν **όλες οι προειδοποιήσεις δεν παραδίδονται** ή αν δεν παραδοθεί **μια μεμονωμένη ειδοποίηση** από ένα συγκεκριμένο αρχείο ή βιβλιοθήκη.

- Εάν **δεν παραδίδονται όλες οι ειδοποιήσεις από πολλά αρχεία ή βιβλιοθήκες**, επισκεφθείτε τον [πίνακα εργαλείων εύρυθμης λειτουργίας υπηρεσίας](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fadmin.microsoft.com%2FAdminPortal%2FHome%23%2Fservicehealth&data=02%7C01%7Cv-todmc%40microsoft.com%7C2cd2037aa7304711d2bc08d741fae254%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637050418099632638&sdata=35FUOTleK0Sc0z%2B7N7Vm0tOgXplyeOe3LcIzqRziGXc%3D&reserved=0) για να ελέγξετε για τυχόν προειδοποιήσεις/συμβάντα που ενδέχεται να συμβαίνουν με το SharePoint ή το Exchange. Το ζήτημα μπορεί να είναι με τη δυνατότητα προειδοποίησης του SharePoint ή καθυστερήσεις στα μηνύματα ηλεκτρονικού ταχυδρομείου μέσω του Exchange. Σημειώστε επίσης εάν παραδίδεται άλλο μήνυμα ηλεκτρονικού ταχυδρομείου — εάν όχι, το ζήτημα είναι πιθανό με καθυστερήσεις του Exchange.
- Εάν **δεν παραδοθεί μια μεμονωμένη ειδοποίηση από ένα συγκεκριμένο αρχείο ή βιβλιοθήκη**, προσπαθήστε να τη διαγράψετε και να την δημιουργήσετε ξανά. Δείτε [Διαχείριση, προβολή ή διαγραφή ειδοποιήσεων του SharePoint](https://nam06.safelinks.protection.outlook.com/?url=https%3A%2F%2Fsupport.office.com%2Farticle%2Fmanage-view-or-delete-sharepoint-alerts-99dfb19c-9a90-4a8c-aba1-aa8c8afb0de2%3Fui%3Den-US%26rs%3D%26ad%3DUS%23ID0EAADAAA%3DOnline&data=02%7C01%7Cv-todmc%40microsoft.com%7C2cd2037aa7304711d2bc08d741fae254%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C637050418099632638&sdata=AkE%2BjiG6%2BA59llp2DGcg4uHHUjaUDUnAlK5ax%2Fepn3E%3D&reserved=0) για να δημιουργήσετε ξανά την ειδοποίηση.

> [!NOTE]
> - Δεν είναι δυνατή η αποστολή ειδοποιήσεων σε μια ομάδα διανομής. Υποστηρίζονται μόνο ομάδες ασφαλείας και O365.
> - Δεν μπορείτε να προσαρμόσετε τα πρότυπα ηλεκτρονικού ταχυδρομείου ειδοποιήσεων. Πρέπει να χρησιμοποιήσετε το Microsoft Flow ή τη ροή εργασίας του SharePoint Designer για να τα επιτύχετε.
