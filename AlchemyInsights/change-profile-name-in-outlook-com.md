# <a name="change-your-profile-name"></a>Αλλαγή του ονόματος του προφίλ σας

1. Μεταβείτε στις [πληροφορίες του λογαριασμού σας Microsoft](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Επιλέξτε **Επεξεργασία ονόματος**. 
    
3. Πληκτρολογήστε όνομα και επώνυμο και επιλέξτε **Αποθήκευση**. 
    
Παρακαλούμε επιτρέψτε έως και 24 ώρες για να εφαρμοστούν οι αλλαγές στο όνομα του προφίλ σας.
  
Για να μάθετε περισσότερα, ανατρέξτε [στο θέμα Αλλαγή της φωτογραφίας ή του ονόματος προφίλ σας στο Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110).
