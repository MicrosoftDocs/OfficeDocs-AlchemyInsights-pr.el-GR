# <a name="delete-a-folder"></a>Διαγραφή φακέλου

1. Στο αριστερό τμήμα του παραθύρου, κάντε δεξιό κλικ στο φάκελο ή υποφάκελο που θέλετε να διαγράψετε και επιλέξτε **Διαγραφή φακέλου**.
2. Επιλέξτε **OK** για να επιβεβαιώσετε.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [εργασία με φακέλους μηνυμάτων στο Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
