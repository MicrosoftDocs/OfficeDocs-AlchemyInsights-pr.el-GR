# <a name="expired-subscription"></a>Έχει λήξει η συνδρομή

Φαίνεται ότι έχετε μία ή περισσότερες εγγραφές έχουν λήξει. Εάν θέλετε να συνεχίσετε την υπηρεσία, πρέπει να ενεργοποιήστε ξανά τη συνδρομή ή να πραγματοποιήσετε μια νέα αγορά.
  
**Επανενεργοποίηση**
  
1. Στο Κέντρο διαχείρισης, μεταβείτε στην **χρέωσης** \> σελίδα [υπηρεσιών & προϊόντα](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Βρείτε την εγγραφή που θέλετε να επανενεργοποιήσετε. Στην ενότητα **χρέωσης** , επιλέξτε **Επανενεργοποίηση**.

Εάν δεν βλέπετε **Επανενεργοποίηση** ως μια διαθέσιμη ενέργεια, καλέστε την υποστήριξη για να ενεργοποιήσετε ξανά τη συνδρομή σας.

3. Επιβεβαιώστε τις λεπτομέρειες της πληρωμής σας. Μπορείτε να ενημερώσετε τις υπάρχουσες πληροφορίες πληρωμής εδώ. Όταν ολοκληρώσετε την εισαγωγή της μεθόδου πληρωμής σας, επιλέξτε να **ενεργοποιήσετε ξανά**.

Για περισσότερες πληροφορίες, δείτε την [Επανενεργοποίηση της συνδρομής σας](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/reactivate-your-subscription).

**Δημιουργία μιας νέας αγοράς**
  
1. Στο Κέντρο διαχείρισης, μεταβείτε στην **χρέωσης** \> σελίδα [υπηρεσιών αγοράς](https://go.microsoft.com/fwlink/p/?linkid=868433) .

2. Επιλέξτε την εγγραφή που θέλετε να αγοράσετε και, στη συνέχεια, επιλέξτε **αγορά**.

3. Εισαγάγετε τον αριθμό των αδειών χρήσης που χρειάζεστε και επιλέξτε αν θέλετε να καταβάλλουν κάθε μήνα ή για ολόκληρο το έτος. Επιλέξτε εάν θέλετε για την αυτόματη εκχώρηση αδειών χρήσης σε όλους τους χρήστες που δεν έχουν αυτήν τη στιγμή μια άδεια χρήσης. Στη συνέχεια, κάνετε **ανάληψη ελέγχου τώρα**.

4. Ακολουθήστε τα υπόλοιπα βήματα για να ολοκληρώσετε την ανάληψη ελέγχου.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [αγοράσετε μια συνδρομή](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/buy-another-subscription).
