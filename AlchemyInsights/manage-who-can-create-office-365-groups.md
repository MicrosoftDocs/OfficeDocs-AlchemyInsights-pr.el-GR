# <a name="manage-who-can-create-microsoft-365-groups"></a>Διαχείριση των χρηστών που μπορούν να δημιουργήσουν ομάδες του Microsoft 365

Από προεπιλογή, όλοι οι χρήστες του οργανισμού σας μπορούν να δημιουργήσουν ομάδες O365.
  
Εάν θέλετε να αλλάξετε αυτό το πρόβλημα, μπορείτε να δημιουργήσετε μια ομάδα ασφαλείας με το PowerShell που περιορίζει τη δημιουργία ομάδας σε ένα υποσύνολο χρηστών.
  
Ακολουθήστε τις οδηγίες βήμα προς βήμα εδώ: διαχείριση των [χρηστών που μπορούν να δημιουργήσουν ομάδες του Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
