# <a name="become-an-admin"></a>Γίνετε διαχειριστής

Για να αναλάβετε το ρόλο διαχειριστή για τον οργανισμό σας, θα πρέπει να αναλάβετε τον μισθωτή στο Office 365.
  
Ακολουθήστε τις οδηγίες σε αυτόν τον οδηγό: [εξαγορά διαχειριστή](https://docs.microsoft.com/azure/active-directory/users-groups-roles/domains-admin-takeover#azure-ad-powershell-cmdlets-for-the-forcetakeover-option)
