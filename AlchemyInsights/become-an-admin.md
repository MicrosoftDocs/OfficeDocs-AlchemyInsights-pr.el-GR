# <a name="become-an-admin"></a>Γίνετε διαχειριστής

Για να αναλάβετε το ρόλο διαχειριστή για τον οργανισμό σας, θα πρέπει να αναλάβετε τον μισθωτή στο Office 365.
  
Ακολουθήστε τις οδηγίες σε αυτόν τον οδηγό: [Κάντε μια εξαγορά διαχειριστή στο Office 365](https://docs.microsoft.com/office365/admin/misc/become-the-admin)
