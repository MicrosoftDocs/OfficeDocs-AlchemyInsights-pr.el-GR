# <a name="permissions-in-the-sharepoint-modern-experience"></a>Δικαιώματα στη σύγχρονη εμπειρία του SharePoint

Παραδοσιακά, τα δικαιώματα του SharePoint έχουν ρυθμιστεί μέσω ενός συνόλου ομάδων δικαιωμάτων εντός μιας τοποθεσίας (κάτοχοι, μέλη, επισκέπτες, κ. λπ.). Στο SharePoint στο Microsoft 365, αυτό παραμένει αληθές για ορισμένους τύπους τοποθεσιών, αλλά είναι διαθέσιμες πρόσθετες επιλογές.  

Για περισσότερες πληροφορίες, μπορείτε να επισκεφθείτε, να κάνετε [κοινή χρήση και δικαιώματα στη σύγχρονη εμπειρία του SharePoint](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions).
