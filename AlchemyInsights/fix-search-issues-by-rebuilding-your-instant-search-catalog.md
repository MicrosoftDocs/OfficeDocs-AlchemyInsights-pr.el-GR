# <a name="troubleshoot-search-issues"></a>Αντιμετώπιση προβλημάτων αναζήτησης

Αυτά είναι τα πιο συνηθισμένα ζητήματα, και είναι ένα καλό σημείο για να ξεκινήσετε:

- [Αναζήτηση του Outlook επιστρέφει κανένα αποτέλεσμα](https://support.office.com/article/2556b11f-f4d8-46be-b0a7-de33a3f4f066#bkmk_noresults)

Εδώ, μπορείτε να βρείτε πιθανές λύσεις για λιγότερο συνηθισμένα προβλήματα:

- [Μόνο το Outlook 2016: γκρίζο χρώμα καρτέλα "Αναζήτηση"](https://support.office.com/article/2556b11f-f4d8-46be-b0a7-de33a3f4f066#bkmk_greytab)
