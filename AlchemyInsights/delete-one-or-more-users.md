# <a name="delete-one-or-more-users"></a>Διαγραφή ενός ή περισσότερων χρηστών

Για να διαγράψετε χρήστες και να ανακτήσετε ή να διακόψετε την πληρωμή για άδειες χρήσης, διαβάστε το ακόλουθο άρθρο υποστήριξης: [Διαγραφή χρήστη από τον οργανισμό σας](https://docs.microsoft.com/microsoft-365/admin/add-users/delete-a-user?view=o365-worldwide).

Αυτό το άρθρο καλύπτει:

- Ζητήματα κατά τη διαγραφή χρηστών
- Διαγραφή ενός ή περισσότερων χρηστών
- Πώς να σταματήσετε να πληρώνετε για άδειες χρήσης

Για πρόσθετες πληροφορίες σχετικά με την ανεκμετάλνηξη εκχώρησης αδειών χρήσης από χρήστες, εξετάστε το ακόλουθο άρθρο υποστήριξης: [Ανεκχωρίστε άδειες χρήσης από τους χρήστες](https://docs.microsoft.com/microsoft-365/admin/manage/remove-licenses-from-users?view=o365-worldwide).
