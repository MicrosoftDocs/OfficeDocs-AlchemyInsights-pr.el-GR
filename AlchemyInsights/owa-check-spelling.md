# <a name="use-your-browser-to-check-spelling"></a>Χρήση του προγράμματος περιήγησής σας για ορθογραφικό έλεγχο

Δεν υπάρχει επιλογή ορθογραφικού ελέγχου στο Outlook στο Web. Για να κάνετε ορθογραφικό έλεγχο, θα πρέπει να χρησιμοποιήσετε το πρόγραμμα περιήγησής σας στο Web. Ο ορθογραφικός έλεγχος είναι διαθέσιμος στο Microsoft Edge, τον Internet Explorer 10 και σε νεότερες εκδόσεις και τις τρέχουσες εκδόσεις του Firefox, του Chrome και του Safari. Επιλέξτε τις επιλογές για το πρόγραμμα περιήγησης Web για να μάθετε περισσότερα σχετικά με το πώς μπορείτε να κάνετε ορθογραφικό έλεγχο.
