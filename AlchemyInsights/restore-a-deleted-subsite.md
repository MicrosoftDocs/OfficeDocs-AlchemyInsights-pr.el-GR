# <a name="restore-a-deleted-subsite"></a>Επαναφορά διαγραμμένων δευτερεύουσας τοποθεσίας

Αποστέλλονται Διαγραφή δευτερευουσών τοποθεσιών στη συλλογή τοποθεσιών του Κάδου Ανακύκλωσης, όπου είστε διατηρούνται για 93 ημέρες. Για να επαναφέρετε μια διαγραμμένη δευτερεύουσα τοποθεσία:
  
1. Στο νέο κέντρο διαχείρισης του SharePoint, εντοπίστε την τοποθεσία από την οποία διαγράφηκε στη δευτερεύουσα τοποθεσία και βεβαιωθείτε ότι είστε ένα διαχειριστή. συλλογής τοποθεσιών 
    
2. Μεταβείτε στην τοποθεσία. Στο αριστερό τμήμα του παραθύρου, κάντε κλικ στην επιλογή **Κάδος Ανακύκλωσης** . (Εάν δεν βλέπετε τον Κάδο Ανακύκλωσης, κάντε κλικ στο εικονίδιο ρυθμίσεις και, στη συνέχεια, κάντε κλικ στην επιλογή **περιεχόμενα τοποθεσίας**. Ο Κάδος Ανακύκλωσης είναι στη δεξιά άκρη της γραμμής εντολών στο επάνω μέρος.)
    
3. Στο κάτω μέρος της σελίδας, κάντε κλικ στην επιλογή **Κάδος Ανακύκλωσης δεύτερης φάσης**.
    
4. Κάντε κλικ στα αριστερά της δευτερεύουσας τοποθεσίας και, στη συνέχεια, κάντε κλικ στο κουμπί **Επαναφορά**.
