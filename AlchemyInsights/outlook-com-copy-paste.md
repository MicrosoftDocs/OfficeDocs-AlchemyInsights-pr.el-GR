# <a name="copy-and-paste-in-outlookcom"></a>Αντιγραφή και επικόλληση σε Outlook.com

Δεν υπάρχουν κουμπιά αντιγραφής ή επικόλλησης στο Outlook.com, αλλά μπορείτε να χρησιμοποιήσετε συντομεύσεις πληκτρολογίου:

**Σε υπολογιστή:** CTRL+C για αντιγραφή, CTRL+X προς αποκοπή και CTRL+V για επικόλληση.

**Σε Mac:** CMD + C για να αντιγράψετε, CMD + X για να κόψει, και CMD + V για να επικολλήσετε.
