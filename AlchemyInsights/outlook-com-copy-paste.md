# <a name="copy-and-paste-in-outlookcom"></a>Αντιγράψτε και επικολλήστε στο Outlook.com

Δεν υπάρχουν κουμπιά αντιγραφή ή επικόλληση σε Outlook.com, αλλά μπορείτε να χρησιμοποιήσετε συντομεύσεις πληκτρολογίου:

**Σε PC:** CTRL + C για να αντιγράψετε, CTRL + X για αποκοπή και CTRL + V για να επικολλήσετε.

**Σε έναν υπολογιστή Mac:** CMD + C για να αντιγράψετε, CMD + X για αποκοπή και CMD + V για να επικολλήσετε.
