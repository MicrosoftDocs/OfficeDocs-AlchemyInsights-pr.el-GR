# <a name="question-about-licenses"></a>Ερώτηση σχετικά με άδειες χρήσης

Εάν θέλετε να γνωρίζετε τον αριθμό αδειών χρήσης που έχετε αγοράσει ή πόσες άδειες που έχουν ανατεθεί από μια εγγραφή:
  
1. Στο Κέντρο διαχείρισης, μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**.
    
2. Επιλέξτε την εγγραφή που θέλετε να μάθετε περισσότερα σχετικά με.
    
3. Στην αριστερή πλευρά της κάρτας συνδρομή είναι πληροφορίες σχετικά με τις άδειες χρήσης σας:
    
  - Ο αριθμός **αγοράστηκε** είναι ο αριθμός των αδειών που πληρώνετε για. 
    
  - Ο αριθμός **ανάθεση** είναι ο αριθμός των αδειών που εκχωρούνται σε ένα χρήστη. 
    
4. Εάν θέλετε να προσθέσετε περισσότερες άδειες χρήσης ή εάν έχετε πάρα πολλά, επιλέξτε **Add/Remove άδειες χρήσης** για να προσαρμόσετε το ποσό που έχει αγοραστεί.
