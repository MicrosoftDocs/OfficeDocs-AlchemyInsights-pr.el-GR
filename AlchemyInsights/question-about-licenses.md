# <a name="questions-about-your-microsoft-365-license"></a>Ερωτήσεις σχετικά με την άδεια χρήσης του Microsoft 365

Εάν θέλετε να μάθετε τον αριθμό των αδειών χρήσης που έχετε αγοράσει ή τον αριθμό των αδειών χρήσης που έχουν εκχωρηθεί από μια συνδρομή:
  
1. Στο κέντρο διαχείρισης, μεταβείτε στη σελίδα **χρέωση** των \> **[προϊόντων σας](https://go.microsoft.com/fwlink/p/?linkid=842054)** .

2. Εντοπίστε τη συνδρομή για την οποία θέλετε να μάθετε περισσότερα. Μπορείτε να δείτε τον αριθμό των αδειών χρήσης για τις οποίες πληρώνετε και πόσα είναι διαθέσιμα για ανάθεση.

Εάν θέλετε να προσθέσετε περισσότερες άδειες χρήσης ή εάν έχετε πάρα πολλές, επιλέξτε **Προσθήκη/κατάργηση αδειών χρήσης** για να προσαρμόσετε το ποσό που αγοράσατε.
