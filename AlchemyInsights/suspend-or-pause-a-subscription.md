# <a name="suspend-or-pause-a-subscription"></a>Αναστολή ή παύση μιας συνδρομής

Για να αναστείλετε ή να διακόψετε προσωρινά τη συνδρομή σας προσωρινά, μπορείτε να ακυρώσετε τη συνδρομή από το **Κέντρο διαχείρισης του Microsoft 365**.

1. Μεταβείτε στις **χρεώσεις > [τα προϊόντα σας](https://go.microsoft.com/fwlink/p/?linkid=842054)**.

2. Εντοπίστε τη συνδρομή που θέλετε να ακυρώσετε και, στην περιοχή **ρυθμίσεις & ενέργειες**, κάντε κλικ στην επιλογή **Ακύρωση συνδρομής**.

3. Η συνδρομή σας θα εμφανίζεται τώρα σε κατάσταση απενεργοποίησης για 90 ημέρες και θα έχει μειωμένη λειτουργικότητα μέχρι να διαγραφεί ή να [επανενεργοποιηθεί](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/reactivate-your-subscription?view=o365-worldwide).

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Τι συμβαίνει όταν ακυρώνετε μια συνδρομή](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription?view=o365-worldwide#what-happens-when-you-cancel-a-subscription) και [αποθηκεύετε τα δεδομένα σας](https://go.microsoft.com/fwlink/p/?linkid=842054).
