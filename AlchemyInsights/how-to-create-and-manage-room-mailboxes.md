# <a name="how-to-create-and-manage-room-mailboxes"></a>Τρόπος δημιουργίας και διαχείρισης γραμματοκιβωτίων αίθουσας

Για να δημιουργήσετε και να τροποποιήσετε γραμματοκιβώτια δωματίων, ανατρέξτε στο θέμα [Δημιουργία και διαχείριση γραμματοκιβωτίων αίθουσας](https://technet.microsoft.com/library/jj215781.aspx).

Για να μάθετε περισσότερα σχετικά με τα γραμματοκιβώτια δωματίου και εξοπλισμού (γνωστά και ως γραμματοκιβώτια πόρων), ανατρέξτε στο θέμα [Γραμματοκιβώτια δωματίων και εξοπλισμού](https://docs.microsoft.com/microsoft-365/admin/manage/room-and-equipment-mailboxes).
