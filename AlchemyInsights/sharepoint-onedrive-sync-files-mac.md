# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Εγκατάσταση, το πρόγραμμα εγκατάστασης, και να συγχρονίσετε αρχεία OneDrive ή SharePoint σε Mac 

Εγκατάσταση, το πρόγραμμα εγκατάστασης και συγχρονισμού OneDrive ή SharePoint αρχεία σε έναν υπολογιστή Mac με τις οδηγίες από [αρχεία συγχρονισμού με τον υπολογιστή-πελάτη συγχρονισμού του OneDrive σε Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Εάν είναι εγκατεστημένη η εφαρμογή OneDrive και εγκατάσταση σε Mac, αλλά οι χρήστες αντιμετωπίζουν ζητήματα συγχρονισμού, ανατρέξτε στο θέμα [Συγχρονισμός επιδιόρθωση ζητημάτων με OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US).

> [!NOTE]
> Η δυνατότητα γνωστή μετακίνηση φακέλου στην συγχρονισμού OneDrive για Windows δεν είναι διαθέσιμη για Mac.
