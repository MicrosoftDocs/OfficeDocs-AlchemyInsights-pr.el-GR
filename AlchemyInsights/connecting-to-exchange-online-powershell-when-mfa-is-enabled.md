# <a name="connect-to-exchange-online-powershell-when-mfa-is-enabled"></a>Σύνδεση στο Exchange Online PowerShell όταν είναι ενεργοποιημένη η ΣΠΙ

Εάν ο λογαριασμός σας έχει ενεργοποιημένο τον έλεγχο ταυτότητας πολλών παραγόντων (MFA), πρέπει να ακολουθήσετε αυτές τις οδηγίες για να συνδεθείτε στο Exchange Online PowerShell: [Συνδεθείτε στο Exchange Online PowerShell χρησιμοποιώντας έλεγχο ταυτότητας πολλών παραγόντων](https://docs.microsoft.com/powershell/exchange/exchange-online/connect-to-exchange-online-powershell/mfa-connect-to-exchange-online-powershell).

**Σημείωση:** Ακόμα και αν έχετε συνδεθεί στο Exchange Online PowerShell στο παρελθόν [χρησιμοποιώντας τις οδηγίες τακτικής σύνδεσης](https://docs.microsoft.com/powershell/exchange/exchange-online/connect-to-exchange-online-powershell/connect-to-exchange-online-powershell), πρέπει να χρησιμοποιήσετε τις οδηγίες σύνδεσης ΣΠΙ μετά την ενεργοποίηση του Σπι για το λογαριασμό σας.
