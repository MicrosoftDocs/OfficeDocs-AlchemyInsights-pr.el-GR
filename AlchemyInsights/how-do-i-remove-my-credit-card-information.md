# <a name="how-do-i-remove-my-credit-card-information"></a>Πώς μπορώ να αφαιρέσω τα στοιχεία της πιστωτικής μου κάρτας;

1. Στο Κέντρο διαχείρισης Microsoft 365, μεταβείτε στη σελίδα **Χρέωση** \> [Μέθοδοι πληρωμής](https://go.microsoft.com/fwlink/p/?linkid=2018806).

2. Στη σελίδα **Μέθοδοι πληρωμής,** επιλέξτε την πιστωτική κάρτα ή τον τραπεζικό λογαριασμό για διαγραφή.

3. Για να καταργήσετε την πιστωτική κάρτα ή τον τραπεζικό λογαριασμό, **επιλέξτε Διαγραφή μεθόδου πληρωμής.**

> [!NOTE]
> Δεν θα μπορείτε να καταργήσετε μια πιστωτική κάρτα ή έναν τραπεζικό λογαριασμό, αν αυτήν τη στιγμή χρησιμοποιείται για την πληρωμή μιας συνδρομής.

4. Στο παράθυρο **Μέθοδος πληρωμής Διαγραφή,** επιλέξτε **Διαγραφή**.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [Προσθήκη, ενημέρωση ή κατάργηση πιστωτικής κάρτας ή τραπεζικού λογαριασμού](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/add-update-or-remove-credit-card-or-bank-account).
