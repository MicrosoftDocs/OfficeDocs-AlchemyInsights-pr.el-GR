# <a name="need-to-configure-your-live-event-with-an-external-encoder"></a>Θέλετε να ρυθμίσετε τις παραμέτρους του ζωντανού συμβάντος σας με έναν εξωτερικό κωδικοποιητή;

Εάν αντιμετωπίζετε προβλήματα με τη λειτουργία του κωδικοποιητή σας με το Live event των ομάδων σας, ανατρέξτε στην τεκμηρίωση εγκατάστασης του κωδικοποιητή: [Ρύθμιση παραμέτρων κωδικοποιητών για ζωντανή ροή στο Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
