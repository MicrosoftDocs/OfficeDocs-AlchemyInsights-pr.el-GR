# <a name="error-message-when-sharing-files-from-windows-explorer"></a>Μήνυμα λάθους κατά την κοινή χρήση αρχείων από την Εξερεύνηση των Windows

Όταν προσπαθείτε να κάνετε κοινή χρήση από την Εξερεύνηση των Windows, ενδέχεται να λάβετε το ακόλουθο μήνυμα λάθους:
  
"Λυπούμαστε, δεν είναι δυνατή η πρόσβαση στο διακομιστή αυτή τη στιγμή. Παρακαλώ προσπαθήστε ξανά αργότερα "
  
Για να επιλύσετε το μήνυμα λάθους, από τον Internet **Explorer μεταβείτε στις** \> **Επιλογές** \> Internet, κάντε κλικ στην καρτέλα **για προχωρημένους** και, στη συνέχεια, απενεργοποιήστε το πλαίσιο ελέγχου **με τίτλο μην αποθηκεύετε τις κρυπτογραφημένες σελίδες στο δίσκο**.
  
 **Σημείωση**: Ίσως χρειαστεί να επανεκκινήσετε το πρόγραμμα περιήγησής σας για να τεθεί σε ισχύ η ρύθμιση. Για περισσότερες πληροφορίες, ανατρέξτε [στο μήνυμα λάθους "δεν είναι δυνατή η πρόσβαση στο διακομιστή τώρα" κατά την κοινή χρήση αρχείων](https://go.microsoft.com/fwlink/?linkid=2022914).
