# <a name="add-or-remove-an-email-address-for-a-user"></a>Προσθήκη ή κατάργηση διεύθυνσης ηλεκτρονικού ταχυδρομείου για ένα χρήστη

[Ακολουθήστε αυτά τα βήματα](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) στο κέντρο διαχείρισης του Microsoft 365 για να ξεκινήσετε.

 **Σημείωση**: Μπορείτε να έχετε περισσότερες από μία διευθύνσεις ηλεκτρονικού ταχυδρομείου για ένα χρήστη, αλλά πρόσθετα *ψευδώνυμα ηλεκτρονικού ταχυδρομείου* μπορούν να χρησιμοποιηθούν μόνο για τη λήψη μηνυμάτων ηλεκτρονικού ταχυδρομείου. Εάν πρέπει να στείλετε μήνυμα ηλεκτρονικού ταχυδρομείου με ψευδώνυμο, δημιουργήστε ένα [κοινόχρηστο γραμματοκιβώτιο](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox). Με αυτόν τον τρόπο θα λάβετε το ψευδώνυμο για την αποστολή και λήψη αλληλογραφίας.
