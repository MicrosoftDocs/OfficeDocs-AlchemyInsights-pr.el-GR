# <a name="a-user-is-having-issues-sending-email-messages"></a>Ένας χρήστης έχει προβλήματα με την αποστολή μηνυμάτων ηλεκτρονικού ταχυδρομείου

Φαίνεται πως έχεις πρόβλημα με τη ροή του ταχυδρομείου. Έχουμε διαγνωστικά που θα σας βοηθήσουν να εντοπίσετε πολλά κοινά προβλήματα ροής αλληλογραφίας. Κάντε κλικ στην επιλογή **εκτέλεση διαγνωστικών** παρακάτω.
