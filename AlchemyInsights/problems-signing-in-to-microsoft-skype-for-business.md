# <a name="problems-signing-in-to-microsoft-skype-for-business"></a>Προβλήματα κατά την είσοδο στο Microsoft Skype για επιχειρήσεις

Κατεβάστε και χρησιμοποιήστε το [Βοηθό υποστήριξης και αποκατάστασης (SaRA).](https://aka.ms/SaRA-SkypeForBusinessSignIn)
Αυτό το εργαλείο μπορεί να σας βοηθήσει να διαγνώσετε και να διορθώσετε αυτόματα μια σειρά γνωστών ζητημάτων που σχετίζονται με την είσοδο στο Skype για επιχειρήσεις.
