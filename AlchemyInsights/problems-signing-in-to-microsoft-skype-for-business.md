# <a name="problems-signing-in-to-microsoft-skype-for-business"></a>Προβλήματα κατά την είσοδο Microsoft Skype για επιχειρήσεις

Κάντε λήψη και χρησιμοποιήστε την [υποστήριξη και Βοηθός αποκατάστασης (SaRA)](https://aka.ms/SaRA-SkypeForBusinessSignIn).
Αυτό το εργαλείο μπορεί να σας βοηθήσει στην αυτόματη διάγνωση και επιδιόρθωση μιας περιοχής γνωστών θεμάτων που σχετίζονται με την είσοδό σας στο Skype για επιχειρήσεις.
