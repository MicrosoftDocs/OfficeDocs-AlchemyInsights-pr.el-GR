# <a name="when-is-my-credit-card-charged"></a>Πότε χρεώνεται η πιστωτική μου κάρτα;

Με μια αρχική αγορά, μπορεί να χρειαστούν έως και δύο ημέρες για να πραγματοποιηθεί η χρέωση.
  
Οι επαναλαμβανόμενες χρεώσεις πραγματοποιούνται εντός μίας ημέρας από τη δημιουργία του τιμολογίου.
  
Εάν η χρέωση αποτύχει, το σύστημα θα συνεχίσει να προσπαθεί να διευθετήσει το υπόλοιπο τις επόμενες εβδομάδες. Μπορείτε να κάνετε ενημερώσεις στα στοιχεία πληρωμής σας οποιαδήποτε στιγμή για να ασφαλίσετε την επόμενη προσπάθεια φόρτισης είναι επιτυχής. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Διαχείριση μεθόδων πληρωμής](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/manage-payment-methods).
