# <a name="learn-about-bing-maps-licensing"></a>Μάθετε σχετικά με τις άδειες χρήσης των Χαρτών Bing

Η υπηρεσία Χάρτες Bing προσφέρει πολλές ευέλικτες επιλογές παραχώρησης αδειών χρήσης που θα υποστηρίζουν τις ανάγκες και τις τεχνικές απαιτήσεις του οργανισμού σας. Επίσης, με τη δωρεάν σειρά, μπορείτε να ξεκινήσετε τη δημιουργία με τους Χάρτες Bing σήμερα.

Για λεπτομέρειες, ανατρέξτε στο θέμα [Άδειες χρήσης χαρτών Bing.](https://go.microsoft.com/fwlink/?linkid=2150203)
