# <a name="enable-seamless-sso"></a>Ενεργοποίηση απρόσκοπτου SSO

Η υπηρεσία καταλόγου Active Directory Azure (Azure AD) με απρόσκοπτη σύνδεση (Απρόσκοπτη σύνδεση SSO) συνδέεται αυτόματα στους χρήστες όταν βρίσκονται σε εταιρικές επιφάνειες εργασίας που είναι συνδεδεμένες στο εταιρικό σας δίκτυο.
  
[Ακολουθήστε τα βήματα σε αυτό το άρθρο για να ενεργοποιήσετε απρόσκοπτη SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
