# <a name="enable-seamless-sso"></a>Ενεργοποίηση ομαλή SSO

Azure υπηρεσίας καταλόγου Active Directory (Azure AD) ομαλή καθολικής σύνδεσης (SSO ομαλή) αυτόματα συνδέεται σε χρήστες όταν βρίσκονται σε τους εταιρικών υπολογιστών οι οποίοι είναι συνδεδεμένοι στο εταιρικό σας δίκτυο.
  
[Ακολουθήστε τα βήματα σε αυτό το άρθρο για να ενεργοποιήσετε την ομαλή SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
