# <a name="enable-seamless-sso"></a>Ενεργοποίηση χωρίς σύνδεση SSO

Η απρόσκοπτη σύνδεση καθολικής σύνδεσης του Azure Active Directory (Azure AD) (χωρίς σύνδεση SSO) συνδέεται αυτόματα με τους χρήστες όταν βρίσκονται στους εταιρικούς υπολογιστές τους που είναι συνδεδεμένοι στο εταιρικό σας δίκτυο.
  
[Ακολουθήστε τα βήματα σε αυτό το άρθρο για να ενεργοποιήσετε την απρόσκοπτη SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
