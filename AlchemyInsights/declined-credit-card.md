# <a name="declined-credit-card"></a>Απόρριψη πιστωτικής κάρτας

Εάν πληρώνετε με πιστωτική κάρτα και η πιστωτική κάρτα σας απορριφθεί, θα λάβετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου που θα σας ενημερώνει ότι δεν μπορέσαμε να επεξεργαστούμε την πληρωμή. Ελέγξτε ξανά ότι τα [στοιχεία της πιστωτικής κάρτας](https://go.microsoft.com/fwlink/p/?linkid=842054) -αριθμός κάρτας, ημερομηνία λήξης, όνομα στην πιστωτική κάρτα και διεύθυνση, συμπεριλαμβανομένης της πόλης, της πολιτείας και του ταχυδρομικού κώδικα-εμφανίζονται ακριβώς όπως στην πιστωτική κάρτα και τη δήλωση της πιστωτικής κάρτας σας. Μπορείτε να ενημερώσετε τα στοιχεία της πιστωτικής κάρτας σας και να υποβάλετε αμέσως την πληρωμή, χρησιμοποιώντας την επιλογή " **Τακτοποίηση υπολοίπου** " στη σελίδα **χρεώσεων**των  >  [προϊόντων σας](https://go.microsoft.com/fwlink/p/?linkid=842054) .

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα "Τι γίνεται εάν η πιστωτική κάρτα μου έχει λήξει και η πληρωμή μου είναι ληξιπρόθεσμη;" στο [Pay για τη συνδρομή σας στο Microsoft 365 για επαγγελματικές συναλλαγές](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/pay-for-your-subscription#what-if-my-credit-card-was-declined-and-my-payment-is-past-due).
  
Εάν συνεχίσετε να βλέπετε το μήνυμα "απορρίφθηκε", επικοινωνήστε με την τράπεζά σας. Είναι πιθανό η πιστωτική κάρτα σας να μην είναι ενεργή — για παράδειγμα, εάν λάβατε πρόσφατα την πιστωτική κάρτα στην αλληλογραφία με μια ενημερωμένη ημερομηνία λήξης, ίσως χρειαστεί να την ενεργοποιήσετε. Η τράπεζά σας μπορεί επίσης να σας ενημερώσει εάν η πιστωτική κάρτα σας δεν έχει εγκριθεί για ηλεκτρονικές, διεθνείς ή επαναλαμβανόμενες συναλλαγές.
  
Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Διαχείριση μεθόδων πληρωμής](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/manage-payment-methods).
