# <a name="microsoft-store-support"></a>Υποστήριξη του Microsoft Store

Είμαστε σε ακρόαση και θέλουμε να σας βοηθήσουμε. Εάν χρειάζεστε βοήθεια σχετικά με την εύρεση πληροφοριών σχετικά με μια αγορά προϊόντος, την κατάσταση παραγγελίας, θέματα χρέωσης, ειδικές προσφορές, τον εντοπισμό πληροφοριών προϊόντος ή την υποστήριξη του Microsoft Store, ανατρέξτε στο θέμα Επικοινωνία με την υποστήριξη [του Microsoft Store.](https://support.microsoft.com/account-billing/contact-microsoft-store-support-4f615f2a-6bbd-fd69-6695-ae213d63eef0)

Για να μάθετε περισσότερα σχετικά με τις συσκευές που κυκλοφόρησαν πρόσφατα ή τα επερχόμενα προϊόντα από τη Microsoft, ανατρέξτε [στο θέμα Microsoft Store.](https://www.microsoft.com/?ql=1)

Σας ευχαριστούμε που είστε πελάτης της Microsoft.
