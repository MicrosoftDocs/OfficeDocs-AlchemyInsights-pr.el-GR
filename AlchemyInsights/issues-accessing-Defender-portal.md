# <a name="issues-accessing-the-microsoft-defender-atp-portal"></a>Προβλήματα πρόσβασης στην πύλη του Microsoft Defender ATP

Εάν λάβετε ένα μήνυμα "Η συνδρομή σας έχει λήξει", η online συνδρομή σας στην υπηρεσία έχει λήξει. Μια συνδρομή στο Microsoft Defender ATP, όπως και άλλες συνδρομές online υπηρεσιών, έχει ημερομηνία λήξης.

Το μήνυμα "Η συνδρομή σας έχει λήξει" παρουσιάζει μια επιλογή για να κάνετε λήψη του πακέτου εκτός πίνακα της συσκευής, εάν επιλέξετε να μην ανανεώσετε την άδεια χρήσης σας.
