# <a name="got-a-parameter-cannot-be-found-error"></a>Έχετε ένα σφάλμα "Δεν είναι δυνατή η ανήκυξη της παραμέτρου";

Εάν λάβετε το μήνυμα λάθους "Δεν είναι δυνατή η **εύρεση μιας παραμέτρου που ταιριάζει με το όνομα της παραμέτρου 'EmailAddresses",** αυτό σημαίνει ότι χρειάζεται λίγοπερισσότερος χρόνος για να ολοκληρωθεί η ρύθμιση του μισθωτή σας από το Microsoft 365 ή ο προσαρμοσμένος τομέας σας, εάν προσθέσατε πρόσφατα έναν.
  
Η διαδικασία εγκατάστασης μπορεί να διαρκέσει έως και 4 ώρες για να ολοκληρωθεί. Περιμένετε λίγο, ώστε η διαδικασία ρύθμισης να έχει χρόνο να ολοκληρωθεί και, στη συνέχεια, προσπαθήστε ξανά.
