# <a name="outlookcom-premium"></a>Outlook.com Premium

**Υπάρχοντες συνδρομητές Premium**

- Εάν έχετε ήδη μια συνδρομή στο Outlook.com Premium, διαβάστε τις [Συνήθεις ερωτήσεις για το Outlook.com Premium](https://support.office.com/article/cd5f03f6-1407-456a-9410-f8f24804746b?wt.mc_id=Office_Outlook_com_Alchemy).

**Νέοι συνδρομητές**

- Λυπούμαστε, αλλά το Outlook.com Premium είναι πλέον κλειστό για τους νέους συνδρομητές.
- Πολλές από τις [δυνατότητες και τα πλεονεκτήματα](https://support.office.com/article/78c6089c-7faf-44f5-82e2-efa9ebb921d2?wt.mc_id=Office_Outlook_com_Alchemy) περιλαμβάνονται πλέον με τις [συνδρομές στο Microsoft 365 Home και το Microsoft 365 Personal](https://go.microsoft.com/fwlink/?linkid=2017122).
