# <a name="sharepoint-libraries-mapped-to-network-drives-become-disconnected"></a>Οι βιβλιοθήκες του SharePoint που αντιστοιχίζονται σε μονάδες δικτύου καθίστανται αποσυνδεδεμένες

Η αντιστοίχιση μιας βιβλιοθήκης εγγράφων του SharePoint σε μια μονάδα δίσκου δικτύου είναι προσωρινή. Πρέπει περιστασιακά να ανοίξετε την τοποθεσία του SharePoint στον Internet Explorer και να επιλέξετε να **παραμείνετε συνδεδεμένοι** για να αποτρέψετε τη λήξη της περιόδου λειτουργίας. 
  
[Λήψη βοήθειας για τη ρύθμιση και την αντιμετώπιση προβλημάτων αντιστοιχισμένων δίσκων δικτύου](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives).
  
Για να εργαστείτε με μια βιβλιοθήκη στην Εξερεύνηση αρχείων, [Συγχρονίστε τα αρχεία του SharePoint με το νέο πρόγραμμα-πελάτη συγχρονισμού OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) που περιλαμβάνει τα [αρχεία κατ ' απαίτηση](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Αποκτήστε πρόσβαση σε όλα τα αρχεία σας στο OneDrive χωρίς να χρησιμοποιείτε τοπικό αποθηκευτικό χώρο.
