# <a name="allow-users-to-send-email-as-a-distribution-group"></a>Να επιτρέπεται στους χρήστες να στέλνουν μηνύματα ηλεκτρονικού ταχυδρομείου ως ομάδα διανομής

Για να επιτρέψετε στους χρήστες να στέλνουν μηνύματα που εμφανίζονται να προέρχονται από μια υπάρχουσα ομάδα διανομής, ακολουθήστε τα εξής βήματα:

1. Στο [Κέντρο διαχείρισης Exchange](https://outlook.office365.com/ecp/), μεταβείτε στις ομάδες **παραληπτών** \> ****.

2. Επιλέξτε την ομάδα διανομής που θέλετε να τροποποιήσετε και, στη συνέχεια, κάντε κλικ στο κουμπί **Επεξεργασία**.

3. Στις ιδιότητες της ομάδας, μεταβείτε στην καρτέλα **αντιπροσώπευση ομάδας** για να καθορίσετε τους χρήστες που μπορούν να στείλουν ή να στείλουν εκ μέρους της ομάδας διανομής.

Για περισσότερες πληροφορίες, ανατρέξτε στο στοιχείο [Επιλογές ηλεκτρονικού ταχυδρομείου](https://technet.microsoft.com/library/bb124513.aspx#groupdelegation).
