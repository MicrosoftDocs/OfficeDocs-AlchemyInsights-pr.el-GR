# <a name="control-creation-of-microsoft-365-groups"></a>Δημιουργία ελέγχου ομάδων του Microsoft 365

Από προεπιλογή, όλοι οι χρήστες της εταιρείας μπορούν να δημιουργήσουν ομάδες. Εάν χρειάζεται, μπορείτε να το αλλάξετε και να εκχωρήσετε το δικαίωμα δημιουργίας ομάδων σε ένα σύνολο ατόμων εντός της εταιρείας σας. Σημειώστε ότι αυτό θα έχει αντίκτυπο στη δημιουργία ομάδων από οποιαδήποτε υπηρεσία της Microsoft 365 (όπως το Outlook, ομάδες, Planner και ούτω καθεξής).
  
Οδηγίες: [Διαχείριση των χρηστών που μπορούν να δημιουργήσουν ομάδες του Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
