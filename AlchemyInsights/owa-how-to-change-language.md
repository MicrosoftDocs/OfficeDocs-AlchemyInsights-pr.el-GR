# <a name="change-your-language-date-and-time-format-or-time-zone"></a>Αλλαγή σας γλώσσα, ημερομηνία και ώρα μορφή ή ζώνης ώρας

1. Στην κορυφή της σελίδας, επιλέξτε **Ρυθμίσεις** > **προβάλετε όλες τις ρυθμίσεις Outlook**.
2. Ενεργοποιήστε την επιλογή **Γενικά** > **γλώσσα και την ώρα**.
3. Επιλέξτε τη γλώσσα που θέλετε να χρησιμοποιήσετε.
4. Επιλέξτε **Αποθήκευση**.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [Αλλαγή της ζώνης ώρας και τις ρυθμίσεις γλώσσας του Outlook στο web](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
