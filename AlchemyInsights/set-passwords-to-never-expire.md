# <a name="set-passwords-to-never-expire"></a>Ορισμός κωδικών πρόσβασης χωρίς λήξη

Ορισμός κωδικών πρόσβασης χωρίς λήξη από [ασφαλείας &amp; απορρήτου](https://portal.office.com/adminportal/home#/settings/security). Κάντε κλικ στο κουμπί "Επεξεργασία" δίπλα από την πολιτική κωδικών πρόσβασης και ενεργοποιήστε το ρυθμιστικό για να "στην ενέργεια".
  
Για περισσότερες πληροφορίες ανατρέξτε στο θέμα: [Ορισμός η πολιτική λήξης του κωδικού πρόσβασης για τον οργανισμό σας](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
