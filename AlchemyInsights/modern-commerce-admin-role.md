# <a name="modern-commerce-administrator-role"></a>Ρόλος διαχειριστή σύγχρονου εμπορίου

Αυτός ο ρόλος εκχωρείται αυτόματα σε χρήστες που πραγματοποιούν μια αγορά για να τους παρέχουν περιορισμένη πρόσβαση στην πύλη διαχείρισης για τη διαχείριση αυτής της αγοράς. Αυτοί οι χρήστες δεν είναι διαχειριστές και δεν έχουν αυξημένα δικαιώματα.
