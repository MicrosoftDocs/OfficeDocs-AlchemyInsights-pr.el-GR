# <a name="sharepoint-migration-with-spmt"></a>Μετεγκατάσταση του SharePoint με SPMT

Σχεδιασμένο για να χρησιμοποιείται για μετεγκαταστάσεις που κυμαίνονται από το μικρότερο σύνολο αρχείων σε μια μεγάλης κλίμακας επιχειρηματική μετεγκατάσταση, το εργαλείο μετεγκατάστασης του SharePoint θα σας επιτρέψει να μεταφέρετε τις πληροφορίες σας στο cloud και να επωφεληθείτε από την νεότερη συνεργασία, νοημοσύνη και λύσεις ασφαλείας με το Office 365.

- [Λήψη και εγκατάσταση του εργαλείου μετεγκατάστασης του SharePoint](https://docs.microsoft.com/sharepointmigration/introducing-the-sharepoint-migration-tool)
- [Αντιμετώπιση προβλημάτων κοινών θεμάτων και σφαλμάτων SPMT](https://docs.microsoft.com/sharepointmigration/troubleshooting-common-spmt-issues)
- [Αντιμετώπιση προβλημάτων εγκατάστασης SPMT](https://docs.microsoft.com/sharepointmigration/spmt-install-issues#troubleshooting-spmt-installation-issues)
