# <a name="banned-sending-ip"></a>Απαγορευμένη αποστολή IP

Κατά την αποστολή σε έναν παραλήπτη του Office 365, ένας αποστολέας μπορεί να αντιμετωπίσει αυτό το σφάλμα. Ο αποστολέας πρέπει να επισκεφθεί την [πύλη απολίστας IP του Office 365 κατά των ανεπιθύμητων μηνυμάτων IP](https://sender.office.com/) και να παρέχει τη διεύθυνση ηλεκτρονικού ταχυδρομείου και τη διεύθυνση IP που πρέπει να παραθέσετε. Εάν δεν λάβουν ένα μήνυμα ηλεκτρονικού ταχυδρομείου επαλήθευσης από την πύλη, ζητήστε τους να ελέγχουν το φάκελο ανεπιθύμητης αλληλογραφίας ή/και να δοκιμάσουν με διαφορετική διεύθυνση ηλεκτρονικού ταχυδρομείου. 

- Για λεπτομερείς οδηγίες, ανατρέξτε [στο θέμα χρήση της πύλης "Κατάργηση λίστας" για να καταργήσετε τον εαυτό σας από τη λίστα αποκλεισμένων αποστολέων](https://docs.microsoft.com/microsoft-365/security/office-365-security/use-the-delist-portal-to-remove-yourself-from-the-office-365-blocked-senders-lis?view=o365-worldwide).
