# <a name="banned-sending-ip"></a>Απαγορευμένη αποστολή IP

Κατά την αποστολή αλληλογραφίας στο Microsoft 365, ενδέχεται να αντιμετωπίσετε αυτό το σφάλμα σε έναν αποστολέα. Ο αποστολέας https://sender.office.com πρέπει να επισκεφθεί και να ακολουθήσει τις οδηγίες.  Εάν δεν λάβουν ένα μήνυμα ηλεκτρονικού ταχυδρομείου επαλήθευσης από την πύλη, πείτε τους να ελέγξουν το φάκελο ανεπιθύμητης αλληλογραφίας και/ή να δοκιμάσουν με διαφορετική διεύθυνση ηλεκτρονικού ταχυδρομείου.
