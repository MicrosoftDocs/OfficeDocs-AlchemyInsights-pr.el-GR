# <a name="banned-sending-ip"></a>Αποκλεισμένη διεύθυνση IP αποστολής

Κατά την αποστολή σε έναν παραλήπτη του Office 365, ένας αποστολέας μπορεί να αντιμετωπίσετε αυτό το σφάλμα. Ο αποστολέας πρέπει να επισκεφτεί την Πύλη λίστας διευθύνσεων IP του [Office 365](https://sender.office.com/) κατά της ανεπιθύμητης αλληλογραφίας και να παράσχει τη διεύθυνση ηλεκτρονικού ταχυδρομείου και τη διεύθυνση IP που πρέπει να παρατεθεί. Εάν δεν λάβουν ένα μήνυμα ηλεκτρονικού ταχυδρομείου επαλήθευσης από την πύλη, πρέπει να ελέγχουν το φάκελο ανεπιθύμητης αλληλογραφίας ή/και να προσπαθούν με διαφορετική διεύθυνση ηλεκτρονικού ταχυδρομείου. 

- Για λεπτομερείς οδηγίες, ανατρέξτε στο θέμα Χρήση της πύλης κατάργησης λίστας για να καταργήσετε τον εαυτό [σας από τη λίστα αποκλεισμένων αποστολέων.](https://docs.microsoft.com/microsoft-365/security/office-365-security/use-the-delist-portal-to-remove-yourself-from-the-office-365-blocked-senders-lis?view=o365-worldwide)
