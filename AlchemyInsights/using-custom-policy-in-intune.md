# <a name="using-custom-policy-in-intune"></a>Χρήση προσαρμοσμένης πολιτικής στο Intune

Intune διαθέτει πολλές ενσωματωμένες ρυθμίσεις για συσκευές Android, iOS, macOS και Windows. Μπορείτε να επίσης να δημιουργήσετε ένα προσαρμοσμένο προφίλ και, στη συνέχεια, να αναπτύξετε αυτές τις προσαρμοσμένες ρυθμίσεις για τις συσκευές σας χρησιμοποιώντας Intune. Προσαρμοσμένα προφίλ έχουν σχεδιαστεί για να προσθέσετε ρυθμίσεις συσκευής και δυνατότητες που δεν είναι ενσωματωμένες Intune.

[Προσαρμοσμένες ρυθμίσεις για συσκευές Android στο Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Προσαρμοσμένες ρυθμίσεις για συσκευές Android εταιρικού Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Προσαρμοσμένες ρυθμίσεις για συσκευές iOS Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Προσαρμοσμένες ρυθμίσεις για συσκευές macOS Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Προσαρμοσμένες ρυθμίσεις για συσκευές Windows 10 Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Προσαρμοσμένες ρυθμίσεις για Windows ολόγραμμα για επαγγελματικές συσκευές στο Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
