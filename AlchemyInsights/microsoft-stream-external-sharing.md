# <a name="microsoft-stream-external-sharing"></a>Εξωτερική κοινή χρήση ροής της Microsoft

Η Microsoft Stream δεν υποστηρίζει ακόμη σενάρια εξωτερικής κοινής χρήσης για επισκέπτες ή άτομα εκτός της εταιρείας σας. Γνωρίζουμε ότι αυτές οι δυνατότητες είναι ιδιαίτερα απαραίτητες και εργαζόμαστε για τα προγράμματα για τον τρόπο δόμησης αυτών των δυνατοτήτων.

- [Εξωτερική κοινή χρήση σε ροή](https://docs.microsoft.com/stream/portal-share-video#external-sharing)

- [Κοινή χρήση των βίντεό σας στο Microsoft Stream](https://docs.microsoft.com/stream/portal-share-video)
