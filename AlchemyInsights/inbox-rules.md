# <a name="inbox-rules"></a>Κανόνες εισερχομένων

Για περισσότερες πληροφορίες σχετικά με τη χρήση των κανόνων εισερχομένων στο γραμματοκιβώτιό σας, ανατρέξτε στα θέματα:

- [Διαχείριση μηνυμάτων ηλεκτρονικού ταχυδρομείου με χρήση κανόνων στο Outlook για PC](https://support.office.com/article/c24f5dea-9465-4df4-ad17-a50704d66c59.aspx)

- [Διαχείριση των μηνυμάτων σας με χρήση κανόνων εισερχομένων στο Outlook στο web](https://support.office.com/article/8400435c-f14e-4272-9004-1548bb1848f2.aspx)

Μπορείτε επίσης να διαχειριστείτε τους κανόνες εισερχομένων στο Exchange Online PowerShell. Για περισσότερες πληροφορίες, [ανατρέξτε στα θέματα Get-InboxRule](https://docs.microsoft.com/powershell/module/exchange/mailboxes/get-inboxrule) (προβολή), [Κανόνας νεοεισερχόμενων](https://docs.microsoft.com/powershell/module/exchange/mailboxes/new-inboxrule) (δημιουργία), [Κανόνας κατάργησης εισερχομένων](https://docs.microsoft.com/powershell/module/exchange/mailboxes/remove-inboxrule) (διαγραφή), [Κανόνας εισερχομένων](https://docs.microsoft.com/powershell/module/exchange/mailboxes/set-inboxrule) (τροποποίηση υπάρχοντος), [Απενεργοποίηση κανόνας εισερχομένων](https://docs.microsoft.com/powershell/module/exchange/mailboxes/disable-inboxrule) (απενεργοποίηση υπάρχοντος) και [Κανόνας ενεργοποίησης εισερχομένων](https://docs.microsoft.com/powershell/module/exchange/mailboxes/enable-inboxrule) (ενεργοποίηση υπάρχοντος).
