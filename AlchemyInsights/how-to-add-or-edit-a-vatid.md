# <a name="how-to-add-or-edit-a-vatid"></a>Πώς μπορείτε να προσθέσετε ή να επεξεργαστείτε ένα VatID

**Σημείωση**: Εάν βρίσκεστε σε μια χώρα που χρησιμοποιεί φόρο προστιθέμενης ΑΞΊΑς (ΦΠΑ), όπως το Ηνωμένο Βασίλειο, η Microsoft τιμολογήσεις ΥΠΟΛΟΓΊΖΕΙ το ΦΠΑ για εσάς και εφαρμόζει το ποσό στα τιμολόγιά σας. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Αποποίηση ευθυνών του VIES](https://go.microsoft.com/fwlink/p/?LinkID=841741) που δημοσιεύεται από την Ευρωπαϊκή Επιτροπή.

**Προσθήκη ή επεξεργασία VatID**

1. Στο κέντρο διαχείρισης του Microsoft 365, μεταβείτε στη σελίδα **χρέωση** των \> [προϊόντων σας](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Εντοπίστε την κάρτα συνδρομής για τη συνδρομή που θέλετε να ενημερώσετε.

3. Στην ενότητα **ρυθμίσεις & ενεργειών** , επιλέξτε **Επεξεργασία διεύθυνσης χρήσης υπηρεσίας**.

4. Στη σελίδα **Επεξεργασία διεύθυνσης χρήσης υπηρεσίας** , πληκτρολογήστε το ΑΦΜ σας στο πλαίσιο **ΑΦΜ** και, στη συνέχεια, επιλέξτε **Αποθήκευση**.

    Όταν ολοκληρωθεί, ο ΦΠΑ δεν θα χρεωθεί στον επόμενο λογαριασμό σας.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [Τι φόρος θα χρεωθεί για το Microsoft 365;](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/tax-information).

**Πρόσθετοι πόροι**

[Προσθέστε τον ΑΦΜ σας](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/tax-information?view=o365-worldwide#add-your-vat-id-eu-countries-only).
