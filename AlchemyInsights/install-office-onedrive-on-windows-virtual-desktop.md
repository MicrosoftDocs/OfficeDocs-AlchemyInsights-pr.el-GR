# <a name="install-office-and-onedrive-on-windows-virtual-desktop"></a>Εγκατάσταση του Office και του OneDrive στην εικονική επιφάνεια εργασίας των Windows

1. [Προετοιμασία και προσαρμογή μιας κύριας εικόνας VHD.](https://docs.microsoft.com/azure/virtual-desktop/set-up-customize-master-image) Δημιουργήστε μια εικονική μηχανή (VM) εάν δεν έχει ήδη δημιουργηθεί.

1. [Εγκαταστήστε το Office σε λειτουργία ενεργοποίησης κοινόχρηστου υπολογιστή.](https://docs.microsoft.com/azure/virtual-desktop/install-office-on-wvd-master-image#install-office-in-shared-computer-activation-mode) Η ενεργοποίηση κοινόχρηστου υπολογιστή επιτρέπει σε πολλούς χρήστες να αποκτήσουν πρόσβαση στο Office.

1. [Εγκαταστήστε το OneDrive σε λειτουργία ανά υπολογιστή.](https://docs.microsoft.com/azure/virtual-desktop/install-office-on-wvd-master-image#install-onedrive-in-per-machine-mode) Κανονικά, το OneDrive εγκαθίσταται ανά χρήστη, αλλά εδώ θα πρέπει να εγκατασταθεί ανά υπολογιστή.
