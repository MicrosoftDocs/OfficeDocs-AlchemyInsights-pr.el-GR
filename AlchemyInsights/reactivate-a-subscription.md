# <a name="reactivate-a-subscription"></a>Επανενεργοποίηση της συνδρομής

Εάν η συνδρομή σας λήξει, ήταν απενεργοποιημένη από τη Microsoft, ή ακύρωση στη μέση μιας περιόδου συνδρομής, μπορείτε να ενεργοποιήσετε ξανά στο στο Κέντρο διαχείρισης του Office 365.
  
1. Στο Κέντρο διαχείρισης, μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**.
    
2. Επιλέξτε την εγγραφή που θέλετε να ενεργοποιήσετε εκ νέου και στη συνέχεια επιλέξτε **Επανενεργοποίηση**.
    
    Εάν δεν βλέπετε **Επανενεργοποίηση** ως μια διαθέσιμη ενέργεια, χρησιμοποιήστε την επιλογή "ας καλέσουν" για να συνομιλήσετε με έναν μηχανικό υποστήριξης για να ενεργοποιήσετε ξανά τη συνδρομή σας. 
    
3. Εισαγάγετε τις λεπτομέρειες της πληρωμής σας. Μπορείτε να ενημερώσετε τις υπάρχουσες πληροφορίες πληρωμής εδώ.
