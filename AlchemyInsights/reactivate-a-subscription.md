# <a name="reactivate-a-microsoft-365-subscription"></a>Επανενεργοποίηση συνδρομής Microsoft 365

Εάν η συνδρομή σας λήξει, απενεργοποιηθεί από τη Microsoft ή την ακυρώσατε στη μέση μιας περιόδου συνδρομής, μπορείτε να την ενεργοποιήσετε ξανά στο κέντρο διαχείρισης.
  
1. Στο κέντρο διαχείρισης, μεταβείτε στη σελίδα **Χρέωση** \> **[των προϊόντων σας.](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. Επιλέξτε τη συνδρομή που θέλετε να ενεργοποιήσετε ξανά. Στην ενότητα Χρέωση, επιλέξτε **Επανενεργοποίηση**.

    Εάν δεν βλέπετε την **επιλογή "Επανενεργοποίηση"** ως διαθέσιμη ενέργεια, [καλέστε την υποστήριξη](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products) για να ενεργοποιήσετε ξανά τη συνδρομή σας.

3. Επιβεβαιώστε τα στοιχεία πληρωμής σας. Μπορείτε να ενημερώσετε τα υπάρχοντα στοιχεία πληρωμής εδώ. Όταν ολοκληρώσετε την εισαγωγή της μεθόδου πληρωμής σας, **επιλέξτε Επανενεργοποίηση**.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [Επανενεργοποίηση της συνδρομής σας](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/reactivate-your-subscription).
