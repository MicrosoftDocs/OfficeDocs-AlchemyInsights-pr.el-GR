# <a name="join-a-team-by-code"></a>Συμμετοχή σε ομάδα με κωδικό

Η Microsoft γνωρίζει ένα πρόβλημα όπου οι  χρήστες δεν μπορούν να δημιουργήσουν κώδικα ομάδας ή να συμμετάσχουν σε μια ομάδα χρησιμοποιώντας τη δυνατότητα "Συμμετοχή **μέσω κώδικα".** Ως λύση, οι κάτοχοι ομάδας μπορούν να προσθέσουν χρήστες με μη αυτόματο τρόπο, όπως απαιτείται.
