# <a name="phone-system-license"></a>Άδεια χρήσης συστήματος τηλεφώνου

Η **Άδεια χρήσης συστήματος τηλεφώνου** αποτελεί πρόσθετο του **το Microsoft Teams**. Η άδεια χρήσης είναι διαθέσιμη από προεπιλογή με το [Office 365 για μεγάλες επιχειρήσεις E5](https://www.microsoft.com/microsoft-365/business/office-365-enterprise-e5-business-software?rtc=1&activetab=pivot%3aoverviewtab) και θα πρέπει να το αγοράσετε ως πρόσθετο για το [Office 365 για μεγάλες επιχειρήσεις E1](https://products.office.com/business/office-365-enterprise-e1-business-software) ή το[πρόγραμμα E3](https://products.office.com/business/office-365-enterprise-e3-business-software).

1. Είσοδος στο Κέντρο διαχείρισης Microsoft 365.

2. Μετάβαση στο **Χρεώσεις > Αγορά υπηρεσιών > Συνδρομές πρόσθετων**. 

3. Θα δείτε το πρόσθετο Σύστημα τηλεφώνου. Κάντε κλικ στην επιλογή **Αγορά τώρα**.

Αφού αγοράσετε μία Άδεια χρήσης συστήματος τηλεφώνου, θα εμφανιστεί η τιμή και η επιλογή για την αγορά ενός Προγράμματος κλήσεων.

**Σημείωση**:Ενδέχεται να χρειαστούν από 2 έως και 24 ώρες για να τεθούν σε ισχύ οι αλλαγές. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ρύθμιση συστήματος τηλεφώνου στον οργανισμό σας](https://docs.microsoft.com/MicrosoftTeams/setting-up-your-phone-system).
