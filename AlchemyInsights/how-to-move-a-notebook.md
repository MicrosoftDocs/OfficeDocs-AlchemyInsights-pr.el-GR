# <a name="how-to-move-a-notebook"></a>Τρόπος μετακίνησης ενός σημειωματαρίου

* Για να μετακινήσετε ένα σημειωματάριο του OneNote OneDrive, μεταβείτε στο **αρχείο** > **κοινόχρηστο στοιχείο** > **Μετακίνηση σημειωματάρια** του OneNote 2016. Δείτε [Μετακίνηση ένα σημειωματάριο του OneNote για να OneDrive](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082) για περισσότερες πληροφορίες

* Για πρόσθετες πληροφορίες σχετικά με τις βέλτιστες πρακτικές καθώς προετοιμάζετε για να μετακινήσετε τα σημειωματάρια, δείτε [το OneNote συγχρονισμό βέλτιστες πρακτικές](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices)

* [Μετακίνηση ένα σημειωματάριο του OneNote για Windows που έχετε καταχωρήσει με άλλους](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4) παρέχει περισσότερες πληροφορίες σχετικά με τη μετακίνηση των σημειωματαρίων κοινής χρήσης.
