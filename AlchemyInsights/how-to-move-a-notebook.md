# <a name="how-to-move-a-onenote-notebook"></a>Πώς μπορείτε να μετακινήσετε ένα σημειωματάριο του OneNote

* Για να μετακινήσετε ένα σημειωματάριο του OneNote στο OneDrive, **File**μεταβείτε στα  >  **Share**  >  **σημειωματάρια μετακίνησης** κοινόχρηστων αρχείων στο OneNote 2016. Ανατρέξτε [στο θέμα Μετακίνηση σημειωματαρίου του OneNote στο OneDrive](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082) για περισσότερες πληροφορίες.

* Για πρόσθετες πληροφορίες σχετικά με τις βέλτιστες πρακτικές καθώς προετοιμάζετε να μετακινήσετε σημειωματάρια, ανατρέξτε στο θέμα [Συγχρονισμός βέλτιστων πρακτικών στο OneNote](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices).

* [Μετακίνηση ενός σημειωματαρίου του OneNote για Windows που έχετε μοιραστεί με άλλους](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4) παρέχει περισσότερες πληροφορίες σχετικά με τη μετακίνηση κοινόχρηστων σημειωματαρίων.
