# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>Δεν είναι δυνατή η διαγραφή αρχείων ή φωτογραφιών από αρχεία στο Outlook στο Web

Τα **αρχεία** εμφανίζουν τα αρχεία και τις φωτογραφίες που είναι συνημμένα σε μηνύματα στο γραμματοκιβώτιό σας. Για να καταργήσετε αρχεία και φωτογραφίες από αρχεία, πρέπει να διαγράψετε τα μηνύματα ηλεκτρονικού ταχυδρομείου στα οποία είναι συνημμένα. Μετά τη διαγραφή των μηνυμάτων, πρέπει επίσης να τα διαγράψετε από το φάκελο "Διαγραμμένα".
