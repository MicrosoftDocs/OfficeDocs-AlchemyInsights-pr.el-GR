# <a name="sharepoint-online-public-websites-are-being-discontinued"></a>Οι δημόσιες τοποθεσίες Web του SharePoint Online θα καταργηθούν

Εάν δημιουργήσατε μια δημόσια τοποθεσία Web του SharePoint Online, μπορείτε να συνεχίσετε να τη χρησιμοποιείτε μέχρι τις 31 Μαρτίου 2018. Εάν θέλετε να συνεχίσετε να έχετε μια δημόσια τοποθεσία Web, πρέπει να δημιουργήσετε ένα νέο με μια νέα υπηρεσία παροχής. Το περιεχόμενο στη δημόσια τοποθεσία Web του SharePoint Online θα διαγραφεί, επομένως πρέπει να αντιγράψετε το περιεχόμενο που θέλετε να διατηρήσετε.
  
Ένα πανό εμφανίζεται τώρα στο επάνω μέρος της δημόσιας τοποθεσίας Web του SharePoint Online ως ειδοποίηση ότι η τοποθεσία θα καταργηθεί. Δεν μπορείτε να απενεργοποιήσετε αυτό το πανό. Για πληροφορίες σχετικά με την αποθήκευση και την αντιγραφή περιεχομένου σε μια τοποθεσία Web συνεργάτη, ανατρέξτε [στο θέμα λεπτομέρειες σχετικά με τη διαδικασία διακοπής της δυνατότητας](https://go.microsoft.com/fwlink/?linkid=866980).
