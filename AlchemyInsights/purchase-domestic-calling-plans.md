# <a name="purchase-domestic-calling-plans"></a>Αγορά προγραμμάτων τοπικών κλήσεων

Υπάρχουν δύο τύποι προγραμμάτων κλήσεων που είναι διαθέσιμα με το Microsoft 365, για **Τοπικ** και **Τοπικές και διεθνείς**. [Επιλέξτε τη χώρα ή την περιοχή σας για να δείτε τι είναι διαθέσιμο για τον οργανισμό σας](https://docs.microsoft.com/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans#select-your-country-or-region-to-see-whats-available-for-your-organization).

- Ανατρέξτε στο θέμα [Σύστημα τηλεφώνου και Προγράμματα κλήσεων](https://docs.microsoft.com/MicrosoftTeams/calling-plan-landing-page) για περισσότερες λεπτομέρειες.
- Ανατρέξτε στο θέμα [Τιμολόγηση για τηλεφωνικό σύστημα](https://www.microsoft.com/microsoft-365/microsoft-teams/voice-calling#Requirements) για να δείτε προγράμματα και τιμές.
- Προσθέστε [Πιστωτικές μονάδες επικοινωνίας](https://docs.microsoft.com/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans#communications-credits) για τους χρήστες που χρειάζονται τη δυνατότητα να πραγματοποιούν εξερχόμενες κλήσεις σε οποιονδήποτε προορισμό.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Διαθεσιμότητα χώρας και περιοχής για προγράμματα διασκέψεων ήχου και προγράμματα κλήσεων](https://docs.microsoft.com/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans).
