# <a name="disabling-briefing-email"></a>Απενεργοποίηση ενημερωτικού μηνύματος ηλεκτρονικού ταχυδρομείου

Ως διαχειριστής, έχετε την ευελιξία να απενεργοποιήσετε την ενημέρωση για ολόκληρο τον οργανισμό σας ή για μεμονωμένους χρήστες. Για να μάθετε πώς μπορείτε να διαχειριστείτε αυτές τις ρυθμίσεις, ανατρέξτε στο θέμα [Ρύθμιση παραμέτρων ηλεκτρονικού ταχυδρομείου ενημέρωσης](https://docs.microsoft.com/briefing/be-admin).
