# <a name="security-defaults-is-enabled"></a>Οι προεπιλογές ασφαλείας είναι ενεργοποιημένες

Η ρύθμιση "Προεπιλογές ασφαλείας" ζητά από όλους τους χρήστες έλεγχο ταυτότητας πολλών παραγόντων.

Για να διαχειριστείτε τις προεπιλογές ασφαλείας:

1. Εισέλθετε στην [πύλη Azure](https://ms.portal.azure.com/)με τα   διαπιστευτήρια καθολικού διαχειριστή.
2. Μεταβείτε στις Ιδιότητες **της υπηρεσίας καταλόγου Active Directory Azure**  >  **Properties**.
3. Στο κάτω μέρος της σελίδας, κάντε κλικ στην επιλογή **Διαχείριση προεπιλεγμένων ρυθμίσεων ασφαλείας**.
4. Κάντε κλικ στην επιλογή **Ναι** για να ενεργοποιήσετε τις προεπιλογές ασφαλείας ή **Όχι** για να απενεργοποιήσετε τις προεπιλογές ασφαλείας.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ποιες είναι οι προεπιλογές ασφαλείας;](https://docs.microsoft.com/azure/active-directory/fundamentals/concept-fundamentals-security-defaults).
