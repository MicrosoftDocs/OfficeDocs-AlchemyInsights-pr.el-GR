# <a name="how-to-attach-files"></a>Πώς να επισυνάψετε αρχεία

Για να επισυνάψετε αρχεία σε κλασική Outlook.com:
  
1. Συντάξτε ένα νέο μήνυμα ή απαντάτε σε ένα μήνυμα.
    
2. Στην κορυφή του παραθύρου σύνταξης, επιλέξτε **Επισύναψη**. 
    
Για να επισυνάψετε αρχεία σε την έκδοση beta του Outlook.com:
  
1. Συντάξτε ένα νέο μήνυμα ή απαντάτε σε ένα μήνυμα.
    
2. Στο κάτω μέρος του παραθύρου σύνταξης, επιλέξτε ![Επισύναψη](media/da223d01-5fe6-448c-a3a3-e2b5262da4b9.png).
    
Μάθετε περισσότερα σχετικά με [την επισύναψη αρχείων στο Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001702&amp;clcid=0x409).
