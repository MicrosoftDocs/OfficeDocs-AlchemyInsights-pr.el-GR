# <a name="changes-to-external-user-access"></a>Αλλαγές στην πρόσβαση εξωτερικών χρηστών

Από τις 23 Μαρτίου 2018, οι εξωτερικοί χρήστες δεν θα μπορούν πλέον να έχουν πρόσβαση σε περιεχόμενο που έχει τεθεί σε κοινή χρήση με όλους, όλους τους χρήστες με έλεγχο ταυτότητας ή όλους τους χρήστες φορμών, ακόμα και αν έχουν προσκληθεί ή προστεθεί σε εξουσιοδοτημένες ομάδες. Οι εξωτερικοί χρήστες θα μπορούν να έχουν πρόσβαση μόνο σε περιεχόμενο που έχει τεθεί σε κοινή χρήση με το χρήστη ή με ομάδες στις οποίες ανήκει ο χρήστης. Εάν θέλετε να διατηρήσετε τις υπάρχουσες ρυθμίσεις σας, ανατρέξτε στα θέματα: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
