# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>Συγχρονισμός βιβλιοθήκης του SharePoint με το Συγχρονισμό του OneDrive

Για να συγχρονιστούν τα αρχεία του SharePoint με τον τοπικό σκληρό δίσκο, ολοκληρώστε τα ακόλουθα βήματα:

1. Μεταβείτε στη βιβλιοθήκη εγγράφων που θέλετε να συγχρονίσετε με τον τοπικό σκληρό δίσκο.
2. Επιλέξτε το κουμπί Συγχρονισμός από τις επιλογές μενού.
3. Ακολουθήστε βήμα προς βήμα για να ολοκληρώσετε το συγχρονισμό.
