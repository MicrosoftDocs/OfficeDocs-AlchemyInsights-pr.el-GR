# <a name="trying-to-verify-your-domain"></a>Προσπάθεια για να επαληθεύσετε τον τομέα σας;

Πρέπει να προσθέσετε μια εγγραφή TXT σας registrar τομέα για να βεβαιωθείτε ότι έχετε στην κατοχή σας τομέα. Ακολουθήστε τον "Οδηγό" Προσθήκη τομέα κάνοντας κλικ στην επιλογή **Ρυθμίσεις** \> **τομέα**και, στη συνέχεια, **Προσθήκη τομέα**. 
  
Ακολουθήστε τις οδηγίες του άρθρου παρακάτω, για να προσθέσετε μια εγγραφή TXT για να επαληθεύσετε τον τομέα σας. Επιλέξτε σας registrar τομέα για συγκεκριμένες οδηγίες για τον τομέα σας.
  
[Δημιουργία εγγραφών DNS για Office 365](https://support.office.com/article/https://support.office.com/article/Create-DNS-records-for-Office-365-when-you-manage-your-DNS-records-B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23.aspx)
