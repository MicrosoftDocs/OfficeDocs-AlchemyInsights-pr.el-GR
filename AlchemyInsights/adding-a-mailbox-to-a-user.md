# <a name="adding-a-mailbox-to-a-user"></a>Προσθήκη γραμματοκιβωτίου σε χρήστη

Για να προσθέσετε ένα γραμματοκιβώτιο σε ένα χρήστη που έχει ήδη δημιουργηθεί ή σε ένα νέο χρήστη, ο χρήστης πρέπει να εκχωρηθεί μια άδεια χρήσης για το Exchange Online. Εάν ο χρήστης έχει ήδη εκχωρηθεί άδεια χρήσης, επιλέξτε Exchange Online στην ενότητα Εφαρμογές στη σελίδα άδειας χρήσης για να ξεκινήσετε τη διαδικασία δημιουργίας γραμματοκιβωτίου για το χρήστη. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εκχώρηση αδειών χρήσης σε χρήστες](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).
