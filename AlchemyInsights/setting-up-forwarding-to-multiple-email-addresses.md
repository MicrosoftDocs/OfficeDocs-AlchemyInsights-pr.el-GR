# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Ρύθμιση προώθησης σε πολλές διευθύνσεις ηλεκτρονικού ταχυδρομείου

Εάν θέλετε να ορίσετε ηλεκτρονικού ταχυδρομείου προώθησης σε μια ομάδα χρηστών, Δημιουργία λίστας διανομής, προσθέσετε πολλούς χρήστες και, στη συνέχεια, ρυθμίστε τις παραμέτρους προώθησης σε αυτήν τη λίστα. Πρώτα, μεταβείτε στη [Διαχείριση ομάδων](https://portal.office.com/adminportal/home#/groups)
  
1. Δημιουργία μιας *λίστας διανομής*

2. Προσθήκη πολλών χρηστών

3. Στη συνέχεια, ρυθμίστε τις παραμέτρους γραμματοκιβώτιο προώθησης σε αυτήν την ομάδα
