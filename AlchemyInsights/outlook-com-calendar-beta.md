# <a name="new-calendar-experiences-coming-to-outlookcom"></a>Νέες εμπειρίες ημερολογίου που έρχονται στο Outlook.com

Η νέα εμφάνιση φέρνει τα καλύτερα σχέδια και τις δυνατότητες εκμάθησης από την εμπειρία αλληλογραφίας στο ημερολόγιο.

Θα απολαύσετε ένα μοντέρνο σχέδιο που μειώνει την οπτική ακαταστασία και σας παρέχει γρηγορότερους τρόπους για να περιηγηθείτε στο ημερολόγιό σας. Είναι επίσης πιο εύκολο να βρίσκετε ενδιαφέροντα ημερολόγια για τις αγαπημένες σας δραστηριότητες — από ΤΗΛΕΟΠΤΙΚΈς εκπομπές μέχρι αθλητικές ομάδες και εκδηλώσεις.

Διευρύναμε τη συλλογή των εικονιδίων συμβάντων και ενημερώσαμε την εμφάνιση και την αίσθηση των υπαρχόντων — και τώρα προσθέτουμε αυτόματα τα σχετικά εικονίδια στις εκδηλώσεις σας, ώστε να μπορείτε εύκολα να δείτε τι συμβαίνει με μια ματιά.

Μπορείτε να παρέχετε σχόλια για νέες δυνατότητες μέσω του [καναλιού Outlook.com Calendar UserVoice](https://go.microsoft.com/fwlink/?linkid=2103075).
