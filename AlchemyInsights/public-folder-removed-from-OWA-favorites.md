# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Δημόσιος φάκελος που καταργήθηκε από Outlook αγαπημένα στο web (OWA)

Υπάρχει ένα γνωστό πρόβλημα με την κατάργηση δημόσιων φακέλων από τα αγαπημένα στο OWA σε κάθε αποσύνδεση ή ανανέωση οθόνης. Το πρόβλημα θα διορθωθεί σύντομα. Μέχρι τότε, προσθέστε εκ νέο το φάκελο για πρόσβαση από το OWA.
