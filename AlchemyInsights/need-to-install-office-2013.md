# <a name="office-2013-is-no-longer-supported-in-microsoft-365-subscriptions"></a>Το Office 2013 δεν υποστηρίζεται πλέον σε συνδρομές Microsoft 365

Από τις 28 Φεβρουαρίου 2017, το Office 2013 δεν υποστηρίζεται πλέον στις συνδρομές Microsoft 365.
  
Για να εξασφαλίσετε την ασφάλεια της οικογένειας προγραμμάτων του Office και για να λάβετε όλες τις πιο πρόσφατες δυνατότητες, εγκαταστήστε την πιο πρόσφατη έκδοση του Office δωρεάν ως μέρος της συνδρομής microsoft 365. Ανατρέξτε [στο θέμα Αναβάθμιση των χρηστών σας στο πιο πρόσφατο πρόγραμμα-πελάτη του Office](https://docs.microsoft.com/microsoft-365/admin/setup/upgrade-users-to-latest-office-client).
