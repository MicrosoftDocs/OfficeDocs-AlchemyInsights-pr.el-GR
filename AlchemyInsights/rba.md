# <a name="about-room-mailboxes"></a>Πληροφορίες για τα γραμματοκιβώτια δωματίων

Ένα γραμματοκιβώτιο δωματίου είναι ένα γραμματοκιβώτιο πόρων που έχει αντιστοιχιστεί σε μια φυσική θέση, όπως μια αίθουσα συσκέψεων, ένα αμφιθέατρο ή μια αίθουσα εκπαίδευσης. Για περισσότερες πληροφορίες σχετικά με τα γραμματοκιβώτια δωματίων, ανατρέξτε στο θέμα [Δημιουργία και διαχείριση γραμματοκιβωτίων αίθουσας](https://go.microsoft.com/fwlink/p/?linkid=717533).
