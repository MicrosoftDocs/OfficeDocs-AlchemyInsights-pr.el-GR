# <a name="about-room-mailboxes"></a>Σχετικά με τα γραμματοκιβώτια δωμάτιο

Ένα γραμματοκιβώτιο δωμάτιο είναι ένα γραμματοκιβώτιο πόρου που έχει αντιστοιχιστεί σε μια φυσική θέση, όπως μια αίθουσα διασκέψεων, ένα auditorium ή μια αίθουσα εκπαίδευσης. Για περισσότερες πληροφορίες σχετικά με τα γραμματοκιβώτια δωμάτιο, ανατρέξτε στην ενότητα [Δημιουργία και διαχείριση γραμματοκιβώτια δωμάτιο](https://go.microsoft.com/fwlink/p/?linkid=717533).
