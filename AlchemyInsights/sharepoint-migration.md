# <a name="migrate-options-to-sharepoint-online"></a>Μετεγκατάσταση επιλογών στο SharePoint Online

Υπάρχουν διαφορετικές επιλογές που είναι διαθέσιμες για τη μετεγκατάσταση περιεχομένου στο SharePoint Online, ανάλογα με το μέγεθος και την ποσότητα των αρχείων που πρέπει να μετακινήσετε, ανατρέξτε σε μια λίστα με τις επιλογές που [βρίσκονται εδώ](https://docs.microsoft.com/sharepointmigration/migrate-to-sharepoint-online).

Για περισσότερες πληροφορίες σχετικά με τη μετεγκατάσταση περιεχομένου, επισκεφθείτε τις παρακάτω συνδέσεις.

- [Εργαλείο μετεγκατάστασης του SharePoint](https://docs.microsoft.com/sharepointmigration/introducing-the-sharepoint-migration-tool)

- [Ξεκινήστε με τη διαχείριση μετεγκατάστασης](https://docs.microsoft.com/sharepointmigration/mm-get-started)

- [Ηλεκτρονική SharePoint και ταχύτητα μετεγκατάστασης ODB](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed)

- [Αποφύγετε να έχετε επιβραδύνει ή να αποκλειστεί στο SharePoint Online](https://docs.microsoft.com/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online)

- [Εργαλείο αξιολόγησης μετεγκατάστασης του SharePoint (SMAT)](https://www.microsoft.com/download/details.aspx?id=53598&amp;751be11f-ede8-5a0c-058c-2ee190a24fa6=True)

**Σημείωση**: αυτήν τη στιγμή το εργαλείο μετεγκατάστασης του SharePoint υποστηρίζει μόνο μετεγκαταστάσεις από 2010 του sharepoint και 2013. Η έκδοση 2016 ή 2019 δεν υποστηρίζονται αυτήν τη στιγμή.
