# <a name="add-or-manage-pstn-minutes-for-skype-or-teams"></a>Προσθήκη ή διαχείριση λεπτών PSTN για το Skype ή τις ομάδες

Εάν έχετε λάβει μια προειδοποιητική ειδοποίηση ότι τα λεπτά του Skype ή των ομάδων σας έχουν σχεδόν χρησιμοποιηθεί, παρακαλούμε εξετάστε τον τρόπο με τον οποίο μπορείτε να [προσθέσετε κεφάλαια και να διαχειριστείτε μονάδες επικοινωνίας για τον μισθωτή σας](https://docs.microsoft.com/microsoftteams/add-funds-and-manage-communications-credits).
