# <a name="how-to-access-outlookcom-while-traveling"></a>Πώς να αποκτήσετε πρόσβαση Outlook.com ενώ ταξιδεύετε

Για να αποκτήσετε πρόσβαση σε Outlook.com ενώ ταξιδεύετε, βεβαιωθείτε ότι οι πληροφορίες ασφαλείας του λογαριασμού Microsoft - μια εναλλακτική διεύθυνση ηλεκτρονικού ταχυδρομείου ή αριθμός τηλεφώνου όπου μπορείτε να λάβετε κωδικούς ασφαλείας - είναι ενημερωμένες. Για να μάθετε περισσότερα, ανατρέξτε στο [θέμα Πώς να αποκτήσετε πρόσβαση Outlook.com όταν ταξιδεύετε](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
