# <a name="share-notebooks-in-onenote-online-or-onenote-2016"></a>Κοινή χρήση σημειωματάρια του OneNote ηλεκτρονικά ή σε OneNote 2016

Στο OneNote Online, μπορείτε να μοιραστείτε τις σημειώσεις, είτε προσκαλείτε άλλους να προβάλετε και να κάνετε αλλαγές στο σημειωματάριό σας ή δημιουργώντας μια σύνδεση προς μια σελίδα σημειώσεων που μπορείτε να επικολλήσετε σε ένα μήνυμα ηλεκτρονικού ταχυδρομείου, ημερολογίου Web ή ιστοσελίδα. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Χρήση OneNote Online για να κάνετε κοινή χρήση σημειώσεων με άλλα άτομα](https://support.office.com/article/D3481FBE-E06C-4883-B7E9-B2EE9F38AED3)

Στο OneNote 2016, βεβαιωθείτε ότι το σημειωματάριο του OneNote 2016 βρίσκεται στο OneDrive, έτσι ώστε άλλοι χρήστες να τη λάβετε. Για κοινή χρήση, μπορείτε να προσκαλέσετε άτομα στην αλληλογραφία ή να λάβετε μια κοινόχρηστη σύνδεση. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [κοινή χρήση ενός 2016 OneNote για το Σημειωματάριο των Windows με άλλα άτομα](https://support.office.com/article/d14b6033-7a95-4536-9216-bb0a5e0f8285)
