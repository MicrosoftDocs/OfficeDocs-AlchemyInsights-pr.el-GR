# <a name="share-notebooks-in-onenote-for-the-web-or-onenote-2016"></a>Κοινή χρήση σημειωματαρίων στο OneNote για το Web ή το OneNote 2016

Στο OneNote για web, μπορείτε να κάνετε κοινή χρήση σημειώσεων είτε προσκαλώντας άλλους χρήστες να προβάλουν και να κάνουν αλλαγές στο σημειωματάριό σας είτε δημιουργώντας μια σύνδεση προς μια σελίδα σημειώσεων που μπορείτε να επικολλήσετε σε ένα μήνυμα ηλεκτρονικού ταχυδρομείου, ένα ιστολόγιο ή μια ιστοσελίδα. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Χρήση του OneNote για το web για κοινή χρήση σημειώσεων με άλλα άτομα](https://support.office.com/article/D3481FBE-E06C-4883-B7E9-B2EE9F38AED3)

Στο OneNote 2016, βεβαιωθείτε ότι το σημειωματάριο του OneNote 2016 βρίσκεται στο OneDrive, ώστε να μπορούν να φτάσουν σε άλλους χρήστες. Για κοινή χρήση, μπορείτε να προσκαλέσετε άτομα με αλληλογραφία ή να λάβετε μια σύνδεση κοινής χρήσης. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Κοινή χρήση σημειωματαρίου του OneNote 2016 για Windows με άλλα άτομα](https://support.office.com/article/d14b6033-7a95-4536-9216-bb0a5e0f8285)
