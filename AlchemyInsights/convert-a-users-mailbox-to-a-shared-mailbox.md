# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Μετατροπή ενός γραμματοκιβωτίου χρήστη σε ένα κοινόχρηστο γραμματοκιβώτιο

Κατά τη μετατροπή του γραμματοκιβωτίου ενός χρήστη σε ένα κοινόχρηστο γραμματοκιβώτιο, διατηρείται όλο το υπάρχον μήνυμα ηλεκτρονικού ταχυδρομείου και ημερολογίου. Τώρα είναι σε ένα κοινόχρηστο γραμματοκιβώτιο όπου πολλά άτομα θα μπορούν να τον προσπελάσουν αντί για ένα άτομο. Αργότερα, μπορείτε να μετατρέψετε ένα κοινόχρηστο γραμματοκιβώτιο σε ένα γραμματοκιβώτιο του χρήστη (ιδιωτικό).
