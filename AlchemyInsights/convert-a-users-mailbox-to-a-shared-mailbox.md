# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Μετατροπή γραμματοκιβωτίου χρήστη σε κοινόχρηστο γραμματοκιβώτιο

Όταν μετατρέπετε το γραμματοκιβώτιο ενός μεμονωμένου χρήστη σε κοινόχρηστο γραμματοκιβώτιο, όλα τα υπάρχοντα στοιχεία ηλεκτρονικού ταχυδρομείου και ημερολογίου διατηρούνται σε ένα κοινόχρηστο γραμματοκιβώτιο όπου μπορούν να έχουν πρόσβαση πολλά άτομα. Μπορείτε επίσης να μετατρέψετε ένα κοινόχρηστο γραμματοκιβώτιο σε ένα γραμματοκιβώτιο χρήστη (ιδιωτικό).
  
Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Μετατροπή γραμματοκιβωτίου χρήστη σε κοινόχρηστο γραμματοκιβώτιο](https://docs.microsoft.com/office365/admin/email/convert-user-mailbox-to-shared-mailbox).
