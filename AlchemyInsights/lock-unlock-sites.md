# <a name="lock-and-unlock-sites"></a>Κλείδωμα και ξεκλείδωμα τοποθεσιών

Ως καθολικός Microsoft 365 ή διαχειριστής SharePoint, μπορείτε να αποκλείσετε την πρόσβαση σε μια τοποθεσία ή να κάνετε μια τοποθεσία μόνο για ανάγνωση, χρησιμοποιώντας το Microsoft PowerShell για να αλλάξετε την κατάσταση κλειδώματος της τοποθεσίας. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Κλείδωμα και ξεκλείδωμα τοποθεσιών.](/sharepoint/manage-lock-status)
