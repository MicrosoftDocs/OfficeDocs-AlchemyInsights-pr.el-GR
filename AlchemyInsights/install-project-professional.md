# <a name="install-project-professional"></a>Εγκατάσταση επαγγελματικού έργου

Ανατρέξτε στο θέμα [εγκατάσταση έργου](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) για να μάθετε πώς να εγκαταστήσετε την έκδοση εγγραφής του Project Online και την έκδοση χωρίς εγγραφή του έργου 2019, 2016 ή 2013. 

Για το έργο 2010 ή 2007, ακολουθήστε τα βήματα στο [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) ή [στο Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Το Project πωλείται ως αυτόνομη εφαρμογή, οπότε αν δεν το έχετε ακόμα, [Αγοράστε ή δοκιμάστε το Project](https://products.office.com/project).
