# <a name="install-project-professional"></a>Εγκατάσταση του Project Professional

Ανατρέξτε στο θέμα [εγκατάσταση του Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) για να μάθετε πώς να εγκαταστήσετε την έκδοση συνδρομής του Project Online και την έκδοση του Project 2019, 2016 ή 2013 που δεν είναι συνδρομητικά. 

Για το Project 2010 ή το 2007, ακολουθήστε τα βήματα στο [office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)   ή στο [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Το Project πωλείται ως μεμονωμένη εφαρμογή, επομένως, εάν δεν το έχετε κάνει ακόμα, [Αγοράστε ή δοκιμάστε το Project](https://products.office.com/project).
