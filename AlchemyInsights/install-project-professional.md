# <a name="install-project-professional"></a>Εγκατάσταση του Project Professional

Ανατρέξτε στο θέμα "Εγκατάσταση του [Project"](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) για να μάθετε πώς μπορείτε να εγκαταστήσετε τη συνδρομητική έκδοση του Project Online και τη μη συνδρομητική έκδοση του Project 2019, 2016 ή 2013. 

Για το Project 2010 ή 2007, ακολουθήστε τα βήματα στο [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) ή [το Office 2007.](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657) 

Το Project πωλείται ως αυτόνομη εφαρμογή, επομένως, εάν δεν το έχετε ακόμη, αγοράστε [ή δοκιμάστε το Project.](https://www.microsoft.com/evalcenter/evaluate-project)
