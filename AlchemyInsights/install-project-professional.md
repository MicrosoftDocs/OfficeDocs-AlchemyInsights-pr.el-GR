# <a name="install-project-professional"></a>Εγκατάσταση του Project Professional

Ανατρέξτε στην ενότητα [Εγκατάσταση του Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281?ui=en-US&rs=en-US&ad=US) για να μάθετε πώς μπορείτε να εγκαταστήσετε την έκδοση συνδρομής του Project Online και την έκδοση χωρίς εγγραφή του έργου 2019, 2016 ή 2013. 

Για το Project 2010 ή 2007, ακολουθήστε τα βήματα του [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) ή το [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Έργο πωλείται ως ανεξάρτητη εφαρμογή, επομένως εάν δεν διαθέτετε αυτό ακόμα, [αγορά ή δοκιμάστε το έργο](https://products.office.com/project).
