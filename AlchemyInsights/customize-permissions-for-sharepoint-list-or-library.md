# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Πώς μπορείτε να προσαρμόσετε τα δικαιώματα για μια λίστα ή βιβλιοθήκη του SharePoint

Ως διαχειριστής ή κάτοχος μιας βιβλιοθήκης, λίστας ή έρευνας, μπορείτε να αλλάξετε τα δικαιώματα για να επιτρέψετε σε μία ομάδα ατόμων να αποκτήσει πρόσβαση στα δεδομένα που χρειάζονται, περιορίζοντας τους άλλους.
