# <a name="rename-your-onmicrosoftcom-domain"></a>Μετονομασία του τομέα σας onmicrosoft.com

 **Δεν μπορείτε να μετονομάσετε τον τομέα onmicrosoft μετά την εγγραφή.** Για παράδειγμα, εάν ο αρχικός τομέας που επιλέξατε ήταν  *fourthcoffee.onmicrosoft.com*, δεν μπορείτε να τον αλλάξετε ώστε να είναι  *Fabrikam.onmicrosoft.com*. Για να χρησιμοποιήσετε έναν διαφορετικό τομέα του onmicrosoft.com, θα πρέπει να ξεκινήσετε μια νέα συνδρομή με το Microsoft 365.
  
Εάν θέλετε να χρησιμοποιήσετε έναν προσαρμοσμένο τομέα όπως το  *contoso.com*  , μπορείτε να [προσθέσετε έναν δικό σας](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain) ή να [αγοράσετε έναν από τη Microsoft](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/buy-a-domain-name).
