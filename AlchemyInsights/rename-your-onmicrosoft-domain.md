# <a name="rename-your-onmicrosoftcom-domain"></a>Μετονομασία του τομέα onmicrosoft.com

 **Δεν μπορείτε να μετονομάσετε τον τομέα onmicrosoft μετά την εγγραφή.** Για παράδειγμα, εάν ο αρχικός τομέας που επιλέξατε ήταν *fourthcoffee.onmicrosoft.com*, δεν μπορείτε να τον αλλάξετε για να είναι *Fabrikam.onmicrosoft.com*. Για να χρησιμοποιήσετε έναν διαφορετικό τομέα onmicrosoft.com, θα πρέπει να ξεκινήσετε μια νέα συνδρομή με το Office 365.
  
Εάν θέλετε να χρησιμοποιήσετε έναν προσαρμοσμένο τομέα όπως το *contoso.com* , μπορείτε να [προσθέσετε ένα που σας ανήκει](https://docs.microsoft.com/office365/admin/setup/add-domain) ή να [αγοράσετε ένα από τη Microsoft](https://docs.microsoft.com/office365/admin/get-help-with-domains/buy-a-domain-name).
