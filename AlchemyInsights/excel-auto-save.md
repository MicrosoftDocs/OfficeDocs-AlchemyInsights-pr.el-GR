# <a name="what-is-autosave"></a>Τι είναι η αυτόματη αποθήκευση;

Η αυτόματη αποθήκευση είναι μια δυνατότητα που είναι διαθέσιμη στο Excel, το Word και το PowerPoint για συνδρομητές του Microsoft 365 που αποθηκεύουν αυτόματα το αρχείο σας, κάθε λίγα δευτερόλεπτα, καθώς εργάζεστε. 

Για συνήθεις ερωτήσεις, ανατρέξτε στο θέμα [Τι είναι η αυτόματη αποθήκευση;](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
