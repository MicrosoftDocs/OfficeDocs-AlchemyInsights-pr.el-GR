# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Διάγνωση και επίλυση προβλημάτων συνδεσιμότητας του Outlook

Για να διαγνώσετε και να διορθώσετε τα προβλήματα συνδεσιμότητας του Outlook, μπορείτε να εκτελέσετε το [εργαλείο συνδεσιμότητας του Outlook](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) στον υπολογιστή που επηρεάζεται.
  
Αυτό το εργαλείο θα πραγματοποιήσει διαγνωστικούς ελέγχους και θα προτείνει πιθανές λύσεις για να αντιμετωπίσετε το πρόβλημα συνδεσιμότητας του Outlook.
