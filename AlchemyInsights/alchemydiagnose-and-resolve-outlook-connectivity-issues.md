# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Διάγνωση και επίλυση ζητημάτων συνδεσιμότητας του Outlook

Για να διαγνώσετε και να διορθώσετε τα ζητήματα σύνδεσης του Outlook, μπορείτε να εκτελέσετε το [εργαλείο σύνδεσης του Outlook](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) στον υπολογιστή που επηρεάζεται.
  
Αυτό το εργαλείο θα εκτελέσει διαγνωστικούς ελέγχους και θα σας προτείνει πιθανές λύσεις για την αντιμετώπιση του ζητήματος σύνδεσης του Outlook.
