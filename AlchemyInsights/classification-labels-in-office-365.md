# <a name="classification-labels"></a>Ετικέτες ταξινόμησης

Με τις ετικέτες ευαισθησίας ή ταξινόμησης, μπορείτε να ταξινομήσετε και να βοηθήσετε στην προστασία του ευαίσθητου περιεχομένου σας, διασφαλίζοντας παράλληλα ότι η παραγωγικότητα και η ικανότητα συνεργασίας των χρηστών σας δεν παρεμποδίζεται.

Για περισσότερες πληροφορίες, ανατρέξτε στο ακόλουθο [άρθρο](https://docs.microsoft.com/office365/securitycompliance/sensitivity-labels).
