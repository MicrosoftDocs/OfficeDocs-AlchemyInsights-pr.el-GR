# <a name="classification-labels"></a>Ετικέτες ταξινόμησης

Με ετικέτες ευαισθησίας ή ταξινόμησης, μπορείτε να ταξινομήσετε και να προστατεύσετε το ευαίσθητο περιεχόμενό σας, ενώ παράλληλα θα εξασφαλίσετε ότι η παραγωγικότητα και η δυνατότητα συνεργασίας των ατόμων σας δεν παρακωλύεται.

Για περισσότερες πληροφορίες, διαβάστε το παρακάτω [άρθρο](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels).
