# <a name="change-the-account-type-that-you-sign-in-with"></a>Αλλαγή του τύπου λογαριασμού με τον οποίο συνδέεστε

Δείτε πώς μπορείτε να αλλάξετε τον τύπο του λογαριασμού που χρησιμοποιείτε για να συνδεθείτε στα Windows 10:

1. Επιλέξτε **Έναρξη**  >  **λογαριασμών για τις**  >  **Accounts**  >  **πληροφορίες σας**.
2. Εάν χρησιμοποιείτε έναν τοπικό λογαριασμό που δεν είναι συνδεδεμένος με μια διεύθυνση ηλεκτρονικού ταχυδρομείου, αλλά θέλετε να χρησιμοποιήσετε ένα λογαριασμό Microsoft, επιλέξτε **Είσοδος με λογαριασμό Microsoft αντ ' αυτού**.
3. Εάν χρησιμοποιείτε ένα λογαριασμό Microsoft που είναι συνδεδεμένος με μια διεύθυνση ηλεκτρονικού ταχυδρομείου, αλλά θέλετε να χρησιμοποιήσετε έναν τοπικό λογαριασμό, επιλέξτε **Είσοδος με έναν τοπικό λογαριασμό αντ ' αυτού**.
4. Εάν είναι απαραίτητο, πληκτρολογήστε τον κωδικό πρόσβασής σας και, στη συνέχεια, επιλέξτε **Επόμενο**.
5. Επανεκκινήστε τη συσκευή σας. Την επόμενη φορά που θα πραγματοποιήσετε είσοδο, θα εμφανιστεί ο λογαριασμός που αλλάξατε.
