# <a name="change-the-account-type-that-you-sign-in-with"></a>Αλλαγή του τύπου λογαριασμού με τον οποίο συνδέεστε

Δείτε πώς μπορείτε να αλλάξετε τον τύπο του λογαριασμού που χρησιμοποιείτε για να εισέλθετε στα Windows 10:

1. Επιλέξτε **Έναρξη** > **Ρυθμίσεις** > **** λογαριασμούς > **τις πληροφορίες σας**.
2. Αν χρησιμοποιείτε έναν τοπικό λογαριασμό που δεν είναι συνδεδεμένος με μια διεύθυνση ηλεκτρονικού ταχυδρομείου αλλά θέλετε να χρησιμοποιήσετε ένα λογαριασμό Microsoft, επιλέξτε **σύνδεση με λογαριασμό Microsoft**.
3. Εάν χρησιμοποιείτε ένα λογαριασμό Microsoft που είναι συνδεδεμένος με μια διεύθυνση ηλεκτρονικού ταχυδρομείου, αλλά θέλετε να χρησιμοποιήσετε έναν τοπικό λογαριασμό, επιλέξτε **σύνδεση με έναν τοπικό λογαριασμό**.
4. Εάν είναι απαραίτητο, πληκτρολογήστε τον κωδικό πρόσβασής σας και, στη συνέχεια, επιλέξτε **Επόμενο**.
5. Επανεκκινήστε τη συσκευή σας. Την επόμενη φορά που θα εισέλθετε, θα εμφανιστεί ο λογαριασμός που αλλάξατε.
