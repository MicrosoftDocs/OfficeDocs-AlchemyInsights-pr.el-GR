# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Λαμβάνετε ένα σφάλμα δικαιωμάτων κατά το άνοιγμα ενός κοινόχρηστου γραμματοκιβωτίου στο Outlook στο Web;

Εάν λάβετε το σφάλμα **, κάτι δεν πήγε καλά. Δεν έχετε δικαίωμα ανοίγματος αυτού του γραμματοκιβωτίου.** Όταν προσπαθείτε να ανοίξετε ένα κοινόχρηστο γραμματοκιβώτιο στο Outlook στο Web, ο λογαριασμός σας μπορεί να μην έχει δικαίωμα πρόσβασης στο κοινόχρηστο γραμματοκιβώτιο. Ο διαχειριστής του οργανισμού σας μπορεί να εκχωρήσει τα δικαιώματα του λογαριασμού σας στο κοινόχρηστο γραμματοκιβώτιο, ακολουθώντας τις οδηγίες στο θέμα [Δημιουργία κοινόχρηστου γραμματοκιβωτίου](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox).
