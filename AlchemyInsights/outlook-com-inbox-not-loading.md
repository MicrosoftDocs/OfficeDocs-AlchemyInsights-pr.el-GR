# <a name="inbox-not-loading-with-firefox"></a>Εισερχόμενα δεν πραγματοποιεί φόρτωση με Firefox

Ορισμένα πρόσθετα στο πρόγραμμα περιήγησης Firefox μπορεί να εμποδίσει τα Εισερχόμενά σας Outlook.com φόρτωσης.
  
Επισκεφθείτε το Firefox υποστήριξης για να μάθετε πώς μπορείτε να [απενεργοποιήσετε ή να καταργήσετε πρόσθετα](https://support.mozilla.org/kb/disable-or-remove-add-ons).
