# <a name="microsoft-365-language-settings"></a>Ρυθμίσεις γλώσσας του Microsoft 365

Προσδιορίστε εάν θέλετε να αλλάξετε τη γλώσσα εμφάνισης και άλλες ρυθμίσεις για τους χρήστες ή για ολόκληρο το μισθωτή.

- Για τους χρήστες, ανατρέξτε [στο θέμα Αλλαγή της γλώσσας εμφάνισης και της ζώνης ώρας στο Microsoft 365 για όλες τις εφαρμογές](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b).
- Για τον μισθωτή, ανατρέξτε  [στο θέμα καθορισμός των ρυθμίσεων γλώσσας και τοπικών ρυθμίσεων για το Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region).
