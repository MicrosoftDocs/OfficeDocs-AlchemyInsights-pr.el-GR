# <a name="why-cant-i-edit-files"></a>Γιατί δεν μπορώ να επεξεργαστώ αρχεία;

Ρίξτε μια γρήγορη ματιά σε κάποιους από τους λόγους για τους [οποίους ενδέχεται να μην μπορείτε να επεξεργαστείτε το αρχείο](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).

Εάν έχετε βιβλιοθήκες με **ολοκλήρωση αγορών, απαιτούμενες** ή **στήλες επικύρωσης** ή μετα-δεδομένα ή όταν η **ασφάλεια του στοιχείου Προχείρου** ορίζεται είτε **μόνο στους χρήστες που μπορούν να επεξεργαστούν** είτε **μόνο τους χρήστες που μπορούν να εγκρίνουν στοιχεία** στις ρυθμίσεις έκδοσης της βιβλιοθήκης:

- Για τις βιβλιοθήκες εγγράφων του SharePoint, αυτά τα στοιχεία θα συγχρονιστούν ως μόνο για ανάγνωση.

- Για τις βιβλιοθήκες του OneDrive, το OneDrive θα εμφανίσει ένα εικονίδιο κλειδώματος (εικονίδιο κλειδωμένου αρχείου του OneDrive) δίπλα στην κατάσταση συγχρονισμού και ο χρήστης δεν θα έχει τη δυνατότητα να συγχρονίσει τη βιβλιοθήκη μέχρι να καταργηθούν οι ρυθμίσεις που αναφέρθηκαν. 

Για περισσότερες πληροφορίες σχετικά με τα κλειδωμένα αρχεία για το Word ή το Excel, ανατρέξτε στα παρακάτω άρθρα.

- [Έλαβα ένα μήνυμα σφάλματος "η αποστολή απέτυχε" ή "δεν ήταν δυνατή η αυτόματη αποθήκευση" κατά την επεξεργασία ενός αρχείου του Office](https://support.office.com/article/i-got-an-upload-failed-or-couldn-t-save-automatically-error-while-editing-an-office-file-93a14d34-88e3-4a91-9eef-58cc541d31f8)
