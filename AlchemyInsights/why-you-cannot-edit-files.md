# <a name="why-cant-i-edit-files"></a>Γιατί δεν μπορώ να επεξεργαστώ αρχεία;

Ρίξτε μια γρήγορη ματιά σε ορισμένους από τους λόγους για τους [οποίους ενδέχεται να μην μπορείτε να επεξεργαστείτε το αρχείο](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).

Εάν έχετε βιβλιοθήκες με έλεγχο **, απαιτούμενες** ή στήλες **επικύρωσης** ή μετα-δεδομένα ή όταν η **ασφάλεια πρόχειρου στοιχείου** ορίζεται είτε **μόνο στους χρήστες που μπορούν να επεξεργαστούν** είτε **μόνο οι χρήστες που μπορούν να εγκρίνουν στοιχεία** στις ρυθμίσεις έκδοσης της βιβλιοθήκης:

- Για βιβλιοθήκες εγγράφων του SharePoint, αυτά τα στοιχεία θα συγχρονιστούν ως μόνο για ανάγνωση.

- Για βιβλιοθήκες του OneDrive, το OneDrive θα εμφανίσει ένα εικονίδιο κλειδώματος (εικονίδιο κλειδωμένου αρχείου OneDrive) δίπλα στην κατάσταση συγχρονισμού και ο χρήστης δεν θα έχει τη δυνατότητα να συγχρονίσει τη βιβλιοθήκη μέχρι να καταργηθούν οι ρυθμίσεις που αναφέρθηκαν. 

Για περισσότερες πληροφορίες σχετικά με τα κλειδωμένα αρχεία για το Word ή το Excel, παρακαλείσθε να αναφέρετε τα παρακάτω άρθρα.

- [Το αρχείο Excel είναι κλειδωμένο για επεξεργασία από άλλο χρήστη](https://support.office.com/article/Excel-file-is-locked-for-editing-by-another-user-6fa93887-2c2c-45f0-abcc-31b04aed68b3)

- [Το αρχείο του Word είναι κλειδωμένο για επεξεργασία από άλλο χρήστη](https://support.microsoft.com/help/313472/the-document-is-locked-for-editing-by-another-user-error-message-when)
