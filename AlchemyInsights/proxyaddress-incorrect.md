# <a name="proxyaddress-incorrect"></a>ProxyAddress εσφαλμένη

Όταν ένα αντικείμενο συγχρονίζεται με το Azure AD, οι τιμές που καθορίζονται στο χαρακτηριστικό proxyAddresses της υπηρεσίας καταλόγου Active Directory συγκρίνονται με τους κανόνες AD του Azure και, στη συνέχεια, το χαρακτηριστικό proxyAddresses συμπληρώνεται στο Azure AD. Επομένως, οι τιμές του χαρακτηριστικού proxyAddresses για το αντικείμενο στην υπηρεσία καταλόγου Active Directory μπορεί να μην είναι ίδιες με τις τιμές του χαρακτηριστικού proxyAddresses στο Azure AD.
  
Για να μάθετε περισσότερα σχετικά με τον τρόπο με τον οποίο συμπληρώνεται το proxyAddress, ανατρέξτε στο θέμα [Πώς συμπληρώνεται το χαρακτηριστικό proxyAddress στο Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
