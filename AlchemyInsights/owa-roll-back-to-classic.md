# <a name="the-new-outlook-on-the-web"></a>Το νέο του Outlook στο web

Δεν είναι πλέον δυνατό για να επιστρέψετε την κλασική Outlook στην εμπειρία web. Εάν θέλετε να κάνω σχόλια σχετικά με τη νέα εμπειρία, μεταβείτε στο [φόρουμ UserVoice](https://outlook.uservoice.com/forums/313228--outlook-on-the-web-office-365). Μπορείτε να κάνετε επιλογή τις ιδέες ή τα ζητήματα που θα θέλατε να δείτε αντιμετωπίζονται και σας θα απαντήσουμε απευθείας προς τα επάνω τα σχόλια και τις προτάσεις.

[Λήψη Βοήθειας με το νέο του Outlook στο web](https://support.office.com/article/017014cd-2ad0-41ab-8473-6bd8c349d4f8)
