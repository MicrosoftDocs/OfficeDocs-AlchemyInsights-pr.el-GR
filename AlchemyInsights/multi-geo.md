# <a name="microsoft-365-multi-geo"></a>Microsoft 365 Multi-Geo

Με το Microsoft 365 Multi-Geo, ο οργανισμός σας μπορεί να επεκτείνει την παρουσία του Microsoft 365 σε πολλές γεωγραφικές περιοχές ή/και χώρες εντός του υπάρχοντος μισθωτή σας. Για περισσότερες πληροφορίες, ανατρέξτε στις παρακάτω συνδέσεις.

- [Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [Πρόγραμμα για το Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Ρύθμιση παραμέτρων Multi-Geo](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
