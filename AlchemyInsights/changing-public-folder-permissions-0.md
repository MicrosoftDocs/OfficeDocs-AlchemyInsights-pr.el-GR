# <a name="changing-public-folder-permissions"></a>Αλλαγή των δικαιωμάτων του δημόσιου φακέλου

Δικαιώματα δημόσιου φακέλου μπορεί να αλλάξει από τους χρήστες και διαχειριστές στο Outlook. Οι διαχειριστές μπορούν επίσης να ελέγχετε δικαιώματα από το Exchange Admin κέντρο (ΕΚΟ), κάνοντας τα εξής:
  
1. Κέντρο διαχείρισης στο το Microsoft 365, μεταβείτε στο **Κέντρο διαχείρισης** \> **Exchange**.

2. Επιλέξτε **τους δημόσιους φακέλους**.

3. Από εκεί, μπορείτε να αλλάξετε τα δικαιώματα για μεμονωμένους δημόσιους φακέλους, αντιστοιχίζοντας τις ομάδες ασφαλείας στα δικαιώματα. Για τον τελικό χρήστη για να αλλάξετε τα δικαιώματα σε δημόσιους φακέλους, ο χρήστης πρέπει να έχετε δικαιώματα κατόχου στο φάκελο.
