# <a name="move-a-phone-number-from-another-provider-to-microsoft"></a>Μετακίνηση αριθμού τηλεφώνου από άλλη υπηρεσία παροχής στη Microsoft

Χρειάζεται να μετακινήσετε έναν αριθμό τηλεφώνου από άλλη υπηρεσία παροχής στη Microsoft; Μπορείτε να βρείτε αριθμούς τηλεφώνου για διαφορετικές χώρες και περιοχές στην [ενότητα Διαχείριση αριθμών τηλεφώνου για τον οργανισμό σας](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization). Επιλέξτε τη χώρα ή την περιοχή σας για να βρείτε πληροφορίες σχετικά με τη μεταφορά αριθμών.
