# <a name="microsoft-viva-insights-availability"></a>Διαθεσιμότητα του Microsoft Viva Insights

Το Microsoft Viva Insights είναι διαθέσιμο τώρα σε προεπισκόπηση.

[Λήψη](https://aka.ms/InsightsDocumentation) Viva Insights στο Microsoft Teams.

[Μάθετε περισσότερα σχετικά](https://aka.ms/VivaInsights) με τον τρόπο με τον οποίο το Viva Insights βοηθά στη δημιουργία μιας κουλτούρας όπου οι άνθρωποι και οι επιχειρήσεις μπορούν να ευδοκιμούν.

Για περισσότερες πληροφορίες, δείτε πώς το Microsoft Viva Insights βοηθά τους [χρήστες να καλλιεργούν την ευεξία και να είναι οι καλύτεροι.](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-insights-helps-people-nurture-wellbeing-and-be/ba-p/2107010)
