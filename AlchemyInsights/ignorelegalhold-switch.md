# <a name="ignorelegalhold-switch"></a>**– Διακόπτης IgnoreLegalhold**

Η **– IgnoreLegalhold** που χρησιμοποιείται στο παρακάτω commandlet δεν είναι διαθέσιμη ή λειτουργική για αντικείμενα χρήστη του Exchange Online.

- Απενεργοποίηση-γραμματοκιβώτιο
- Κατάργηση-γραμματοκιβώτιο
- Απενεργοποίηση-MailUser
- Κατάργηση-MailUser
