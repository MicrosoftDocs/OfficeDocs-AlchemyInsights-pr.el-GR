# <a name="change-your-email-address"></a>Αλλαγή της διεύθυνσης ηλεκτρονικού ταχυδρομείου

Για να αλλάξετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου σας, θα πρέπει να προσθέσετε ένα άλλο και να είναι το ψευδώνυμο του πρωτεύοντος ηλεκτρονικού ταχυδρομείου σας.
  
Ακολουθήστε τις οδηγίες που εμφανίζονται στην [Προσθήκη ή κατάργηση ενός ψευδωνύμου ηλεκτρονικού ταχυδρομείου στο Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873115).
