# <a name="change-your-email-address"></a>Αλλαγή της διεύθυνσης ηλεκτρονικού ταχυδρομείου σας

Για να αλλάξετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου Outlook.com, θα πρέπει να προσθέσετε μια άλλη και να την κάνετε το κύριο ψευδώνυμο ηλεκτρονικού ταχυδρομείου σας.
  
Ακολουθήστε τις οδηγίες στο θέμα [Προσθήκη ή κατάργηση ψευδωνύμου ηλεκτρονικού ταχυδρομείου στο Outlook.com](https://support.office.com/article/459b1989-356d-40fa-a689-8f285b13f1f2?wt.mc_id=Office_Outlook_com_Alchemy).
