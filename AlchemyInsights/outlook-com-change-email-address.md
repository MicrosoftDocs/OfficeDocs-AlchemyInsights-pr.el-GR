# <a name="change-your-email-address"></a>Αλλαγή της διεύθυνσης ηλεκτρονικού ταχυδρομείου

Για να αλλάξετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου σας Outlook.com, πρέπει να προσθέσετε ένα άλλο και να είναι το ψευδώνυμο του πρωτεύοντος ηλεκτρονικού ταχυδρομείου σας.
  
Ακολουθήστε τις οδηγίες που εμφανίζονται στην [Προσθήκη ή κατάργηση ενός ψευδωνύμου ηλεκτρονικού ταχυδρομείου στο Outlook.com](https://support.office.com/article/459b1989-356d-40fa-a689-8f285b13f1f2?wt.mc_id=Office_Outlook_com_Alchemy).
