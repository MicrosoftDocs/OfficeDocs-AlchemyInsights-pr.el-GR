# <a name="turn-off-focused-inbox"></a>Απενεργοποιήστε την επιλογή "Εισερχόμενα" εστιάζει

1. Στην κορυφή της σελίδας, επιλέξτε " **Ρυθμίσεις**".
2. Στο παράθυρο ρυθμίσεις, μετακινήστε την εναλλαγή **Εστιάζει Εισερχόμενα** απενεργοποιημένη.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Απενεργοποίηση εστιάζει στο φάκελο Εισερχόμενα](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2).
