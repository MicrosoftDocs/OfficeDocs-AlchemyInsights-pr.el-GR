# <a name="in-app-support-in-outlook-for-mac"></a>Υποστήριξη-app στο Outlook για Mac

Για να λάβετε υποστήριξη σε-app για το Outlook για Mac θέματα, ανοίξτε το Outlook και κάντε κλικ στο κουμπί Βοήθεια \> υποστηρίζει την επαφή. Πληκτρολογήστε την ερώτησή σας και ένα μέλος του Outlook για Mac ομάδα υποστήριξης θα επικοινωνήσουμε μαζί σας. [Επικοινωνήστε με την υποστήριξη του Outlook για Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517)
