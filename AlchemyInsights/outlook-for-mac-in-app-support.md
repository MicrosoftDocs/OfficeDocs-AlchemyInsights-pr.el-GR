# <a name="in-app-support-in-outlook-for-mac"></a>Υποστήριξη εντός εφαρμογής στο Outlook για Mac

Για να λάβετε υποστήριξη εντός εφαρμογής για θέματα του Outlook για Mac, ανοίξτε το Outlook και κάντε κλικ στο κουμπί **Βοήθεια** \> **επικοινωνήστε με την υποστήριξη**. Πληκτρολογήστε την ερώτησή σας και ένα μέλος της ομάδας υποστήριξης του Outlook για Mac θα επικοινωνήσει μαζί σας. 

Μάθετε περισσότερα: [επικοινωνήστε με την υποστήριξη στο Outlook για Mac](https://support.office.com//article/d0410177-8e65-4487-93f7-206a3a3d71a8)
