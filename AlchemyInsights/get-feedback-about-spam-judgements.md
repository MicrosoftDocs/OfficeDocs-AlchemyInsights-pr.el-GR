# <a name="get-feedback-about-spam-judgments"></a>Λάβετε σχόλια σχετικά με τις κρίσεις spam

Ως διαχειριστής, μπορείτε εύκολα να αναφέρετε ύποπτα μηνύματα ηλεκτρονικού ταχυδρομείου, διευθύνσεις URL και περιεχόμενο στη Microsoft στην [Εξερεύνηση υποβολών](https://protection.office.com/reportsubmission). Μπορείτε ακόμη να το χρησιμοποιήσετε για να κατανοήσετε γιατί ένα νόμιμο μήνυμα δεν έχει επισημανθεί σωστά. Για περισσότερες πληροφορίες, ανατρέξτε στην παρούσα [ανακοίνωση](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).

Για να γίνει πιο εύκολη, μπορείτε να επιτρέψετε στους χρήστες σας να αναφέρουν απευθείας στη Microsoft μέσω του [προσθέτου "Μήνυμα αναφοράς"](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
