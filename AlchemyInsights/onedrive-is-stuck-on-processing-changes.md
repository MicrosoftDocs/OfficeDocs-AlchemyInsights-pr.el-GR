# <a name="onedrive-is-stuck-on-processing-changes"></a>Το OneDrive έχει κολλήσει στην επεξεργασία αλλαγών

Εάν το OneDrive εμφανίσει "επεξεργασία αλλαγών" για μεγάλο χρονικό διάστημα, αυτό μπορεί να συμβαίνει επειδή έχετε ανοιχτό ένα αρχείο online, πολλά αρχεία που έχουν κολλήσει σε μια ουρά ή ένα πολύ μεγάλο αρχείο συγχρονισμού. [Η ανάγνωση του OneDrive έχει κολλήσει στην "επεξεργασία αλλαγών"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) για Προτεινόμενα βήματα και περισσότερες πληροφορίες.

Εάν χρειάζεστε περισσότερη βοήθεια, μπορείτε να  [στείλετε με ηλεκτρονικό ταχυδρομείο την υποστήριξη του OneDrive](https://go.microsoft.com/fwlink/p/?LinkId=528676).
