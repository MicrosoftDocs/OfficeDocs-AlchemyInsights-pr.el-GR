# <a name="how-to-reactivate-a-subscription"></a>Επανενεργοποίηση μιας συνδρομής

Εάν η συνδρομή σας έχει λήξει, έχει ακυρωθεί ή έχει απενεργοποιηθεί από τη Microsoft, μπορείτε να την ενεργοποιήσετε εκ νέου στο Κέντρο διαχείρισης Microsoft 365.
  
1. Στο κέντρο διαχείρισης, μεταβείτε στη σελίδα **Χρέωση** \> [Προϊόντα και υπηρεσίες](https://go.microsoft.com/fwlink/p/?linkid=842054).

2. Στην περιοχή **Χρέωση**, δίπλα στα στοιχεία **Ανανεώνεται στις** ή **Λήγει στις**, επιλέξτε **Επεξεργασία**.

3. Στο τμήμα παραθύρου **Ρυθμίσεις επαναλαμβανόμενης χρέωσης**, κάντε κλικ στην επιλογή που θέλετε. Εάν η επαναλαμβανόμενη χρέωση είναι ήδη απενεργοποιημένη, επιλέξτε **Μηνιαία** ή **Μία φορά** για να την ενεργοποιήσετε.

4. Επιλέξτε **Υποβολή αλλαγής**.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα: [Ανανέωση του Office 365 για Επιχειρήσεις](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/renew-your-subscription).
