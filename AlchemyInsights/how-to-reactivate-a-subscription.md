# <a name="how-to-reactivate-a-subscription"></a>Πώς να ενεργοποιήσετε εκ νέου μια συνδρομή

Εάν η συνδρομή σας έχει λήξει, έχει ακυρωθεί, ή απενεργοποιηθεί από τη Microsoft, μπορείτε να την επανενεργοποιήσετε στο στο Κέντρο διαχείρισης του Office 365.
  
1. Στο Κέντρο διαχείρισης, μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**.
    
2. Επιλέξτε την εγγραφή που θέλετε να επανενεργοποιήσετε.
    
3. Επιλέξτε το κουμπί **Επανενεργοποίηση** στη δεξιά πλευρά. 
    
4. Θα σας ζητηθεί να ενημερώσετε τα στοιχεία πληρωμής. Όταν όλες οι πληροφορίες είναι σωστές, επιλέξτε " **Υποβολή**".
    
5. Επιβεβαίωση θα εμφανιστούν στο κάτω μέρος της οθόνης. Επιλογή **Close** για να επιστρέψετε στη σελίδα " **εγγραφές** ". 
    
6. Εάν η εγγραφή δεν εμφανίζεται ενεργή μετά από μερικά λεπτά, ανανεώστε τη σελίδα.
    
Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα: [Ανανέωση του Office 365 για επιχειρήσεις](https://support.office.com/article/8d83b530-f4ca-47f6-a666-e5791cbacc7e)
