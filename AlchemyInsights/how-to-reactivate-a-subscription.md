# <a name="how-to-reactivate-a-subscription"></a>Επανενεργοποίηση μιας συνδρομής

Εάν η συνδρομή σας έληξε, έχει ακυρωθεί ή έχει απενεργοποιηθεί από τη Microsoft, μπορείτε να την ενεργοποιήσετε εκ νέου στο Κέντρο διαχείρισης Microsoft 365.
  
1. Στο Κέντρο διαχείρισης, μεταβείτε στη σελίδα [Συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) ή επιλέξτε **Χρέωση** \> **Συνδρομές**.
    
2. Επιλέξτε τη συνδρομή την οποία θέλετε να ενεργοποιήσετε ξανά.
    
3. Επιλέξτε το κουμπί **Επανενεργοποίηση** στη δεξιά πλευρά. 
    
4. Θα σας ζητηθεί να ενημερώσετε τα στοιχεία πληρωμής. Όταν όλες οι πληροφορίες είναι σωστές, επιλέξτε **Υποβολή**.
    
5. Ένα μήνυμα επιβεβαίωσης θα εμφανιστεί στο κάτω μέρος της οθόνης. Επιλέξτε **Κλείσιμο** για να επιστρέψετε στη σελίδα **Συνδρομές**. 
    
6. Εάν η συνδρομή δεν εμφανιστεί ως "Ενεργή" μετά από μερικά λεπτά, ανανεώστε τη σελίδα.
    
Για περισσότερες πληροφορίες, ανατρέξτε στα θέματα: [Ανανέωση του Office 365 για Επιχειρήσεις](https://support.office.com/article/8d83b530-f4ca-47f6-a666-e5791cbacc7e).
