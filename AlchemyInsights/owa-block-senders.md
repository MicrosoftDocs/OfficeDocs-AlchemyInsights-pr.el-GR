# <a name="block-or-unblock-senders"></a>Αποκλεισμός ή κατάργηση αποκλεισμού αποστολέων

**Από τα Εισερχόμενά σας:**

- Στη λίστα μηνυμάτων, επιλέξτε το μήνυμα και στη συνέχεια επιλέξτε **ανεπιθύμητης** > **μπλοκ**.

**Από τις ρυθμίσεις:**

1. Στην κορυφή της σελίδας, επιλέξτε το εικονίδιο του πηδαλίου για να ανοίξετε τις ρυθμίσεις και στη συνέχεια επιλέξτε να **προβάλετε όλες τις ρυθμίσεις Outlook**.
2. Επιλέξτε **Αλληλογραφία** > **ανεπιθύμητης αλληλογραφίας ηλεκτρονικού ταχυδρομείου**.
3. **Αποκλεισμένοι αποστολείς και τομείς**, επιλέξτε " **Προσθήκη**".
4. Πληκτρολογήστε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του τομέα που θέλετε να αποκλείσετε και πατήστε το πλήκτρο Enter.
5. Επιλέξτε **Αποθήκευση**.

Για να καταργήσετε τον αποκλεισμό ενός αποστολέα, επιλέξτε το εικονίδιο του κάδου απορριμμάτων δίπλα από τη διεύθυνση ηλεκτρονικού ταχυδρομείου ή τον τομέα που θέλετε να καταργήσετε τον αποκλεισμό.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [αποκλεισμός αποστολείς ή κατάργηση αποκλεισμού αποστολέων του Outlook στο web](https://support.office.com/article/9bf812d4-6995-4d19-901a-76d6e26939b0).
