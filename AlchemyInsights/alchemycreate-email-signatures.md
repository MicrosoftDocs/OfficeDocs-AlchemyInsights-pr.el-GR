# <a name="create-email-signatures"></a>Δημιουργία υπογραφών ηλεκτρονικού ταχυδρομείου

Αυτά τα θέματα θα σας βοηθήσουν να δημιουργήσετε υπογραφές ηλεκτρονικού ταχυδρομείου:
  
- Δημιουργήστε μια υπογραφή ηλεκτρονικού ταχυδρομείου για τα δικά σας μηνύματα: [Δημιουργήστε και προσθέστε μια υπογραφή σε μηνύματα](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) ή [Δημιουργήστε και προσθέστε μια υπογραφή ηλεκτρονικού ταχυδρομείου στο Outlook Web App](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx).
    
- Δημιουργήστε μια καθολική υπογραφή ηλεκτρονικού ταχυδρομείου για όλα τα μηνύματα που αποστέλλονται από την εταιρεία σας (γνωστή και ως αποποίηση ευθυνών): [Αποποίηση ευθυνών σε ολόκληρο τον οργανισμό, υπογραφές, υποσέλιδα ή κεφαλίδες στο Office 365](https://go.microsoft.com/fwlink/p/?linkid=391096).
