# <a name="security-defaults"></a>Προεπιλογές ασφαλείας

Για πληροφορίες σχετικά με τις προεπιλογές ασφαλείας, ανατρέξτε στο θέμα [Τι είναι οι προεπιλογές ασφαλείας;](https://docs.microsoft.com/azure/active-directory/conditional-access/concept-conditional-access-security-defaults).

Οι προεπιλογές ασφαλείας στο Azure Active Directory (Azure AD) διευκολύνουν την ασφάλειά σας και συμβάλλουν στην προστασία του οργανισμού σας. Οι προεπιλογές ασφαλείας περιέχουν προκαθορισμένες ρυθμίσεις ασφαλείας για συνηθισμένες επιθέσεις.

Η Microsoft κάνει τις προεπιλογές ασφαλείας διαθέσιμες σε όλους. Ο στόχος είναι να εξασφαλίσετε ότι όλες οι εταιρείες έχουν ένα βασικό επίπεδο ασφάλειας ενεργοποιημένο χωρίς επιπλέον κόστος. Ενεργοποιείτε τις προεπιλογές ασφαλείας στην πύλη Azure.
