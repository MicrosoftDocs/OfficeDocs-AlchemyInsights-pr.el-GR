# <a name="change-your-language"></a>Αλλαγή της γλώσσας

1. Στην επάνω δεξιά γωνία, επιλέξτε **Ρυθμίσεις**![ρυθμίσεις](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) \> **Επιλογές**.
    
2. Στην ενότητα **Επιλογές**, επιλέξτε **Γενική** \> **περιοχής και ζώνης ώρας**.
    
3. Στην περιοχή **Τοπικές ρυθμίσεις και ρυθμίσεις ζώνης ώρας**, επιλέξτε τη γλώσσα σας.
    
4. Επιλέξτε **Αποθήκευση**.
    
Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [Αλλαγή της ζώνης ώρας και τις ρυθμίσεις γλώσσας στο Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873132).
