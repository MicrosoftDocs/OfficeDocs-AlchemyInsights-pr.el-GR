# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>Σφάλμα διεύθυνσης διακομιστή μεσολάβησης κατά τη δημιουργία ενός κοινόχρηστου γραμματοκιβωτίου

Εάν προσπαθήσατε να δημιουργήσετε ένα κοινόχρηστο γραμματοκιβώτιο και λάβατε αυτό το σφάλμα "η διεύθυνση διακομιστή μεσολάβησης" SMTP:alias@domain.com "χρησιμοποιείται ήδη...", η διεύθυνση ηλεκτρονικού ταχυδρομείου που επιλέξατε έχει ήδη ληφθεί από άλλο γραμματοκιβώτιο του οργανισμού σας.
  
Πρέπει να εντοπίσετε το χρήστη, την ομάδα ή το κοινόχρηστο γραμματοκιβώτιο που έχει αυτήν τη διεύθυνση ηλεκτρονικού ταχυδρομείου και να το διαγράψετε ή να αλλάξετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του. Στη συνέχεια, μπορείτε να δημιουργήσετε ένα κοινόχρηστο γραμματοκιβώτιο με την ελεύθερη διεύθυνση ηλεκτρονικού ταχυδρομείου. Χρησιμοποιήστε την αναζήτηση στην αρχική σελίδα για να τη βρείτε.
  
Εάν δεν θέλετε να διαγράψετε την υπάρχουσα διεύθυνση ηλεκτρονικού ταχυδρομείου, επιλέξτε μια νέα διεύθυνση ηλεκτρονικού ταχυδρομείου για το κοινόχρηστο γραμματοκιβώτιό σας.
