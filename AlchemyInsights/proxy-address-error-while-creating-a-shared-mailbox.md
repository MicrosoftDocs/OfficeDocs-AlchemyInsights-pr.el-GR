# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>Σφάλμα διεύθυνσης διακομιστή μεσολάβησης κατά τη δημιουργία κοινόχρηστου γραμματοκιβωτίου

Εάν προσπαθήσατε να δημιουργήσετε ένα κοινόχρηστο γραμματοκιβώτιο και λάβατε αυτό το σφάλμα "Η διεύθυνση διακομιστή μεσολάβησης "SMTP:alias@domain.com" χρησιμοποιείται ήδη...", η διεύθυνση ηλεκτρονικού ταχυδρομείου που επιλέξατε έχει ήδη ληφθεί από ένα άλλο γραμματοκιβώτιο στον οργανισμό σας.
  
Πρέπει να βρείτε το χρήστη, την ομάδα ή το κοινόχρηστο γραμματοκιβώτιο που έχει αυτήν τη διεύθυνση ηλεκτρονικού ταχυδρομείου και να τη διαγράψετε ή να αλλάξετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου του. Στη συνέχεια, μπορείτε να δημιουργήσετε ένα κοινόχρηστο γραμματοκιβώτιο με τη διεύθυνση ηλεκτρονικού ταχυδρομείου που έχει απελευθερωθεί. Χρησιμοποιήστε την "Αναζήτηση" στην Αρχική σελίδα για να τη βρείτε.
  
Εάν δεν θέλετε να διαγράψετε την υπάρχουσα διεύθυνση ηλεκτρονικού ταχυδρομείου, επιλέξτε μια νέα διεύθυνση ηλεκτρονικού ταχυδρομείου για το κοινόχρηστο γραμματοκιβώτιό σας.
