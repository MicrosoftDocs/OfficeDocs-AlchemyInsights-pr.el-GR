# <a name="change-your-display-name"></a>Αλλαγή του ονόματος εμφάνισης
  
Για να αλλάξετε το εμφανιζόμενο όνομα στο Outlook, ακολουθήστε τα παρακάτω βήματα.
  
1. Στο Outlook, επιλέξτε **Ρυθμίσεις λογαριασμού** **αρχείων** \> **Ρυθμίσεις** \> λογαριασμού.

2. Επιλέξτε το λογαριασμό ηλεκτρονικού ταχυδρομείου και, στη συνέχεια, επιλέξτε **Αλλαγή**.

    > [!NOTE]
    > Εάν δεν βλέπετε την επιλογή για να αλλάξετε το εμφανιζόμενο όνομα, πιθανώς χρησιμοποιείτε ένα λογαριασμό ηλεκτρονικού ταχυδρομείου του Microsoft Exchange. Σε αυτήν την περίπτωση, θα πρέπει να επικοινωνήσετε με το διαχειριστή σας για να αλλάξετε το εμφανιζόμενο όνομα. Ανατρέξτε [στην αλλαγή ενός ονόματος χρήστη και μιας διεύθυνσης ηλεκτρονικού ταχυδρομείου στο Office 365](https://docs.microsoft.com/office365/admin/add-users/change-a-user-name-and-email-address).
  
3. Αφού αλλάξετε το εμφανιζόμενο όνομα, κάντε κλικ στο κουμπί **OK** για να αποθηκεύσετε τις αλλαγές σας. Μπορεί να χρειαστούν λίγες ώρες για να απεικονιστούν οι αλλαγές σας σε όλο τον οργανισμό σας.

Για περισσότερες πληροφορίες, ανατρέξτε [στο στοιχείο Αλλαγή του ονόματος εμφάνισης που βλέπουν οι παραλήπτες ηλεκτρονικού ταχυδρομείου](https://support.office.com/article/2b53331a-ba2a-4803-88dc-ac9fe376c8a9.aspx).
