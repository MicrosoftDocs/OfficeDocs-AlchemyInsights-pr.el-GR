# <a name="change-your-display-name"></a>Αλλαγή του εμφανιζόμενου ονόματος
  
Για να αλλάξετε το εμφανιζόμενο όνομα στο Outlook, ακολουθήστε τα παρακάτω βήματα.
  
1. Στο Outlook, επιλέξτε Ρυθμίσεις λογαριασμού **File** \> **αρχείου** \> **Account Settings**.

2. Επιλέξτε το λογαριασμό ηλεκτρονικού ταχυδρομείου και, στη συνέχεια, επιλέξτε **Αλλαγή**.

    > [!NOTE]
    > Εάν δεν βλέπετε την επιλογή αλλαγής του εμφανιζόμενου ονόματός σας, πιθανότατα χρησιμοποιείτε ένα λογαριασμό ηλεκτρονικού ταχυδρομείου του Microsoft Exchange. Σε αυτήν την περίπτωση, πρέπει να επικοινωνήσετε με το διαχειριστή σας για να αλλάξετε το εμφανιζόμενο όνομα. Ανατρέξτε στο θέμα [Αλλαγή ονόματος χρήστη και διεύθυνσης ηλεκτρονικού ταχυδρομείου](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address).
  
3. Αφού αλλάξετε το εμφανιζόμενο όνομά σας, κάντε κλικ στο κουμπί **OK** για να αποθηκεύσετε τις αλλαγές σας. Μπορεί να χρειαστούν μερικές ώρες ώστε οι αλλαγές σας να αντικατοπτρίζουν σε ολόκληρο τον οργανισμό σας.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Αλλαγή του εμφανιζόμενου ονόματος που βλέπουν οι παραλήπτες ηλεκτρονικού ταχυδρομείου](https://support.office.com/article/2b53331a-ba2a-4803-88dc-ac9fe376c8a9.aspx), ανατρέξτε στο θέμα .
