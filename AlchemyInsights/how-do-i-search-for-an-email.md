# <a name="how-do-i-search-for-an-email"></a>Πώς μπορώ να κάνω αναζήτηση για ένα μήνυμα ηλεκτρονικού ταχυδρομείου;

Μπορείτε να αναζητήσετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου αναζητώντας την επαφή που σας έστειλε το μήνυμα ηλεκτρονικού ταχυδρομείου ή αναζητώντας μια λέξη-κλειδί που περιέχεται στο μήνυμα ηλεκτρονικού ταχυδρομείου.

Πατήστε τον μεγεθυντικό φακό στην κάτω γραμμή περιήγησης. Στη συνέχεια, πληκτρολογήστε το αίτημα αναζήτησής σας στο πλαίσιο στο επάνω μέρος της οθόνης σας. 

**Tip:** Μπορείτε να πατήσετε δύο φορές τον μεγεθυντικό φακό για να εστιάσετε στο πλαίσιο αναζήτησης. Τα αποτελέσματά σας θα εμφανιστούν καθώς πληκτρολογείτε το αίτημά σας.
