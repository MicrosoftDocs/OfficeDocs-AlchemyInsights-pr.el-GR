# <a name="set-up-mobile-apps-for-office-365-and-microsoft-365"></a>Ρύθμιση εφαρμογών για κινητές συσκευές για το Office 365 και το Microsoft 365

Ο [οδηγός](https://go.microsoft.com/fwlink/?linkid=2142115) ρύθμισης εφαρμογών για κινητές συσκευές παρέχει οδηγίες βήμα προς βήμα για τη λήψη και την εγκατάσταση των εφαρμογών Microsoft 365 και Office 365 σε κινητές συσκευές Windows, iOS και Android.
