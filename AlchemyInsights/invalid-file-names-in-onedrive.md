# <a name="invalid-file-and-folder-names-in-onedrive-and-sharepoint"></a>Τα ονόματα αρχείων και φακέλων δεν είναι έγκυρα στο OneDrive και το SharePoint

Αυτοί οι χαρακτήρες δεν επιτρέπονται \* στα \< \> ονόματα αρχείων και φακέλων " : ? / \ | 
  
Ορισμένοι οργανισμοί δεν έχουν ακόμη υποστήριξη για # και % ενεργοποιημένη. Για να μάθετε πώς μπορείτε να επιτρέψετε αυτούς τους χαρακτήρες στον οργανισμό σας, [ανατρέξτε](https://go.microsoft.com/fwlink/?linkid=862611)στο θέμα Ενεργοποίηση # και % υποστήριξης .
