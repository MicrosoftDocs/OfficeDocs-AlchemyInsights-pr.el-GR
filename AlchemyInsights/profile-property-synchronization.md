# <a name="profile-property-synchronization"></a>Συγχρονισμός ιδιότητας προφίλ

Δεν είναι δυνατή η αντιστοίχιση πρόσθετων ιδιοτήτων για την εισαγωγή διαφημίσεων για συγχρονισμό από το Azure Active Directory στην εφαρμογή προφίλ χρήστη. Η εισαγωγή διαφημίσεων περιορίζεται σε μια προρυθμισμένη ομάδα ιδιοτήτων για να εγγυηθεί τη συνεπή απόδοση της εργασίας χρονομέτρησης.
  
[Μάθετε περισσότερα σχετικά με τις ιδιότητες προφίλ που συγχρονίζονται](https://go.microsoft.com/fwlink/?linkid=875671)
