# <a name="profile-property-synchronization"></a>Συγχρονισμός ιδιοτήτων προφίλ

Δεν είναι δυνατό να αντιστοιχίσετε πρόσθετες ιδιότητες για εισαγωγή AD συγχρονισμό από τον ενεργό κατάλογο Azure με την εφαρμογή προφίλ χρήστη. Εισαγωγή AD περιορίζεται σε ένα προκαθορισμένο σύνολο των ιδιοτήτων που εγγυάται συνέπεια στις επιδόσεις της εργασίας χρονομέτρησης.
  
[Μάθετε περισσότερα σχετικά με τις ιδιότητες προφίλ που συγχρονίζονται](https://go.microsoft.com/fwlink/?linkid=875671)
