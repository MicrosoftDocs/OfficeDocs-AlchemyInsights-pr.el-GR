# <a name="profile-property-synchronization"></a>Συγχρονισμός ιδιότητας προφίλ

Δεν είναι δυνατό να αντιστοιχίσετε πρόσθετες ιδιότητες για την εισαγωγή AD για συγχρονισμό από το Azure Active Directory με την εφαρμογή προφίλ χρήστη. Η εισαγωγή AD περιορίζεται σε ένα προκαθορισμένο σύνολο ιδιοτήτων για την εγγύηση της συνεπούς απόδοσης της εργασίας χρονομέτρησης.
  
[Μάθετε περισσότερα σχετικά με τις ιδιότητες προφίλ που συγχρονίζονται](https://go.microsoft.com/fwlink/?linkid=875671)
