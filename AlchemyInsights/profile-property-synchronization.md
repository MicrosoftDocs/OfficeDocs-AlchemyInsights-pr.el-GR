# <a name="profile-property-synchronization"></a>Συγχρονισμός ιδιοτήτων προφίλ

Δεν είναι δυνατή η αντιστοίχιση πρόσθετων ιδιοτήτων για την εισαγωγή AD για συγχρονισμό από την υπηρεσία καταλόγου Azure Active Directory στην εφαρμογή προφίλ χρήστη. Η εισαγωγή AD περιορίζεται σε ένα προκαθορισμένο σύνολο ιδιοτήτων για να εγγυηθεί τη συνεπή απόδοση της εργασίας χρονομέτρων.
  
[Μάθετε περισσότερα σχετικά με τις ιδιότητες προφίλ που συγχρονίζονται](https://go.microsoft.com/fwlink/?linkid=875671)
