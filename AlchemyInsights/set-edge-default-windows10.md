# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Ορισμός του Microsoft Edge ως προεπιλεγμένου προγράμματος περιήγησης σε μια συσκευή Windows 10

Αυτό μπορεί να γίνει ως εξής:

1. Επιλέξτε το κουμπί **"Έναρξη"** και, στη συνέχεια, εισαγάγετε **τις προεπιλεγμένες εφαρμογές.**

1. Στα αποτελέσματα αναζήτησης, επιλέξτε **"Προεπιλεγμένες εφαρμογές".**

1. Στην περιοχή **Πρόγραμμα περιήγησης Web,** **επιλέξτε Microsoft Edge.**
