# <a name="planner-common-issues-and-resolutions"></a>Κοινά ζητήματα και ψηφίσματα του σχεδιαστή

Ακολουθούν ορισμένα θέματα που ζητούνται συχνά με το προγραμματιστή της Microsoft.
 
- **Γρήγορα αποτελέσματα:** [αρχίστε να χρησιμοποιείτε το σχεδιασμό](https://support.office.com/article/microsoft-planner-help-4a9a13c6-3adf-4a60-a6fc-15c0b15e16fc?ui=en-US&rs=en-US&ad=US)και μάθετε πώς μπορείτε να [ενεργοποιήσετε το σχεδιαστή για τον οργανισμό σας, να προσθέσετε χρήστες και να διαχειριστείτε την πρόσβαση στο περιβάλλον σας](https://docs.microsoft.com/office365/planner/planner-for-admins).
 
- **Επισκέπτες χρήστες:** Διαχειριστείτε την [πρόσβαση του επισκέπτη χρήστη](https://support.office.com/article/guest-access-in-microsoft-planner-cc5d7f96-dced-4da4-ab62-08c72d9759c6?ui=en-US&rs=en-US&ad=US) στο σχεδιασμό.
 
- **Κοινοποιήσεις:** Εξετάστε τους [τύπους των ειδοποιήσεων που θα λάβετε](https://support.office.com/article/stay-on-top-of-tasks-and-plans-with-email-and-notifications-cce223d6-b0ae-43cf-a080-266e2414a859), τις δραστηριότητες που τις ενεργοποιούν και όπου μπορείτε να αλλάξετε τις ρυθμίσεις ειδοποιήσεων.
 
- **Σχόλια:** 
   - Πληροφορίες σχετικά [με ζητήματα με την προβολή ή την προσθήκη σχολίων](https://docs.microsoft.com/office365/planner/planner-for-admins#can-people-in-my-organization-use-planner-if-they-dont-have-an-exchange-online-mailbox) σε μια εργασία χωρίς γραμματοκιβώτιο του Exchange Online.
   - Μάθετε πώς μπορείτε να βεβαιωθείτε [ότι όλοι οι χρήστες μπορούν να πάρουν μηνύματα ηλεκτρονικού ταχυδρομείου για προγραμματισμό](https://docs.microsoft.com/office365/planner/planner-for-admins#how-do-i-make-sure-all-my-users-can-get-emails-forplanner).

- **Διαχείριση εργασιών:** Εξετάστε τις πληροφορίες σχετικά με το [σχολιασμό εργασιών και τη](https://support.office.com/article/comment-on-tasks-in-microsoft-planner-fd4aedde-7785-4cd0-96ee-122fbc9140e1?ui=en-US&rs=en-US&ad=US) [Διαγραφή εργασιών ή σχεδίων](https://support.office.com/article/delete-a-task-or-plan-39e10e78-13f0-446d-94cd-9e562648497a).
