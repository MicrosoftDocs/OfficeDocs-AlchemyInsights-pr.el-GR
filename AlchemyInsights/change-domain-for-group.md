# <a name="change-the-domain-for-office-365-group"></a>Αλλαγή του τομέα για την ομάδα Office 365

Εάν θέλετε να αλλάξετε τον τομέα για μια ομάδα Office 365 κατά τη δημιουργία, ακολουθήστε τα βήματα για [να επιλέξετε τον τομέα που θα χρησιμοποιηθεί κατά τη δημιουργία ομάδων 365 του Office](https://docs.microsoft.com/office365/admin/create-groups/choose-domain-to-create-groups).
  
Αφού η ομάδα Office 365 έχει ήδη δημιουργηθεί, δεν υπάρχει κανένας τρόπος για να αλλάξετε τον τομέα. Ο μόνος τρόπος είναι να δημιουργήσετε μια νέα ομάδα Office 365 και να διαγράψετε το παλιό, αν είναι απαραίτητο (πρόσθετες διευθύνσεις ηλεκτρονικού ταχυδρομείου μπορούν να προστεθούν μέσω του [Κέντρου διαχείρισης Exchange](https://outlook.office365.com/ecp.aspx)).
