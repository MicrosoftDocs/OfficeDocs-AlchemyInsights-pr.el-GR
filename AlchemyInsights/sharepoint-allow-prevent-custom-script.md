# <a name="allow-custom-script-to-enable-features"></a>Να επιτρέπεται η προσαρμοσμένη δέσμη ενεργειών για την ενεργοποίηση δυνατοτήτων

Οι διαχειριστές ενδέχεται να χρειαστεί να **επιτρέψουν στην προσαρμοσμένη δέσμη ενεργειών** να ενεργοποιήσει ορισμένες δυνατότητες. Για παράδειγμα, οι χρήστες ενδέχεται να λάβετε Σφάλμα 403 κατά τη σύνδεση στο SharePoint με το SharePoint Designer. Για λεπτομερή βήματα, παραδείγματα και θέματα, ανατρέξτε στο εξής: [Αποδοχή ή αποτροπή προσαρμοσμένης δέσμης ενεργειών](https://docs.microsoft.com/sharepoint/allow-or-prevent-custom-script).

Εάν οι χρήστες αντιμετωπίζουν ένα **σφάλμα 403** με το SharePoint ή το OneDrive, μπορείτε να ελέγξετε τον [πίνακα εργαλείων εύρυθμης λειτουργίας υπηρεσίας](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) για να προσδιορίσετε αν πρόκειται για ένα γνωστό ζήτημα με μια ανάλυση σε εξέλιξη, πριν να καλέσετε την υποστήριξη ή να ξοδέψετε χρόνο αντιμετώπισης προβλημάτων. Το ζήτημα μπορεί να είναι πολύ προσωρινό και ήδη ταυτοποιηθεί από τη Microsoft, αλλά ενδέχεται να υπάρχει μια χρονική λήξη μεταξύ του αναγνωριστικού έκδοσης και της καταχώρησης στον πίνακα εργαλείων εύρυθμης λειτουργίας υπηρεσίας. Παρακαλούμε Συνεχίστε να ελέγχετε τον πίνακα εργαλείων εύρυθμης λειτουργίας υπηρεσίας για περαιτέρω ενημερώσεις.
