# <a name="find-the-page-that-shows-my-contacts"></a>Εύρεση της σελίδας που εμφανίζει τις επαφές μου

Μπορείτε να αναζητήσετε τις επαφές σας με τον ίδιο τρόπο που αναζητάτε τα μηνύματα ηλεκτρονικού ταχυδρομείου σας.
 
Πατήστε το εικονίδιο αναζήτησης μεγεθυντικού φακού και, στη συνέχεια, πληκτρολογήστε το αίτημα αναζήτησης επαφών στο πλαίσιο στο επάνω μέρος της οθόνης σας. (**Protip**: Μπορείτε να πατήσετε δύο φορές στο μεγεθυντικό φακό για να εστιάσετε στο πλαίσιο αναζήτησης.) Τα αποτελέσματά σας θα εμφανιστούν στην ενότητα Επαφές.
