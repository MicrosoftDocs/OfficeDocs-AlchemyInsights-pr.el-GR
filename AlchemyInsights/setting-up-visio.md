# <a name="setting-up-visio"></a>Ρύθμιση του Visio

Φαίνεται ότι χρειάζεστε βοήθεια για τη ρύθμιση του Visio με τη συνδρομή σας στο Microsoft 365.
  
Ειδοποίηση για χρήστες Mac: το προϊόν Visio δεν είναι διαθέσιμο για το MacOS αυτή τη στιγμή. Εάν έχετε υπολογιστή, θα πρέπει να έχετε τη δυνατότητα να εγκαταστήσετε το λογισμικό ακολουθώντας τα παρακάτω βήματα.
  
Μπορείτε να ακολουθήσετε τα βήματα που περιλαμβάνετε σε [αυτό το άρθρο](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710.aspx) για να ολοκληρώσετε την εγκατάστασή σας.
