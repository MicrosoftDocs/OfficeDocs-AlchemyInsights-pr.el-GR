# <a name="distribution-group-aimkiobject-not-showing-in-groups-list-after-aimkitimeelapsed-minutes"></a>Η ομάδα διανομής "{AimKIObject}" δεν εμφανίζεται στη λίστα "ομάδες" μετά από {AimKITimeElapsed} λεπτά

Η ομάδα σας "{AimKIObject}" δημιουργήθηκε {AimKITimeElapsed} λεπτά πριν. Συνήθως, δεν χρειάζεται περισσότερο από 60 λεπτά για να εμφανιστεί η νέα ομάδα διανομής σας στο κέντρο διαχείρισης.
  
Ενδέχεται να μπορείτε να χρησιμοποιήσετε τη δυνατότητα "παραλήπτες > ομάδες" στο [Κέντρο διαχείρισης του Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx) για να διαχειριστείτε την ομάδα σας τώρα ή να χρησιμοποιήσετε την επιλογή "να σας καλέσουμε" για να μιλήσετε με έναν μηχανικό υποστήριξης.
