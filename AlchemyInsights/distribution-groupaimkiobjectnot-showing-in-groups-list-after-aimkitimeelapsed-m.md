# <a name="distribution-group-aimkiobject-not-showing-in-groups-list-after-aimkitimeelapsed-minutes"></a>Ομάδα διανομής "{AimKIObject}" δεν εμφανίζεται στη λίστα "ομάδες" μετά από λεπτά {AimKITimeElapsed}

Η ομάδα "{AimKIObject}" δημιουργήθηκε {AimKITimeElapsed} λεπτά. Κανονικά δεν χρειάζεται περισσότερο από 60 λεπτά για τη νέα ομάδα διανομής να εμφανίζεται στο Κέντρο διαχείρισης.
  
Που μπορεί να έχετε τη δυνατότητα να χρησιμοποιήσετε τη δυνατότητα ομάδες > παραλήπτες στο [Κέντρο διαχείρισης του Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx) για να διαχειριστείτε την ομάδα σας τώρα ή χρησιμοποιήστε την επιλογή "ας γίνει κλήσης" για να συνομιλήσετε με έναν μηχανικό υποστήριξης.
