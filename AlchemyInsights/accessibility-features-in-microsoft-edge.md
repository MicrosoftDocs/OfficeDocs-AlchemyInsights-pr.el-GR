# <a name="accessibility-features-in-microsoft-edge"></a>Δυνατότητες προσβασιμότητας στον Microsoft Edge

Ο Microsoft Edge διαθέτει ενσωματωμένες επιλογές προσβασιμότητας για να σας βοηθήσει να περιηγηθείτε στο Web με τρόπο που ανταποκρίνεται καλύτερα στις ανάγκες σας.

Για να μάθετε περισσότερα, ανατρέξτε [στο θέμα Δυνατότητες προσβασιμότητας στον Microsoft Edge.](https://go.microsoft.com/fwlink/?linkid=2153648)
