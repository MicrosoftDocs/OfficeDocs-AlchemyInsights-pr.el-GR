# <a name="pop-imap--smtp-settings-for-outlook-on-the-web"></a>POP, ρυθμίσεις & SMTP IMAP για το Outlook στο web

Εάν θέλετε να προσθέσετε το λογαριασμό του Office 365 σε μια άλλη εφαρμογή αλληλογραφίας που υποστηρίζει POP, IMAP ή SMTP, ακολουθούν τις ρυθμίσεις διακομιστή με μη αυτόματο τρόπο, θα πρέπει:
  
- **Όνομα διακομιστή IMAP:** outlook.office365.com
- **Θύρα IMAP:** 993
- **Μέθοδος κρυπτογράφησης IMAP:** ΈΛΕΓΧΟΣ ΤΑΥΤΌΤΗΤΑΣ SSL/TLS
- **Όνομα διακομιστή POP:** outlook.office365.com  
- **Θύρα POP:** 995  
- **Μέθοδος κρυπτογράφησης POP:** ΈΛΕΓΧΟΣ ΤΑΥΤΌΤΗΤΑΣ SSL/TLS  
- **Όνομα διακομιστή SMTP:** smtp.office365.com
- **Θύρα SMTP:** 587
- **Μέθοδος κρυπτογράφησης SMTP:** STARTTLS
