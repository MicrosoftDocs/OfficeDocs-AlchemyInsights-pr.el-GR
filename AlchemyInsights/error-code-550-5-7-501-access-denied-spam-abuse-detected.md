# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 δεν επιτρέπεται η πρόσβαση, εντοπίστηκε κατάχρηση ανεπιθύμητης αλληλογραφίας

Συνήθως, αυτό το μήνυμα εμφανίζεται όταν οι χρήστες στέλνουν μηνύματα ηλεκτρονικού ταχυδρομείου από διευθύνσεις IP χρησιμοποιώντας τον αρχικό τομέα *. onmicrosoft.com* που έχει εκχωρηθεί σε νέους μισθωτές στο Microsoft 365. Ο ευκολότερος τρόπος για να επιλύσετε αυτό το πρόβλημα είναι να κάνετε τα εξής:

1. [Προσθέστε έναν τομέα στον μισθωτή σας](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain).

2. [Αλλάξτε την κύρια διεύθυνση ηλεκτρονικού ταχυδρομείου των χρηστών σας](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address) στο νέο προσαρμοσμένο τομέα που μόλις προσθέσατε.
