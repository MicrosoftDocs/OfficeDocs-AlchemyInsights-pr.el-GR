# <a name="your-users-mailbox-is-being-created"></a>Δημιουργείται το γραμματοκιβώτιο του χρήστη

Μπορεί να χρειαστούν έως και μία ώρα για να είναι έτοιμο ένα νέο γραμματοκιβώτιο.
  
Εάν το γραμματοκιβώτιο ενός χρήστη έχει "προετοιμάσει" για περισσότερο από μία ώρα, μπορείτε να δοκιμάσετε να διαγράψετε και, στη συνέχεια, να δημιουργήσετε ξανά το χρήστη ή να χρησιμοποιήσετε την επιλογή "να σας καλέσουμε" για να μιλήσετε με έναν μηχανικό υποστήριξης.
