# <a name="your-users-mailbox-is-being-created"></a>Δημιουργείται το γραμματοκιβώτιο του χρήστη

Αυτό ενδέχεται να διαρκέσει έως μία ώρα πριν από ένα νέο γραμματοκιβώτιο είναι έτοιμη για χρήση.
  
Εάν το γραμματοκιβώτιο του χρήστη έχει "Προετοιμασία" για περισσότερο από μία ώρα, μπορείτε να δοκιμάσετε τη διαγραφή και στη συνέχεια να δημιουργήσετε ξανά το χρήστη ή χρησιμοποιήστε την επιλογή "ας γίνει κλήσης" για να συνομιλήσετε με έναν μηχανικό υποστήριξης.
