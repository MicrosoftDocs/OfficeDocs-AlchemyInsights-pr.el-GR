# <a name="your-users-mailbox-is-being-created"></a>Δημιουργείται το γραμματοκιβώτιο του χρήστη σας

Μπορεί να διαρκέσει έως και μία ώρα μέχρι να είναι έτοιμο για χρήση ένα νέο γραμματοκιβώτιο.
  
Εάν το γραμματοκιβώτιο ενός χρήστη "προετοιμάζεται" για περισσότερο από μία ώρα, μπορείτε να δοκιμάσετε να διαγραφείτε και, στη συνέχεια, να δημιουργήσετε ξανά το χρήστη ή να χρησιμοποιήσετε την επιλογή "Ας σας καλέσουμε" για να μιλήσετε με έναν μηχανικό υποστήριξης.
