# <a name="restrict-users-who-can-upload-to-stream"></a>Περιορισμός χρηστών που μπορούν να αποσταλούν στη Ροή

Ως οργανισμός, μπορεί να θέλετε να περιορίσετε τους υπαλλήλους που μπορούν να ανεβάσουν βίντεο. Από προεπιλογή, όλα τα παιδιά στον οργανισμό σας μπορούν να ανεβάσουν βίντεο.

- [Περιορισμός υπαλλήλων που μπορούν να δημιουργήσουν περιεχόμενο στο Microsoft Stream](https://docs.microsoft.com/stream/restrict-uploaders)
