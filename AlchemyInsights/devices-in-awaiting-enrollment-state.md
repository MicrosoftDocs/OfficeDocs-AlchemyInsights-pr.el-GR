# <a name="devices-are-in-awaiting-enrollment-state"></a>Συσκευές που βρίσκονται σε κατάσταση "Αναμονή εγγραφής"

Ανάλυση επιφάνειας εργασίας δεν διαθέτει διαγνωστικά δεδομένα για αυτήν τη συσκευή. 

Αυτό το ζήτημα ενδέχεται να προκύψει, επειδή έχετε προσθέσει πρόσφατα τη συσκευή στη συλλογή του προορισμού και δεν έχει σταλεί ακόμη δεδομένα. Μπορεί επίσης να σημαίνει τη συσκευή δεν είναι σωστά επικοινωνίας με την υπηρεσία και τα πιο πρόσφατα δεδομένα διαγνωστικών είναι περισσότερο από 28 ημέρες.

Βεβαιωθείτε ότι η συσκευή μπορεί να επικοινωνήσει με την υπηρεσία. Για περισσότερες πληροφορίες, ανατρέξτε στην ενότητα " [τελικά σημεία](https://docs.microsoft.com/sccm/desktop-analytics/enable-data-sharing#endpoints)".

Για περισσότερες πληροφορίες σχετικά με την κατάσταση της εγγραφής awainting, δείτε [Την οθόνη σύνδεσης υγείας](https://docs.microsoft.com/sccm/desktop-analytics/monitor-connection-health#awaiting-enrollment).
