# <a name="troubleshoot-issues-with-microsoft-intune"></a>Αντιμετώπιση προβλημάτων σχετικά με Microsoft Intune

Εξετάστε τους πόρους που αναφέρονται παρακάτω για να επιλύσετε το θέμα σας τώρα.
  
1. Χρησιμοποιήστε [Πύλη του Intune αντιμετώπιση προβλημάτων](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) για να διαγνώσετε και να επιλύσετε ορισμένα κοινά σφάλματα. Εξετάστε [αυτό](https://docs.microsoft.com/en-us/intune/help-desk-operators) το έγγραφο για περισσότερες λεπτομέρειες.  
2. Εξετάστε [Intune TechNet ](https://social.technet.microsoft.com/forums/en-us/home?forum=microsoftintuneprod)για να βρείτε απαντήσεις και λύσεις σε συνηθισμένα προβλήματα.  
3. Αναλάβετε τον έλεγχο [Υπηρεσιών υγείας](https://portal.office.com/AdminPortal/Home#/servicehealth) για να δείτε την κατάσταση της υπηρεσίας.   
4. Εξετάστε [Intune τεκμηρίωση](https://docs.microsoft.com/en-us/intune/) για το σχεδιασμό, διαδικασιών, αντιμετώπιση προβλημάτων, άρθρα και πολλά άλλα.
