# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Υποστηριζόμενη έκδοση του εργαλείου μετεγκατάστασης του SharePoint



Αυτήν τη στιγμή το εργαλείο μετεγκατάστασης του SharePoint υποστηρίζει μόνο μετεγκαταστάσεις από 2013 του SharePoint. Οι εκδόσεις 2010 και 2016 δεν υποστηρίζονται αυτήν τη στιγμή.
  
Για περισσότερες πληροφορίες, ανατρέξτε [στην παρουσίαση του εργαλείου μετεγκατάστασης του SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
