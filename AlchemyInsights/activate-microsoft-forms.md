# <a name="activate-microsoft-forms-for-an-organization"></a>Ενεργοποίηση της Microsoft φόρμες για μια εταιρεία

Ένας admin μπορεί να ενεργοποιήσετε Microsoft Forms εντός ή εκτός της εταιρείας τους. Πληροφορίες σχετικά με [τον έλεγχο ενεργοποίηση άδειας χρήσης και της κατάστασης του CollabDBService Azure](https://support.office.com/article/Turn-off-or-turn-on-Microsoft-Forms-8dcbf3ab-f2d6-459a-b8be-8d9892132a43) για σας μίσθωσης.
