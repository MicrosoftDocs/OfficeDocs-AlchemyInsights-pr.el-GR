# <a name="move-a-folder-in-outlookcom"></a>Η μετακίνηση ενός φακέλου στο Outlook.com

Μπορείτε να αναδιατάξετε τους φακέλους στην περιοχή **"Αγαπημένα"**, αλλά δεν μπορείτε να αναδιατάξετε τους φακέλους κάτω από τους **φακέλους** ή να δημιουργήσετε έναν υποφάκελο ενός φακέλου. Εάν κατά λάθος μπορείτε να μετακινήσετε ένα φάκελο σε άλλο φάκελο, δημιουργήστε έναν νέο φάκελο και αντιγράψτε όλα τα email σας από τον υποφάκελο στον νέο φάκελο.
  
Για να μάθετε περισσότερα, διαβάστε την [εργασία με φακέλους μηνυμάτων στο Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
