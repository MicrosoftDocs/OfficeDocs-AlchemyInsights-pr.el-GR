# <a name="document-versioning"></a>Διαχείριση εκδόσεων εγγράφων

Η διαχείριση εκδόσεων είναι ενεργοποιημένη από προεπιλογή για τις βιβλιοθήκες εγγράφων και του OneDrive για επιχειρήσεις. Για να επαναφέρετε μόνοι σας μια έκδοση των αρχείων, ανατρέξτε στο θέμα [Επαναφορά προηγούμενης έκδοσης ενός αρχείου](https://support.office.com/article/restore-a-previous-version-of-a-file-in-onedrive-159cad6d-d76e-4981-88ef-de6e96c93893?ui=en-US&rs=en-US&ad=US).  

Για περισσότερες πληροφορίες σχετικά με τη διαχείριση εκδόσεων, ανατρέξτε στα παρακάτω άρθρα:  

- Για πληροφορίες σχετικά με τα όρια διαχείρισης εκδόσεων, ανατρέξτε στο θέμα [Όρια του SharePoint](https://docs.microsoft.com/office365/servicedescriptions/sharepoint-online-service-description/sharepoint-online-limits).     
- [Πώς λειτουργεί η διαχείριση εκδόσεων σε μια λίστα ή βιβλιοθήκη του SharePoint](https://support.office.com/article/how-does-versioning-work-in-a-sharepoint-list-or-library-0f6cd105-974f-44a4-aadb-43ac5bdfd247).
- [Ενεργοποίηση και ρύθμιση παραμέτρων διαχείρισης εκδόσεων για μια λίστα ή βιβλιοθήκη](https://support.office.com/article/enable-and-configure-versioning-for-a-list-or-library-1555d642-23ee-446a-990a-bcab618c7a37?ocmsassetID=HA102772148&CTT=3&CorrelationId=52441bb1-a619-4375-89d5-19d28769890f&ui=en-US&rs=en-US&ad=US).
- [Προβολή του ιστορικού εκδόσεων ενός στοιχείου ή αρχείου σε μια λίστα ή βιβλιοθήκη](https://support.office.com/article/View-the-version-history-of-an-item-or-file-in-a-list-or-library-53262060-5092-424D-A50B-C798B0EC32B1).
- [Προβολή προηγούμενων εκδόσεων των αρχείων του Office](https://support.office.com/article/view-previous-versions-of-office-files-5c1e076f-a9c9-41b8-8ace-f77b9642e2c2).
