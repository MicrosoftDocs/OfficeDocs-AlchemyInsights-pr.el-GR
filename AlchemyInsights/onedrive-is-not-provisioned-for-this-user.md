# <a name="onedrive-is-not-provisioned-for-this-user"></a>Το OneDrive δεν προμηθεύεται για αυτόν το χρήστη

Για τους λογαριασμούς που δημιουργήθηκαν πρόσφατα, οι χρήστες πρέπει να πραγματοποιήσουν είσοδο στο κέντρο διαχείρισης του Microsoft 365 με δικό τους λογαριασμό και να κάνουν κλικ στην επιλογή OneDrive για να ξεκινήσουν με μη αυτόματο τρόπο την παροχή της τοποθεσίας του OneDrive.
  
Ωστόσο, εάν ο χρήστης έχει ήδη κάνει κλικ στο OneDrive και η προσωπική τοποθεσία του δεν έχει αποδοθεί εντός 24 ωρών, καταγράψτε ένα δελτίο υποστήριξης για περαιτέρω βοήθεια.
