# <a name="error-546-or-5414-related-to-hop-count-exceeded"></a>Σφάλμα 5.4.6 ή 5.4.14 που σχετίζεται με υπέρβαση του ορίου μεταπηδάσεών

Εάν εμφανίζεται ο κωδικός σφάλματος 5.4.6 ή 5.4.14 που σχετίζεται με την υπέρβαση του ορίου μεταπηδάσεών σας, αυτό πιθανώς οφείλεται σε ένα πρόβλημα βρόχου αλληλογραφίας στο Exchange Online.

Για να κατανοήσετε την πιθανή αιτία και τη λύση, ανατρέξτε στο θέμα [Καταμέτρηση μεταπηδών.](/exchange/mail-flow-best-practices/non-delivery-reports-in-exchange-online/fix-error-code-5-4-6-through-5-4-20-in-exchange-online)
