# <a name="change-your-email-address"></a>Αλλαγή της διεύθυνσης ηλεκτρονικού ταχυδρομείου σας 

Για να αλλάξετε τη διεύθυνση ηλεκτρονικού ταχυδρομείου σας, θα χρειαστείτε δικαιώματα διαχειριστή.
  
Ακολουθήστε τις οδηγίες στο θέμα [Αλλαγή ονόματος χρήστη και διεύθυνσης ηλεκτρονικού ταχυδρομείου στο Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address).
