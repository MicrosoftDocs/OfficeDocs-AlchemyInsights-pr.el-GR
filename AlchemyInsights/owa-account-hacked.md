# <a name="outlook-on-the-web-account-hacked"></a>Το Outlook στο λογαριασμό Web έχει παραβιαστεί

Εάν ο λογαριασμός σας έχει παραβιαστεί, ακολουθήστε τις οδηγίες σχετικά με τον [τρόπο ασφαλούς και επαναφοράς της συνάρτησης ηλεκτρονικού ταχυδρομείου σε έναν ύποπτο κίνδυνο λογαριασμού Office 365 και γραμματοκιβώτιο](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account#how-to-secure-and-restore-email-function-to-a-suspected-compromised-office-365-account-and-mailbox).

Για να μάθετε περισσότερα σχετικά με τα πρόσθετα βήματα που μπορείτε να κάνετε για να αποτρέψετε τον κίνδυνο του λογαριασμού σας, ανατρέξτε [στο θέμα ανταπόκριση σε έναν λογαριασμό ηλεκτρονικού ταχυδρομείου που έχει παραβιαστεί στο Office 365](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account).
