# <a name="bitlocker-recovery"></a>Ανάκτηση BitLocker

Μεταβείτε στην [επιλογή Συσκευές](https://account.microsoft.com/devices/recoverykey) για να προβάλετε τα κλειδιά αποκατάστασης bitLocker.

Ανατρέξτε [στο θέμα Εύρεση του κλειδιού](https://support.microsoft.com/help/4026181) αποκατάστασης BitLocker για περισσότερες πληροφορίες.
