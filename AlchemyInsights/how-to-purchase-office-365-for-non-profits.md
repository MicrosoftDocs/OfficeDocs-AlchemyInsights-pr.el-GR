# <a name="microsoft-365-for-nonprofits---under-review"></a>Microsoft 365 για μη κερδοσκοπικούς οργανισμούς-υπό αναθεώρηση

Η καταλληλότητά σας ως πιστοποιημένη μη κερδοσκοπική εταιρεία της Microsoft βρίσκεται **υπό αναθεώρηση**. Θα λάβετε μια ειδοποίηση για τον προσδιορισμό της καταλληλότητάς σας **48 ώρες** από την ώρα της υποβολής σας. Περιμένετε τουλάχιστον **48 ώρες** για να λάβετε αυτή την ειδοποίηση πριν ανοίξετε μια αίτηση εξυπηρέτησης. 

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Microsoft 365 για μη κερδοσκοπικούς οργανισμούς](https://www.microsoft.com/nonprofits/microsoft-365).
