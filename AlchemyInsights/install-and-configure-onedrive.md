# <a name="install-and-configure-onedrive"></a>Εγκατάσταση και ρύθμιση παραμέτρων OneDrive

Με τα Windows 10 προεγκατεστημένα στον υπολογιστή-πελάτη συγχρονισμού του OneDrive. Για να εγκαταστήσετε το OneDrive για τα Windows 7 ή Mac, ή για τη λήψη της εφαρμογής για κινητές συσκευές για το τηλέφωνο ή το tablet, μεταβείτε στο [http://onedrive.com/download](http://onedrive.com/download).
  
Για να ρυθμίσετε τον υπολογιστή-πελάτη συγχρονισμού του OneDrive στα Windows, ανατρέξτε στο θέμα:
  
- [Ρύθμιση του υπολογιστή για συγχρονισμό σας OneDrive για επαγγελματική αρχεία στο Office 365](https://go.microsoft.com/fwlink/?linkid=533375)
    
- [Συγχρονισμός αρχείων του SharePoint με το πρόγραμμα-πελάτης συγχρονισμού νέα OneDrive](https://go.microsoft.com/fwlink/?linkid=871666)
