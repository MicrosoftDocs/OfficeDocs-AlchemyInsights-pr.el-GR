# <a name="error-onedrive-cannot-connect-to-windows"></a>Σφάλμα: το OneDrive δεν μπορεί να συνδεθεί στα Windows

Για να επιλύσετε το μήνυμα λάθους, αναζητήστε το αρχείο cldflt. sys για να δείτε αν εκτελείται, και αν δεν είναι, ενεργοποιήστε το. 
  
Για πρόσθετες πληροφορίες σχετικά με τον τρόπο ενεργοποίησης του αρχείου cldflt. sys, δείτε το [OneDrive δεν μπορεί να συνδεθεί με τα Windows "σφάλμα κατά την πρόσβαση σε αρχεία στο OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
