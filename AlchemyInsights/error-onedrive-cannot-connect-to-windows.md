# <a name="error-onedrive-cannot-connect-to-windows"></a>Σφάλμα: Το OneDrive δεν είναι δυνατό να συνδεθεί στα Windows

Για να επιλύσετε το μήνυμα λάθους, αναζητήστε το αρχείο cldflt.sys για να δείτε εάν εκτελείται και, εάν δεν εκτελείται, ενεργοποιήστε το. 
  
Για πρόσθετες πληροφορίες σχετικά με τον τρόπο ενεργοποίησης του αρχείου cldflt.sys, ανατρέξτε στο θέμα [Δεν είναι δυνατή η σύνδεση του OneDrive με τα Windows" κατά την πρόσβαση σε αρχεία στο OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
