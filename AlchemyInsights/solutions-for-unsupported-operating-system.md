# <a name="solutions-for-unsupported-operating-system"></a>Λύσεις για "Μη υποστηριζόμενο λειτουργικό σύστημα"

Όταν προσπαθείτε να εγκαταστήσετε το Office 365, μπορεί να δείτε ένα μήνυμα λάθους **μη υποστηριζόμενο λειτουργικό σύστημα** , εάν η συσκευή σας δεν πληροί τις [απαιτήσεις του συστήματος](https://products.office.com/office-system-requirements).
  
Office 365 απαιτεί:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 ή Windows Server 2008 R2

    OR

- Mac OS X 10.10

Εάν λάβατε αυτό το μήνυμα κατά την εγκατάσταση του Office σε μια κινητή συσκευή ή ένα Chromebook, δείτε [πώς μπορείτε να εγκαταστήσετε και να εκτελέσετε το Microsoft Office σε ένα Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
