# <a name="solutions-for-unsupported-operating-system"></a>Λύσεις για το "Μη υποστηριζόμενο λειτουργικό σύστημα"

Όταν προσπαθείτε να εγκαταστήσετε το Microsoft 365, ενδέχεται να εμφανιστεί ένα μήνυμα **σφάλματος λειτουργικού συστήματος που δεν υποστηρίζεται,** εάν η συσκευή σας δεν πληροί [τις απαιτήσεις συστήματος](https://products.office.com/office-system-requirements).
  
Το Microsoft 365 απαιτεί:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 ή Windows Server 2008 R2

    Ή

- Mac OS X 10.10

Εάν λάβατε αυτό το μήνυμα ενώ προσπαθείτε να εγκαταστήσετε το Office σε κινητή συσκευή ή chromebook, ανατρέξτε στο θέμα [Τρόπος εγκατάστασης και εκτέλεσης του Microsoft Office σε Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
