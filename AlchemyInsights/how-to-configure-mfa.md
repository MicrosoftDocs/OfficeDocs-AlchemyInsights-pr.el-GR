# <a name="configure-multi-factor-authentication-in-office-365"></a>Ρύθμιση των παραμέτρων ελέγχου ταυτότητας πολλών παραγόντων στο Office 365

Για να ρυθμίσετε τις παραμέτρους της ΣΠΙ, ανατρέξτε στην ενότητα [Ρύθμιση του ελέγχου ταυτότητας πολλών παραγόντων για τους χρήστες του Office 365](https://support.office.com/article/8f0454b2-f51a-4d9c-bcde-2c48e41621c6.aspx).

Σημειώστε ότι πρέπει να είναι μια καθολικής διαχείρισης του Office 365, για να ενεργοποιήσετε και να ρυθμίσετε τις παραμέτρους της ΣΠΙ.
