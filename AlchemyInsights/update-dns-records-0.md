# <a name="update-dns-records"></a>Ενημέρωση εγγραφών DNS

Ορισμός σας εγγραφές DNS είναι σημαντικό με την ολοκλήρωση της εργασίας με το όνομα του προσαρμοσμένου τομέα σας τη συνδρομή σας Office 365. Χρησιμοποιήστε τις παρακάτω συνδέσεις για να προσθέσετε ή να ενημερώσετε τις εγγραφές DNS σας.
  
- [Ενημέρωση των εγγραφών DNS στο Office 365](https://support.office.com/article/B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23)
    
- [Προσθήκη ή επεξεργασία προσαρμοσμένων εγγραφές DNS](https://support.office.com/article/AF00A516-DD39-4EDA-AF3E-1EAF686C8DC9)
