# <a name="update-dns-records"></a>Ενημέρωση εγγραφών DNS

Η ρύθμιση των εγγραφών DNS είναι σημαντική για να κάνετε τη συνδρομή σας στο Office 365 να λειτουργεί με το προσαρμοσμένο όνομα τομέα σας. Χρησιμοποιήστε τις παρακάτω συνδέσεις για να προσθέσετε ή να ενημερώσετε τις εγγραφές σας DNS.
  
- [Ενημέρωση εγγραφών DNS στο Office 365](https://support.office.com/article/B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23)

- [Προσθήκη ή επεξεργασία προσαρμοσμένων εγγραφών DNS](https://support.office.com/article/AF00A516-DD39-4EDA-AF3E-1EAF686C8DC9)
