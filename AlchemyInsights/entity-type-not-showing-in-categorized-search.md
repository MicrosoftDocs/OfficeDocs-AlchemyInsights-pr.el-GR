# <a name="entity-type-not-showing-in-search-results"></a>Τύπος οντότητας δεν εμφανίζεται στα αποτελέσματα αναζήτησης

Για έναν τύπο οντότητας για την εμφάνιση σε κατηγοριοποιημένες αναζήτησης ή συνάφεια αναζήτησης στο Dynamics 365 ενοποιημένο περιβάλλον εργασίας του, πρέπει να πρώτα προστεθεί στη λειτουργική μονάδα της εφαρμογής.

Σημείωση: Κατά την αναζήτηση συνημμένων σε συνάφεια αναζήτησης, η οντότητα συνημμένο πρέπει να προστεθεί στη λειτουργική μονάδα app ακόμα και όταν η γονική καρτέλα είναι τι θα εμφανίζεται στα αποτελέσματα.
