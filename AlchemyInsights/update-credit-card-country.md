# <a name="update-credit-card-country-information"></a>Ενημέρωση πληροφοριών χώρας πιστωτικής κάρτας

Δεν μπορείτε να ενημερώσετε τις πληροφορίες χώρας για τη μέθοδο πληρωμής σας. Η χώρα που έχει εκχωρηθεί στην πληρωμή είναι η χώρα που επιλέχθηκε κατά τη ρύθμιση του οργανισμού σας στο Microsoft 365.
