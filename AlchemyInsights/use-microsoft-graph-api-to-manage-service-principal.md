# <a name="use-microsoft-graph-api-to-manage-service-principal"></a>Χρήση του Microsoft Graph API για τη διαχείριση της κύριας υπηρεσίας

Με το [κύριο API υπηρεσίας](https://docs.microsoft.com/graph/api/resources/serviceprincipal), μπορείτε να διαχειριστείτε μέσω προγραμματισμού τις παρουσίες των εφαρμογών και να ελέγξετε τι μπορεί να κάνει μια εφαρμογή εντός του μισθωτή σας.

Για περισσότερες πληροφορίες σχετικά με τη σχέση μεταξύ εφαρμογών και αρχών υπηρεσίας, ανατρέξτε [στο θέμα εφαρμογές και κύρια αντικείμενα υπηρεσίας στο Azure Active Directory](https://docs.microsoft.com/azure/active-directory/develop/app-objects-and-service-principals).
