# <a name="edit-a-read-only-file"></a>Επεξεργασία ενός αρχείου μόνο για ανάγνωση

Σε βιβλιοθήκες εγγράφων, όταν κάνετε κλικ στο όνομα του εγγράφου, το πρόγραμμα του Office ενδέχεται να ανοίγουν απροσδόκητα το ως αρχείο μόνο για ανάγνωση. Για να το ανοίξετε για επεξεργασία, κάντε κλικ στο κουμπί **Άνοιγμα** (τρεις τελείες) και, στη συνέχεια, κάντε κλικ στο κουμπί **Επεξεργασία**
