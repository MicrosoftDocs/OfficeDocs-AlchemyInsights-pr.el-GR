# <a name="admin-roles"></a>Ρόλοι διαχειριστή

- Επιλέξτε [Ρόλοι](https://admin.microsoft.com/Adminportal/Home#/roles) στο κέντρο διαχείρισης για να εκχωρήσετε ρόλους διαχειριστή ή να μάθετε περισσότερα.

- Στη συνέχεια, επιλέξτε τα ονόματα των ρόλων και κάντε κλικ στην **εκχώρηση ρόλου** ή κλικ στην επιλογή **σύγκριση ρόλων** για να μάθετε περισσότερα.
