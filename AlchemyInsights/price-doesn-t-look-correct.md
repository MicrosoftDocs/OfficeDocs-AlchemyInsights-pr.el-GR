# <a name="price-doesnt-look-correct"></a>Η τιμή δεν φαίνεται σωστή;

Αν η τιμή στο τιμολόγιό σας δεν φαίνεται σωστή, αυτό μπορεί να οφείλεται σε αλλαγή τιμής κατά την ανανέωση. Εάν δεν πιστεύετε ότι αυτό εξηγεί το ζήτημα, παρακαλούμε χρησιμοποιήστε την επιλογή "Αφήστε μας να σας καλέσουμε" για να μιλήσετε με έναν μηχανικό υποστήριξης.
