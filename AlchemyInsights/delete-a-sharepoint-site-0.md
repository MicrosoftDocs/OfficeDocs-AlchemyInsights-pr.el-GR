# <a name="delete-sites-that-belong-to-an-office-365-group"></a>Διαγραφή τοποθεσιών που ανήκουν σε μια ομάδα του Office 365

Όταν οι χρήστες δημιουργούν μια νέα τοποθεσία ομάδας από το SharePoint Home ή μπορείτε να δημιουργήσετε μια νέα τοποθεσία ομάδας από το νέο κέντρο διαχείρισης του SharePoint, δημιουργείται αυτόματα μια αντίστοιχη ομάδα Office 365. Οι τοποθεσίες ομάδας που συνδέονται με την ομάδα Office 365 δημιουργούνται επίσης όταν δημιουργείτε μια ομάδα Office 365 ή μια ομάδα στις ομάδες της Microsoft. Για να διαγράψετε μια τοποθεσία του SharePoint που ανήκει σε μια ομάδα Office 365, πρέπει να διαγράψετε την ίδια την ομάδα του Office 365. 
  
1. Μεταβείτε στη [σελίδα ομάδες του κέντρου διαχείρισης Microsoft 365](https://portal.office.com/adminportal/home#/groups).
    
2. Επιλέξτε το πλαίσιο ελέγχου δίπλα στην ομάδα.
    
3. Επιλέξτε **Διαγραφή**.
    
ΣΗΜΕΙΩΣΗ: όταν διαγράφετε μια ομάδα, μπορεί να υπάρξει καθυστέρηση πριν από την κατάργηση της τοποθεσίας από το SharePoint.
  
**Διαγραφή τοποθεσιών επικοινωνίας ή κλασικών τοποθεσιών:**

1. Πήγαινε στην τοποθεσία.
  
2. Κάντε κλικ στο εικονίδιο ρυθμίσεων στην επάνω δεξιά **περιοχή** . 
  
3. Κάντε κλικ **στις ρυθμίσεις τοποθεσίας**. Στην περιοχή **πληροφορίες τοποθεσίας** ή **Ενέργειες τοποθεσίας**, επιλέξτε **Διαγραφή αυτής της τοποθεσίας**και, στη συνέχεια, κάντε κλικ στο κουμπί **Διαγραφή**.
  
**Διαγραφή μιας συλλογής τοποθεσιών του SharePoint:**

Για να διαγράψετε μια τοποθεσία:
  
1. Στο νέο κέντρο διαχείρισης του SharePoint, στη σελίδα "τοποθεσίες", κάντε κλικ στο **πλαίσιο ελέγχου** δίπλα στην τοποθεσία που θέλετε να διαγράψετε. 
    
2. Στη γραμμή εργαλείων κορδέλας, κάντε κλικ στο κουμπί **Διαγραφή.**
    
**Διαγράφηκε μια τοποθεσία κατά λάθος;**

Ανατρέξτε [στην περιοχή Επαναφορά διαγραμμένης συλλογής τοποθεσιών](https://go.microsoft.com/fwlink/?linkid=867660).
