# <a name="delete-a-sharepoint-site-collection"></a>Διαγραφή μιας συλλογής τοποθεσιών του SharePoint

1. Στο κλασικό στο Κέντρο διαχείρισης του SharePoint, στη σελίδα "συλλογές τοποθεσιών", επιλέξτε το πλαίσιο ελέγχου δίπλα από τη συλλογή τοποθεσιών.
    
2. Στην καρτέλα "συλλογές τοποθεσιών", επιλέξτε " **Διαγραφή**".
