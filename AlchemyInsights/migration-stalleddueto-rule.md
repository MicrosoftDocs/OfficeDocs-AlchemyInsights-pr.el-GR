# <a name="stalleddueto-errors-during-migration"></a>Σφάλματα χρονοτριβείπροτεσε τη μετεγκατάσταση

Μπορεί να δείτε μια ποικιλία σφαλμάτων που αρχίζουν με "StalledDueTo..." κατά τη μεταποίησή σας στο Exchange Online. Πολλά από αυτά τα σφάλματα είναι απολύτως φυσιολογικά. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Επιτάχυνση βάσει πόρων και Προτεραιότητα στο Exchange Online Μετεγκαταστάσεις](https://blogs.technet.microsoft.com/exchange/2018/06/25/resource-based-throttling-and-prioritization-in-exchange-online-migrations/).
