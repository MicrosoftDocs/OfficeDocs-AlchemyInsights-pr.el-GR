# <a name="learn-about-microsoft-365-groups"></a>Μάθετε σχετικά με τις ομάδες του Microsoft 365

Οι ομάδες Microsoft 365 είναι μια υπηρεσία που λειτουργεί με τα εργαλεία του Office 365 που χρησιμοποιείτε ήδη, ώστε να μπορείτε να συνεργάζεστε με τους συναδέλφους σας κατά τη σύνταξη εγγράφων, τη δημιουργία υπολογιστικών φύλλων, την εργασία σε σχέδια έργων, τον προγραμματισμό συσκέψεων ή την αποστολή μηνυμάτων ηλεκτρονικού ταχυδρομείου. Για περισσότερες πληροφορίες, ανατρέξτε στο [θέμα μάθετε σχετικά με τις ομάδες του Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
