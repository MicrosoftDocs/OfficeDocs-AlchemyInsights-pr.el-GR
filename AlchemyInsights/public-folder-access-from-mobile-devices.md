# <a name="public-folder-access-from-mobile-devices"></a>Πρόσβαση δημόσιου φακέλου από κινητές συσκευές

Η πρόσβαση σε δημόσιους φακέλους δεν λειτουργεί με τη χρήση κινητών συσκευών, χρησιμοποιήστε το OWA (Outlook στο Web) ή το Outlook για υπολογιστή με Windows ή το Outlook για Mac για να αποκτήσετε πρόσβαση σε δημόσιους φακέλους.

Εάν θέλετε να έχετε λειτουργίες παρόμοιες με τους δημόσιους φακέλους και να έχετε περιεχόμενο προσβάσιμο σε κινητές συσκευές, ανατρέξτε στο [θέμα μάθετε σχετικά με τις ομάδες του Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) για μια καλύτερη εναλλακτική λύση.
