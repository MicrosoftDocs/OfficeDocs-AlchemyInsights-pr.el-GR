# <a name="webresource-visibility-property"></a>Ιδιότητα Ορατότητα Webresource

Πρέπει να ενεργοποιήσετε την επιλογή **Ορατό από προεπιλογή** για να εμφανίζεται ο πόρος web στη φόρμα και να είναι διαθέσιμος στους χρήστες. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ιδιότητες πόρων Web για φόρμες εφαρμογών που βασίζονται σε μοντέλο](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy).
