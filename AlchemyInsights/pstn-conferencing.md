# <a name="pstn-calling-with-skype-for-business-online"></a>Κλήσεις PSTN με το Skype για ηλεκτρονικές συναλλαγές

Για να πραγματοποιήσετε κλήσεις PSTN στο Microsoft 365, χρειάζεστε ένα [πρόγραμμα κλήσεων του microsoft 365](https://docs.microsoft.com/microsoftteams/what-is-phone-system-in-office-365#more-about-calling-plans) ή μια σύνδεση με την υποδομή τηλεφωνίας του PSTN εσωτερικής εγκατάστασης. Για περισσότερες πληροφορίες, ανατρέξτε στους ακόλουθους πόρους: 

- [Διαθεσιμότητα χώρας και περιοχής για προγράμματα φωνητικής διάσκεψης και κλήσεων](https://docs.microsoft.com/microsoftteams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans) 
- [Συνήθεις ερωτήσεις για τη διάσκεψη ήχου](https://docs.microsoft.com/microsoftteams/audio-conferencing-common-questions)
- [Προσθήκη κεφαλαίων και διαχείριση πιστώσεων επικοινωνίας](https://docs.microsoft.com/microsoftteams/add-funds-and-manage-communications-credits)
- [Οι καλούντες PSTN δεν μπορούν να συμμετέχουν σε κλήση διάσκεψης με κλήση σύνδεσης του Skype για Business](https://docs.microsoft.com/SkypeForBusiness/troubleshoot/online-conferencing/pstn-callers-cant-join-dial-in-call)
