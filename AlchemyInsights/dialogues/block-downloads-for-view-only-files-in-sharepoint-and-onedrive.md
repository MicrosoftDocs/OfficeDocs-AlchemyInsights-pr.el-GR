# <a name="block-downloads-for-view-only-files-in-sharepoint-and-onedrive"></a>Αποκλεισμός λήψεων για αρχεία μόνο για προβολή στο SharePoint και το OneDrive

Όταν κάνετε κοινή χρήση στοιχείων με άλλους χρήστες στο OneDrive ή στο SharePoint, μπορείτε να τους δώσετε πρόσβαση για προβολή, αλλά όχι λήψη. Για περισσότερες πληροφορίες, ανατρέξτε [στο θέμα "Αποκλεισμός λήψεων για αρχεία μόνο για προβολή" στο SharePoint και το OneDrive.](https://support.microsoft.com/office/block-downloads-for-view-only-files-in-sharepoint-and-onedrive-6051184b-62ac-4149-b874-13dcd40ef91e)
