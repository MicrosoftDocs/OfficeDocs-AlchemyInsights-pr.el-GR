# <a name="buy-a-new-domain-from-microsoft"></a>Αγορά νέου τομέα από τη Microsoft

Για να αγοράσετε έναν νέο τομέα από τη Microsoft, εκτελέστε τα ακόλουθα βήματα:

1. Μεταβείτε στην **επιλογή "> Τομέων" και** κάντε κλικ στην **επιλογή "Αγορά τομέα".** 
2. Ακολουθήστε τις οδηγίες στον οδηγό.
