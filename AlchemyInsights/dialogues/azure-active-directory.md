# <a name="azure-active-directory"></a>Azure Active Directory

Το Microsoft Azure Active Directory (Azure AD) είναι μια υπηρεσία διαχείρισης ταυτοτήτων και πρόσβασης που βασίζεται στο cloud, η οποία βοηθά τους υπαλλήλους να πραγματοποιήσουν είσοδο και πρόσβαση σε πόρους τόσο εξωτερικά όσο και εσωτερικά. Για να μάθετε περισσότερα, ανατρέξτε στους ακόλουθους πόρους:

- [Τι είναι το Azure Active Directory;](https://go.microsoft.com/fwlink/?linkid=2081145)
- [Ενοποίηση του Azure με το Office 365](https://go.microsoft.com/fwlink/?linkid=2081218)
- [Δωρεάν δοκιμαστική έκδοση του Azure Active Directory](https://go.microsoft.com/fwlink/?linkid=2081144)
