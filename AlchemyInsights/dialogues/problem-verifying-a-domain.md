# <a name="problem-verifying-a-domain"></a>Πρόβλημα επαλήθευσης τομέα

Εάν το Office 365 δεν επαληθεύσει τον τομέα σας σε μία ή δύο ώρες, υπάρχει πρόβλημα.

Επιλέξτε **"Όχι"** στις παρακάτω  οδηγίες για να δηλώσετε ότι δεν έλυνε το πρόβλημά σας και, στη συνέχεια, θα δείτε τις επιλογές για τη συζήτηση με έναν άνθρωπο.
