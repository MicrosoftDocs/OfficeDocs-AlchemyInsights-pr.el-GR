# <a name="trying-to-remove-your-domain"></a>Θέλετε να καταργήσετε τον τομέα σας;

Για να καταργήσετε τον τομέα σας από τη Microsoft, ακολουθήστε τα βήματα αυτού του άρθρου: [Κατάργηση τομέα](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain)
  
Θα χρειαστεί να ενημερώσετε τους χρήστες ή τις ομάδες που έχουν αντιστοιχιστεί σε αυτό το όνομα τομέα, πριν να καταργήσετε τον τομέα.
  
[Ελέγξτε το άρθρο τομέων](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider) για άλλες κοινές λύσεις.
