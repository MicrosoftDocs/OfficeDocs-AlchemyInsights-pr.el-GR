# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Σφάλμα του Outlook: Δεν είναι δυνατή η εκκίνηση του Microsoft Outlook

Αντιμετωπίζετε προβλήματα με την εκκίνηση του Outlook 2016, του Outlook 2013 ή του Outlook 2010 ή της λήψης "Δεν είναι δυνατή η εκκίνηση του Microsoft Outlook. Δεν είναι δυνατό το άνοιγμα των μηνυμάτων λάθους του παραθύρου του Outlook;;
  
Αρχικά, χρησιμοποιήστε τα βήματα αντιμετώπισης προβλημάτων σε [αυτό το άρθρο](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) για να επιλύσετε αυτά τα ζητήματα εκκίνησης. 
  
Μπορείτε επίσης να εκτελέσετε το [εργαλείο σύνθετων διαγνωστικών](https://aka.ms/SaRA-OutlookAdvDiagnostics) στον υπολογιστή που επηρεάζεται.
