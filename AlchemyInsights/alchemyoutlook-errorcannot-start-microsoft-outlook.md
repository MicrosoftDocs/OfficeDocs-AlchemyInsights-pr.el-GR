# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Το Outlook σφάλμα: "Δεν είναι δυνατή η εκκίνηση του Microsoft Outlook"

Αντιμετωπίζετε προβλήματα κατά την εκκίνηση του Outlook 2016, 2013 το Outlook ή το Outlook 2010 ή παραλαβή "δεν είναι δυνατή εκκίνηση του Microsoft Outlook. Δεν είναι δυνατό το άνοιγμα του παραθύρου του Outlook"μηνύματα λάθους;
  
Πρώτα, χρησιμοποιήστε τα βήματα αντιμετώπισης προβλημάτων σε [αυτό το άρθρο](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) για να επιλύσετε αυτά τα ζητήματα εκκίνησης. 
  
Μπορείτε επίσης να εκτελέσετε το [εργαλείο διαγνωστικών για προχωρημένους](https://aka.ms/SaRA-OutlookAdvDiagnostics) στον υπολογιστή που επηρεάζεται.
