# <a name="activate-or-reactivate-a-subscription"></a>Ενεργοποίηση ή επανενεργοποίηση μιας συνδρομής

**Ενεργοποίηση συνδρομής**

- Για να παραχωρήσετε στους χρήστες πρόσβαση σε μια υπηρεσία, εκχωρήστε τους μια άδεια χρήσης. Για να μάθετε πώς, ανατρέξτε [στο θέμα εκχώρηση αδειών χρήσης σε χρήστες](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).
- Εάν έχετε ήδη μια άδεια χρήσης που σας έχει ανατεθεί, μπορείτε να εντοπίσετε τις εφαρμογές που έχετε τη δυνατότητα να κάνετε λήψη από τη [σελίδα "ο λογαριασμός μου](https://portal.office.com/account/#installs)". Αυτή η σελίδα παρέχει μια λίστα με τις εφαρμογές που είναι διαθέσιμες για λήψη με βάση τις άδειες χρήσης που έχουν εκχωρηθεί.
- Εάν έχετε ήδη κάνει λήψη των εφαρμογών του Office, ίσως χρειαστεί να πραγματοποιήσετε είσοδο στις εφαρμογές με τον εταιρικό ή σχολικό λογαριασμό σας. Αυτό μπορείτε να το κάνετε σε οποιαδήποτε εφαρμογή του Office (Word, Excel, κ. λπ. **File**) κάνοντας κλικ στην επιλογή  >  **λογαριασμός του Office** File (κοντά στο κάτω μέρος). Στην περιοχή **πληροφορίες χρήστη**, επιλέξτε **Αλλαγή λογαριασμού**. Για περισσότερες πληροφορίες, ανατρέξτε [στο θέμα εγκατάσταση εφαρμογών του Office](https://docs.microsoft.com/microsoft-365/admin/setup/install-applications).

**Επανενεργοποίηση μιας συνδρομής**

Εάν η συνδρομή σας λήξει, έχει απενεργοποιηθεί από τη Microsoft ή την ακυρώσατε στο μέσο μιας περιόδου συνδρομής, μπορείτε να την ενεργοποιήσετε εκ νέου στο κέντρο διαχείρισης.
  
1. Στο κέντρο διαχείρισης, μεταβείτε στη σελίδα **χρέωση**των  >  [προϊόντων σας](https://go.microsoft.com/fwlink/p/?linkid=842054) .
2. Στην καρτέλα **προϊόντα** , εντοπίστε τη συνδρομή που θέλετε να ενεργοποιήσετε ξανά.
3. Επιλέξτε **περισσότερες ενέργειες** (τρεις τελείες) και, στη συνέχεια, επιλέξτε **Επανενεργοποίηση αυτής της συνδρομής**.
    > [!IMPORTANT]
    > Εάν δεν βλέπετε την εκ **νέου ενεργοποίηση αυτής της συνδρομής** ως διαθέσιμη ενέργεια, [επικοινωνήστε](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products) με την υποστήριξη για να ενεργοποιήσετε ξανά τη συνδρομή σας.
4. Στο παράθυρο **Επανενεργοποίηση αυτής της συνδρομής** , επιλέξτε έναν υπάρχοντα τρόπο πληρωμής ή επιλέξτε **Προσθήκη μεθόδου πληρωμής**.
5. Επιλέξτε **Επανενεργοποίηση**.

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [Επανενεργοποίηση της συνδρομής σας](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/reactivate-your-subscription).

**Ενεργοποίηση νέας συνδρομής με έναν αριθμό-κλειδί προϊόντος**

1. Για να ενεργοποιήσετε μια νέα συνδρομή Microsoft 365 με τον αριθμό-κλειδί προϊόντος, ανατρέξτε στο θέμα [Πού μπορώ να εισαγάγω τον αριθμό-κλειδί προϊόντος του Office](https://support.office.com/article/where-to-enter-your-office-product-key-0a82e5ae-739e-4b92-a6f4-2ec780c185db).
2. Εισαγάγετε τον αριθμό-κλειδί προϊόντος του Microsoft 365 και ακολουθήστε τις οδηγίες για να ενεργοποιήσετε τη συνδρομή σας. Εάν αντιμετωπίζετε προβλήματα με την ενεργοποίηση, ανατρέξτε σε αυτή τη λίστα με τα [συνηθισμένα σφάλματα και λύσεις για τα βασικά προϊόντα](https://docs.microsoft.com/microsoft-365/commerce/product-key-errors-and-solutions).
