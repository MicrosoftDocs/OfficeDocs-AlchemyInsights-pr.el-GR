# <a name="get-help-with-windows-10-desktop"></a>Λήψη βοήθειας για τα Windows 10 Desktop

- Για να δημιουργήσετε μια συντόμευση επιφάνειας εργασίας, [ανατρέξτε στο θέμα Καρφίτσωμα εφαρμογών και φακέλων στην επιφάνεια εργασίας ή στη γραμμή εργασιών.](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0)

- Για να κάνετε τα εικονίδια ή το κείμενο μεγαλύτερα στα Windows 10, ανατρέξτε στο [θέμα Αλλαγή του μεγέθους των εικονιδίων της επιφάνειας εργασίας σας.](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211)

- Για να εμφανίσετε εικονίδια επιφάνειας εργασίας στα Windows 10, ανατρέξτε στο θέμα [Εμφάνιση εικονιδίων επιφάνειας εργασίας στα Windows 10.](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20)
