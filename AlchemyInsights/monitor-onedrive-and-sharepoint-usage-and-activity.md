# <a name="view-reports-on-onedrive-and-sharepoint-activity-and-usage"></a>Προβολή εκθέσεων για τη δραστηριότητα του OneDrive και του SharePoint και χρήση

Μεταβείτε στη [σελίδα αναφορές χρήσης στο Κέντρο διαχείρισης Microsoft 365](https://admin.microsoft.com/AdminPortal/Home). Στην περιοχή **Επιλέξτε μια αναφορά**, επιλέξτε την OneDrive ή SharePoint αναφορά δραστηριότητας ή χρήσης.
