# <a name="new-feature-to-help-diagnose-eop-and-atp"></a>Νέα δυνατότητα για βοήθεια στη διάγνωση του EOP και του ATP

Ως διαχειριστής, μπορείτε πλέον να αναφέρετε εύκολα τα ύποπτα μηνύματα ηλεκτρονικού ταχυδρομείου, τις διευθύνσεις URL και το περιεχόμενο στη Microsoft στην [Εξερεύνηση υποβολών](https://protection.office.com/reportsubmission). Μπορείτε ακόμη και να το χρησιμοποιήσετε για να καταλάβετε γιατί ένα νόμιμο μήνυμα επισημάνθηκε εσφαλμένα. Για περισσότερες πληροφορίες, ανατρέξτε σε αυτήν την [ανακοίνωση](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).
