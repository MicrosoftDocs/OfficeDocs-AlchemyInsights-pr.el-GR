# <a name="new-feature-to-help-diagnose-eop-and-atp"></a>Νέα δυνατότητα για τη διάγνωση του EOP και της ATP

Ως διαχειριστής, μπορείτε πλέον να αναφέρετε εύκολα ύποπτα μηνύματα ηλεκτρονικού ταχυδρομείου, διευθύνσεις URL και περιεχόμενο στη Microsoft στην [Εξερεύνηση υποβολών](https://protection.office.com/reportsubmission). Μπορείτε ακόμη να το χρησιμοποιήσετε για να κατανοήσετε γιατί ένα νόμιμο μήνυμα δεν έχει επισημανθεί σωστά. Για περισσότερες πληροφορίες, ανατρέξτε στην παρούσα [ανακοίνωση](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).
