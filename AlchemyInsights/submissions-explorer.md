# <a name="new-feature-to-help-diagnose-eop-and-atp"></a>Νέα δυνατότητα για να διαγνώσετε EOP και ATP

Ως ένας διαχειριστής, μπορείτε να τώρα εύκολα αναφέρετε μηνύματα ύποπτο μήνυμα ηλεκτρονικού ταχυδρομείου, διευθύνσεις URL και περιεχόμενο στη Microsoft στην [Εξερεύνηση υποβολές](https://protection.office.com/reportsubmission). Μπορείτε ακόμη και να το χρησιμοποιήσετε για να σας βοηθήσει να κατανοήσετε γιατί ένα αξιόπιστο μήνυμα έχει επισημανθεί εσφαλμένα. Για περισσότερες πληροφορίες, ανατρέξτε στην ενότητα αυτήν την [ανακοίνωση](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).
