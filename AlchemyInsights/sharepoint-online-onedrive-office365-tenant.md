# <a name="sharepoint-online-or-onedrive-setting-up"></a>Ηλεκτρονική SharePoint ή OneDrive "ρύθμιση"

Όταν ένας χρήστης προστίθεται σε μια υπάρχουσα Office 365 μισθωτών, ενδέχεται να δείτε το SharePoint Online ή OneDrive για επαγγελματική εικονίδια που αναφέρονται ως "ρύθμιση" μέχρι να ολοκληρωθεί η διαδικασία προετοιμασίας.
Ακολουθήστε αυτόν τον [Οδηγό αντιμετώπισης προβλημάτων](https://docs.microsoft.com/sharepoint/support/sites/troubleshooting-guide-for-sites-stopped-at-provisioning) για βοήθεια.

Εάν έχουν περάσει περισσότερες από 24 ώρες από την πρώτη φορά που είδατε αυτό το θέμα, παρακαλούμε καταγράψτε μια υπόθεση υποστήριξης.

Καταλαβαίνουμε ότι 24 ώρες μπορεί να φαίνεται σαν ένα μεγάλο χρονικό διάστημα. Σε πολλές περιπτώσεις, είμαστε ήδη εργάζονται σε μια λύση. Παρακαλώ δώστε μας τουλάχιστον 24 ώρες για να ολοκληρώσουμε μια λύση.
