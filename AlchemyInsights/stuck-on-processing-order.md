# <a name="stuck-on-processing-order"></a>Κολλημένοι στη διαδικασία επεξεργασίας της παραγγελίας

Εάν έχετε επιλέξει να αναβαθμίσετε τη συνδρομή σας στο Microsoft 365, αλλά η εντολή έχει κολλήσει στη θέση "Επεξεργασία της παραγγελίας σας", ανατρέξτε στη δημοσίευση [Πίνακας εργαλείων εύρυθμης λειτουργίας υπηρεσίας](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/servicehealth)**SP207374**. Η διαδικασία αναβάθμισης του μισθωτή σας θα απαιτήσει προμήθεια, η οποία μπορεί να καθυστερήσει. Αναμένουμε την ολοκλήρωση της διαδικασίας εντός 24 ωρών. Ωστόσο, έχουν υπάρξει σπάνιες περιπτώσεις όπου για την ολοκλήρωσή τους απαιτούνται έως και 48 ώρες. Εάν η επεξεργασία της παραγγελίας διαρκέσει πάνω από 48 ώρες, δημιουργήστε μια αίτηση υπηρεσίας.
