# <a name="sign-in-and-authentication"></a>Είσοδος και έλεγχος ταυτότητας

Microsoft Edge προσφέρει αυτόματες και μη αυτόματες μεθόδους εισόδου στην τοποθεσία Web. Μπορείτε επίσης να επιλέξετε ένα SSO για μία μόνο επιλογή είσοδου. Για να μάθετε περισσότερα σχετικά με την είσοδο στον Edge, ανατρέξτε [στο θέμα Πώς οι χρήστες μπορούν να πραγματοποιήσουν Microsoft Edge.](https://docs.microsoft.com/deployedge/microsoft-edge-security-identity#how-users-can-sign-into-microsoft-edge)
