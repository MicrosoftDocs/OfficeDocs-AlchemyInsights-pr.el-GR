# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Χρήση συνθηκών ή προηγμένων τύπων σε μια ροή

Στο Microsoft Flow, μπορείτε να προσαρμόσετε μια ροή, ώστε να εκτελεί εργασίες μόνο εάν μια συνθήκη είναι αληθής. Για πληροφορίες, ανατρέξτε στο θέμα [Προσθήκη συνθήκης σε ροή](https://go.microsoft.com/fwlink/?linkid=872112).
  
Για να δημιουργήσετε έναν προηγμένο τύπο, ανατρέξτε στο θέμα [Σχήμα γλώσσας ορισμού ροής εργασίας για εφαρμογές λογικής Azure](https://aka.ms/logicexpressions).
