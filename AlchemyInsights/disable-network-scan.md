# <a name="disable-network-scan"></a>Απενεργοποίηση σάρωσης δικτύου

Οι σαρώσεις κοινής χρήσης δικτύου ενδέχεται να επηρεάσουν τις επιδόσεις.  Για να εξασφαλίσετε ότι το πρόγραμμα-πελάτης δεν σαρώνει κοινόχρηστα στοιχεία δικτύου/αρχεία από προεπιλογή, ρυθμίστε τις παραμέτρους των ακόλουθων ρυθμίσεων στην εφαρμογή Windows Defender στην **τιμή True:**

- DisableScanningMappedNetworkDrivesForFullScan
- DisableScanningNetworkFiles
