# <a name="why-cant-i-edit-this-file"></a>Γιατί δεν μπορώ να επεξεργαστώ αυτό το αρχείο;

Εάν ένα έγγραφο, ένα φύλλο εργασίας ή μια παρουσίαση είναι κλειδωμένο για επεξεργασία και δεν είναι δυνατή η επεξεργασία του στις εφαρμογές υπολογιστή με Windows, ενδέχεται το αρχείο να έχει υποστεί επεξεργασία από κάποιον άλλο ή να έχει γίνει ανάληψη ελέγχου του αρχείου.

Ρίξτε μια γρήγορη ματιά σε κάποιους από τους λόγους για τους [οποίους ενδέχεται να μην μπορείτε να επεξεργαστείτε το αρχείο](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
