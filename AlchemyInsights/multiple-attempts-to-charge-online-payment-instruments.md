# <a name="past-due-account"></a>Ληξιπρόθεσμος λογαριασμός;

Μια πληρωμή μπορεί να απορριφθεί για διάφορους λόγους, συμπεριλαμβανομένου του εάν η πιστωτική κάρτα έχει λήξει. Μπορείτε να χρησιμοποιήσετε τη δυνατότητα **Τακτοποίηση υπολοίπου** ανά πάσα στιγμή για να αλλάξετε τα στοιχεία της πιστωτικής κάρτας σας και να υποβάλετε άμεσα την πληρωμή.

1. Στο κέντρο διαχείρισης, μεταβείτε στην καρτέλα **χρεώσεις > [προϊόντα & Services](https://go.microsoft.com/fwlink/p/?linkid=842054)**.
Εύρεση της συνδρομής με το υπόλοιπο που εκκρεμεί. Στην ενότητα " **χρέωση** ", δίπλα στη **συνδρομή έχει**ληξιπρόθεσμη πληρωμή, κάντε κλικ στην επιλογή " **Τακτοποίηση υπολοίπου** " για να πληρώσετε ολόκληρο το εκκρεμές υπόλοιπο.
2. Εάν είστε ο κάτοχος της πιστωτικής κάρτας, μπορείτε να αλλάξετε την ημερομηνία λήξης, να επιλέξετε μια διαφορετική κάρτα ή να [προσθέσετε μια νέα κάρτα](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/manage-payment-methods?view=o365-worldwide).
3. Εάν δεν είστε κάτοχος της πιστωτικής κάρτας, μπορείτε να επιλέξετε μια διαφορετική πιστωτική κάρτα ή να [προσθέσετε μια νέα κάρτα](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/manage-payment-methods?view=o365-worldwide).
4. Αφού ολοκληρώσετε τις ενημερώσεις σας, κάντε κλικ στην επιλογή **Αποθήκευση**.

**Σημείωση**: μετά την πραγματοποίηση μιας πληρωμής, η προηγούμενη ειδοποίηση μπορεί να εξακολουθεί να είναι ορατή για σύντομο χρονικό διάστημα. Παρακαλούμε περιμένετε τουλάχιστον **48 ώρες** πριν επικοινωνήσετε με την υποστήριξη.
