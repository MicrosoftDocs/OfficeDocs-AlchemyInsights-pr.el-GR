# <a name="microsoft-stream-licensing-overview"></a>Επισκόπηση της άδειας χρήσης ροής της Microsoft

Το Microsoft Stream είναι μια υπηρεσία cloud με άδεια χρήσης ανά συνδρομητική βάση χρήστη. Διατίθεται ως μέρος των επιλέξιμων συνδρομών του Office 365. Ανατρέξτε στην ενότητα Τιμολόγηση του Office 365 για πληροφορίες σχετικά με κάθε τύπο συνδρομής.

- [Επισκόπηση της άδειας χρήσης ροής της Microsoft](https://docs.microsoft.com/en-us/stream/license-overview)
