# <a name="microsoft-stream-licensing-overview"></a>Επισκόπηση της άδειας χρήσης ροής της Microsoft

Το Microsoft Stream είναι μια υπηρεσία cloud με άδεια χρήσης ανά συνδρομητική βάση χρήστη. Διατίθεται ως μέρος των επιλέξιμων συνδρομών microsoft 365. Ανατρέξτε στην ενότητα Τιμολόγηση της Microsoft 365 για πληροφορίες σχετικά με κάθε τύπο συνδρομής.

- [Επισκόπηση της άδειας χρήσης ροής της Microsoft](https://docs.microsoft.com/stream/license-overview)
