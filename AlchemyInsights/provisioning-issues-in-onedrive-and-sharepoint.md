# <a name="provisioning-issues-in-onedrive-and-sharepoint"></a>Προμήθεια ζητημάτων στο OneDrive και το SharePoint

Όταν ένας χρήστης προστίθεται σε έναν υπάρχοντα μισθωτή του Office 365, ενδέχεται να δείτε το SharePoint Online ή OneDrive για επαγγελματική εικονίδια που αναφέρονται ως "ρύθμιση" μέχρι να ολοκληρωθεί η διαδικασία προμήθειας. Για τους λογαριασμούς που δημιουργήθηκαν πρόσφατα, οι χρήστες πρέπει να εισέλθετε στο κέντρο διαχείρισης Microsoft 365 με το δικό τους λογαριασμό και, στη συνέχεια, κάντε κλικ στο κουμπί OneDrive για να ξεκινήσετε με μη αυτόματο τρόπο την παροχή της τοποθεσίας του OneDrive.
  
Εάν έχουν περάσει περισσότερες από 24 ώρες από την πρώτη φορά που είδατε αυτό το ζήτημα, παρακαλούμε να καταγράψετε μια υπόθεση υποστήριξης. Σε πολλές περιπτώσεις, ήδη εργαζόμαστε για μια λύση. Παρακαλώ δώστε μας τουλάχιστον 24 ώρες για να ολοκληρώσουμε μια λύση.
