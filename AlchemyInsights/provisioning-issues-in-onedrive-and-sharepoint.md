# <a name="provisioning-issues-in-onedrive-and-sharepoint"></a>Ζητήματα προμήθειας στο OneDrive και το SharePoint

Όταν ένας χρήστης προστίθεται σε έναν υπάρχοντα μισθωτή, μπορεί να δει τα εικονίδια του SharePoint Online ή του OneDrive για επαγγελματικές που αναφέρονται ως "ρύθμιση" μέχρι να ολοκληρωθεί η διαδικασία προμήθειας. Για τους λογαριασμούς που δημιουργήθηκαν πρόσφατα, οι χρήστες πρέπει να πραγματοποιήσουν είσοδο στο κέντρο διαχείρισης του Microsoft 365 με το δικό τους λογαριασμό και, στη συνέχεια, να κάνουν κλικ στο OneDrive για να ξεκινήσουν με μη αυτόματο τρόπο την παροχή της τοποθεσίας του OneDrive.
  
Εάν έχουν περάσει περισσότερες από 24 ώρες από την πρώτη φορά που είδατε αυτό το πρόβλημα, καταγράψτε μια υπόθεση υποστήριξης. Σε πολλές περιπτώσεις, ήδη εργαζόμαστε σε μια λύση. Παρακαλούμε δώστε μας τουλάχιστον 24 ώρες για να ολοκληρώσετε μια λύση.
