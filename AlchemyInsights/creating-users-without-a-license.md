# <a name="create-mailboxes-that-dont-require-licenses"></a>Δημιουργία γραμματοκιβωτίων που δεν απαιτούν άδειες χρήσης

- Εάν χρειάζεστε μια ομαδική διεύθυνση ηλεκτρονικού ταχυδρομείου για πολλά άτομα, ανατρέξτε στο περίπτωση [Δημιουργία κοινόχρηστου γραμματοκιβωτίου](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox).

- Για να επιτρέψετε σε μια ομάδα ατόμων να συνεργαστεί και να κάνει κοινή χρήση μηνυμάτων ηλεκτρονικού ταχυδρομείου και αρχείων, ανατρέξτε [στο θέμα πληροφορίες σχετικά με τις ομάδες 365 του Office](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
