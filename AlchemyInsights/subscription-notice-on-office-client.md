# <a name="subscription-notice-in-your-office-client"></a>Ειδοποίηση συνδρομής στο πρόγραμμα-πελάτη του Office;

Φαίνεται ότι βλέπετε μια ειδοποίηση στο πρόγραμμα-πελάτη του Office που σχετίζεται με την κατάσταση της συνδρομής σας. Μπορείτε να δείτε πληροφορίες σχετικά με το πώς μπορείτε να διορθώσετε το πρόβλημα σε αυτό το άρθρο: [εμφανίζεται μια ειδοποίηση συνδρομής όταν ανοίγω μια εφαρμογή Microsoft 365](https://support.office.com/article/A-subscription-notice-appears-when-I-open-an-Office-365-application-4cabe32c-f594-4c0e-9191-3d3ade10cceb.aspx)
  
Εάν η συνδρομή σας είναι ενημερωμένη, μπορεί να έχετε επιπλήξει ένα προσωρινό πρόβλημα που προκάλεσε την αποτυχία της ενεργοποίησης του Office. Υπάρχουν απλά βήματα που μπορείτε να ακολουθήσετε για να αποφύγετε αυτά τα μηνύματα, όπως να ελέγξετε την ώρα του υπολογιστή σας, να εκτελέσετε το Office ως διαχειριστής και να συνδεθείτε για να ενεργοποιήσετε. Μπορείτε να δείτε αναλυτικά βήματα για να διορθώσετε αυτά τα προβλήματα σε αυτό το άρθρο: [σφάλματα προϊόντος και ενεργοποίησης χωρίς άδεια χρήσης](https://support.office.com/article/Unlicensed-Product-and-activation-errors-in-Office-0d23d3c0-c19c-4b2f-9845-5344fedc4380.aspx).
  
Εάν αυτά δεν λειτουργήσουν για εσάς, παρακαλούμε επικοινωνήστε μαζί μας με τους παρακάτω συνδέσμους και θα χαρούμε να σας βοηθήσουμε.
