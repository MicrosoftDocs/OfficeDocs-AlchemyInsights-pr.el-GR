# <a name="microsoft-teams---guest-access"></a>Ομάδες της Microsoft - η πρόσβαση ως Guest

Αν θέλετε να ενεργοποιήσετε ή να ρυθμίσετε το ρόλο της πρόσβασης επισκεπτών σε ομάδες της Microsoft, έχουμε έναν οδηγό βήμα προς βήμα οδηγίες για: [Λίστα ελέγχου πρόσβασης επισκεπτών](https://docs.microsoft.com/en-us/microsoftteams/guest-access-checklist)
  
Ίσως χρειαστεί να περιμένετε έως και 24 ώρες για ομάδες αλλαγές ρύθμισης παραμέτρων ώστε να εφαρμόσετε.
