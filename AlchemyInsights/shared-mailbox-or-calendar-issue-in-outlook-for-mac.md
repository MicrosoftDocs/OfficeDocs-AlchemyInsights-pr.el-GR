# <a name="shared-mailbox-or-calendar-issue-in-outlook-for-mac"></a>Κοινόχρηστο γραμματοκιβώτιο ή πρόβλημα ημερολογίου στο Outlook για Mac;

Μπορούμε να σας βοηθήσουμε πιο γρήγορα εάν επικοινωνήσετε απευθείας μαζί μας από την εφαρμογή. **Μεταβείτε στο Mac του χρήστη που επηρεάζεται, ανοίξτε το Outlook και κάντε κλικ στην επιλογή βοήθεια \> Επικοινωνήστε με την υποστήριξη**. 
  
Για να μάθετε περισσότερα σχετικά με τη χρήση της υποστήριξης εντός εφαρμογής του Outlook για Mac, ανατρέξτε σε αυτό το άρθρο: [νέα δυνατότητα υποστήριξης επαφών στο Outlook 2016 για Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
