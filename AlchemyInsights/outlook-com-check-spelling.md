# <a name="use-your-browser-to-check-spelling"></a>Χρησιμοποιήστε το πρόγραμμα περιήγησης για να κάνετε ορθογραφικό έλεγχο

Αν και δεν έχει ορθογραφικό έλεγχο συνάρτηση Outlook.com, πολλά προγράμματα περιήγησης web ελέγχουν αυτόματα την ορθογραφία, συμπεριλαμβανομένης της Microsoft άκρη, Internet Explorer 10 και νεότερες εκδόσεις και τρέχουσες εκδόσεις του Firefox, Chrome και Safari. Ελέγξτε τις επιλογές για το πρόγραμμα περιήγησης web για να μάθετε περισσότερα σχετικά με τον έλεγχο ορθογραφίας.
