# <a name="use-your-browser-to-check-spelling"></a>Χρήση του προγράμματος περιήγησής σας για ορθογραφικό έλεγχο

Παρόλο που το Outlook.com δεν διαθέτει λειτουργία ορθογραφικού ελέγχου, πολλά προγράμματα περιήγησης Web ελέγχουν αυτόματα την ορθογραφία σας, συμπεριλαμβανομένων των Microsoft Edge, Internet Explorer 10 και νεότερες εκδόσεις, καθώς και τις τρέχουσες εκδόσεις του Firefox, του Chrome και του Safari. Επιλέξτε τις επιλογές για το πρόγραμμα περιήγησης Web για να μάθετε περισσότερα σχετικά με το πώς μπορείτε να κάνετε ορθογραφικό έλεγχο.
