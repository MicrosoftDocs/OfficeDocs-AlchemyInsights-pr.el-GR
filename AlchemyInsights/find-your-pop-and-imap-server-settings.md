# <a name="find-your-pop-or-imap-server-settings"></a>Εύρεση των ρυθμίσεων διακομιστή POP ή IMAP

Εάν δεν γνωρίζετε ποιες ρυθμίσεις χρειάζεστε για να προσθέσετε το λογαριασμό POP ή IMAP στο Outlook, μπορείτε να τις αναζητήσετε στις [Ρυθμίσεις ηλεκτρονικού ταχυδρομείου pop και IMAP για το Outlook](https://support.office.com/article/8361e398-8af4-4e97-b147-6c6c4ac95353.aspx).

> Γνωρίζατε ότι μπορείτε να [ενεργοποιήσετε και να ρυθμίσετε τις παραμέτρους του POP3 σε ένα διακομιστή Exchange 2016](https://technet.microsoft.com/library/bb124934%28v=exchg.160%29.aspx);
