# <a name="cant-create-a-team-in-teams"></a>Δεν είναι δυνατή η δημιουργία ομάδας στο teams

Ο διαχειριστής των ομάδων σας μπορεί να μην σας έχει δώσει τα απαραίτητα δικαιώματα για τη δημιουργία ομάδων.  

Ανατρέξτε στο θέμα διαχειριστής σχετικά με τη δημιουργία ή την αλλαγή των ρυθμίσεων της ομάδας ασφαλείας του οργανισμού στο Microsoft 365. Εάν έχετε προστεθεί στην ομάδα ασφάλεια, μπορείτε να δημιουργήσετε ομάδες.
