# <a name="create-and-manage-room-mailboxes"></a>Δημιουργήστε και διαχειριστείτε τα γραμματοκιβώτια δωμάτιο

Για να δημιουργήσετε και να τροποποιήσετε τα γραμματοκιβώτια δωμάτιο, ανατρέξτε στην ενότητα [Δημιουργία και διαχείριση γραμματοκιβώτια δωμάτιο](https://technet.microsoft.com/library/jj215781.aspx).
  
Για να μάθετε περισσότερα σχετικά με τα γραμματοκιβώτια δωμάτιο και του εξοπλισμού (που είναι επίσης γνωστή ως γραμματοκιβώτια πόρων), ανατρέξτε στην ενότητα [γραμματοκιβώτια δωμάτιο και του εξοπλισμού](https://support.office.com/article/9f518a6d-1e2c-4d44-93f3-e19013a1552b.aspx).
