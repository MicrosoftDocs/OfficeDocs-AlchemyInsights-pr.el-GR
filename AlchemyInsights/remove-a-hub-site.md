# <a name="remove-a-hub-site"></a>Κατάργηση μιας τοποθεσίας διανομέα

Κατάργηση καταχώρησης μιας τοποθεσίας διανομέα στο νέο κέντρο διαχείρισης του SharePoint
  
1. Στο νέο κέντρο διαχείρισης του SharePoint, στην περιοχή **τοποθεσίες**, κάντε κλικ στην επιλογή **ενεργές τοποθεσίες**.

2. Επιλέξτε την τοποθεσία, κάντε κλικ στην **τοποθεσία του διανομέα**και, στη συνέχεια, κάντε κλικ στο κουμπί **κατάργησης καταχώρησης ως τοποθεσία διανομέα**.

3. Κάντε κλικ στο κουμπί **OK**.

[Μάθετε περισσότερα σχετικά με τις τοποθεσίες του διανομέα](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
