# <a name="fix-a-flow-that-failed"></a>Επιδιόρθωση ροής που απέτυχε

1. Μεταβείτε στη [ροή της Microsoft](https://flow.microsoft.com/)και κάντε κλικ στο εικονίδιο ειδοποιήσεων στην επάνω δεξιά γωνία. (Ή στην εφαρμογή για κινητές συσκευές, επιλέξτε την καρτέλα δραστηριότητα.) Επιλέξτε τη ροή που δεν μπόρεσε να κοιτάξει τις λεπτομέρειες.
    
2. Βρείτε το βήμα με το κόκκινο θαυμαστικό εικονίδιο για να δείτε το μήνυμα σφάλματος για τη ροή.
    
3. Ανάλογα με το μήνυμα σφάλματος, θα πρέπει να έχετε τη δυνατότητα να επεξεργαστείτε τη ροή και να τη διορθώσετε. 
    
[Συνηθισμένες αποτυχίες ροής](https://go.microsoft.com/fwlink/?linkid=872110)
