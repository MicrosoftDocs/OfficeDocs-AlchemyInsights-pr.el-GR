# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: Αντιμετώπιση προβλημάτων 'Επεξεργασία αλλαγών'

Εάν μπορείτε να δείτε "Επεξεργασία αλλάζει" για μια παρατεταμένη χρονική περίοδο, αυτό μπορεί να συμβαίνει επειδή έχετε μια ηλεκτρονική ανοίγματος αρχείου, πολλά αρχεία κολλήσει σε μια ουρά ή συγχρονισμό πολύ μεγάλο αρχείο δείτε [OneDrive έχει κολλήσει επεξεργασίας](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Διαβάστε αυτό το άρθρο για την αντιμετώπιση "Αλλαγές επεξεργασίας". Επίσης θα δείτε αυτό το μήνυμα Εάν έχετε ένα αρχείο που αλλάζει συνεχώς (όπως το Outlook. Αρχείο PST ή μια βάση δεδομένων). Μάθετε πώς μπορείτε να [μετακινήσετε τα αρχεία δεδομένων του Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) που χρησιμοποιείτε τακτικά από OneDrive.
