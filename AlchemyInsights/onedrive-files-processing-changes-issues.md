# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: αντιμετώπιση προβλημάτων "επεξεργασία αλλαγών"

Εάν δείτε "επεξεργασία αλλαγών" για ένα παρατεταμένο χρονικό διάστημα, θα μπορούσε να οφείλεται στο γεγονός ότι έχετε ένα ηλεκτρονικό αρχείο ανοιχτό, πολλά αρχεία που έχουν κολλήσει σε μια ουρά ή ένα πολύ μεγάλο συγχρονισμό αρχείων, δείτε [OneDrive κολλήσει στην επεξεργασία](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Διαβάστε αυτό το άρθρο για την αντιμετώπιση προβλημάτων "επεξεργασία αλλαγών". Θα δείτε επίσης αυτό το μήνυμα, εάν έχετε ένα αρχείο που αλλάζει όλη την ώρα (όπως ένα Outlook. Αρχείο PST ή μια βάση δεδομένων). Μάθετε πώς να [μετακινείτε τα αρχεία δεδομένων του Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) που χρησιμοποιείτε τακτικά από το OneDrive.
