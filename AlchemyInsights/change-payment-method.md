# <a name="change-payment-method"></a>Αλλαγή μεθόδου πληρωμής

Μπορείτε να επιβεβαιώσετε τη μέθοδο πληρωμής που έχει ρυθμιστεί στο Κέντρο διαχείρισης Microsoft 365. Επιλέξτε την εγγραφή που θέλετε να επαληθεύσετε. Η μέθοδος πληρωμής που έχει ρυθμιστεί για την εγγραφή που εμφανίζεται στην περιοχή **χρέωσης**. 
  
1. Στο Κέντρο διαχείρισης, μεταβείτε στην **χρέωσης** \> σελίδα [υπηρεσιών & προϊόντα](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Δίπλα από τη μέθοδο πληρωμής για την εγγραφή που θέλετε να αλλάξετε τη μέθοδο πληρωμής για, επιλέξτε " **Επεξεργασία**".

3. Ακολουθήστε τα βήματα στην οθόνη για να ολοκληρώσετε την αλλαγή.

Για οδηγίες βήμα προς βήμα, δείτε: [Αλλαγή τη μέθοδο πληρωμής σας για Office 365 για επιχειρήσεις](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/change-payment-method)
