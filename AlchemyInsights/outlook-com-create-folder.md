# <a name="create-a-folder"></a>Δημιουργήστε ένα φάκελο

1. Στο κάτω μέρος του παραθύρου φακέλων, επιλέξτε **νέο φάκελο**.
2. Στο πλαίσιο της νέας φακέλου, πληκτρολογήστε ένα όνομα για το φάκελο και, στη συνέχεια, πιέστε το πλήκτρο Enter.

Για να μάθετε περισσότερα, ανατρέξτε στην ενότητα [Δημιουργία ενός φακέλου στο Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
