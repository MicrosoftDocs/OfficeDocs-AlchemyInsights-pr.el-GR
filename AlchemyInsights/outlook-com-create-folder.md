# <a name="create-a-folder"></a>Δημιουργήστε ένα φάκελο

1. Στο κάτω μέρος του παραθύρου φακέλων, επιλέξτε **νέο φάκελο**. Ίσως χρειαστεί να μετακινηθείτε προς τα κάτω στο παράθυρο φακέλων για να δείτε το κουμπί " **Νέος φάκελος** ".
2. Στο πλαίσιο κειμένου, πληκτρολογήστε ένα όνομα για το φάκελο και, στη συνέχεια, πιέστε το πλήκτρο Enter.

Για να μάθετε περισσότερα, ανατρέξτε στην ενότητα [Δημιουργία ενός φακέλου στο Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
