# <a name="create-a-folder-in-outlookcom"></a>Δημιουργία φακέλου στο Outlook.com

1. Στο κάτω μέρος του παραθύρου φακέλων, επιλέξτε **Νέος φάκελος**. Ίσως χρειαστεί να κάνετε κύλιση προς τα κάτω στο παράθυρο φακέλων για να δείτε το κουμπί **"Νέος φάκελος".**
2. Στο πλαίσιο κειμένου, πληκτρολογήστε ένα όνομα για το φάκελο και, στη συνέχεια, πατήστε το πλήκτρο Enter.

Για να μάθετε περισσότερα, [ανατρέξτε](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)στο θέμα Δημιουργία φακέλου στο Outlook.com .
