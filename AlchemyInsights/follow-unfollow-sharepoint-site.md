# <a name="follow-or-un-follow-a-sharepoint-site"></a>Παρακολούθηση ή κατάργηση της παρακολούθησης μιας τοποθεσίας του SharePoint

Υπάρχουν πολλοί τρόποι για να ακολουθήσετε μια τοποθεσία που σας ενδιαφέρει στο SharePoint. Αυτό θα σας επιτρέψει να βρείτε την τοποθεσία εύκολα στη λίστα σας και να σας επιτρέψει να δείτε τη δραστηριότητα από την τοποθεσία. 

Παρακολουθήστε το βίντεο για να [ακολουθήσετε μια τοποθεσία του SharePoint](https://support.office.com/article/Video-Follow-a-SharePoint-site-33DB6FA5-9528-45D7-BCC7-F9C1FAAACAE0). 

Εάν δεν μπορείτε να ακολουθήσετε ή να μην ακολουθήσετε μια τοποθεσία και αισθανθείτε ότι τα αποτελέσματα δεν είναι αυτά που αναμένονται, δείτε εάν το ζήτημα σχετίζεται με έναν μεμονωμένο χρήστη ή με όλους τους χρήστες του οργανισμού σας. Εάν το ζήτημα παρουσιαστεί από όλους τους χρήστες, ενδέχεται να υπάρχει ένα ζήτημα προσωρινής υπηρεσίας. 

Ελέγξτε τον [πίνακα εργαλείων εύρυθμης λειτουργίας υπηρεσίας](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) , για να δείτε αν επηρεάζεται ο οργανισμός σας.
