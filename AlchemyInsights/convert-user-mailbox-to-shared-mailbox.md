# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Μετατροπή ενός γραμματοκιβωτίου χρήστη σε ένα γραμματοκιβώτιο αίθουσα ή εξοπλισμός

Μπορείτε επίσης να μετατρέψετε ένα γραμματοκιβώτιο από ένα γραμματοκιβώτιο τύπο σε έναν άλλο, αλλά θα πρέπει να εκτελέσετε αυτές τις εργασίες χρησιμοποιώντας το Exchange Online PowerShell. Αναλάβετε τον έλεγχο [μετατρέψετε ένα γραμματοκιβώτιο](https://go.microsoft.com/fwlink/p/?LinkId=832875) για περισσότερες λεπτομέρειες.
