# <a name="delete-a-sharepoint-page"></a>Διαγράψετε μια σελίδα του SharePoint

1. Μεταβείτε στην τοποθεσία όπου είναι η σελίδα.
    
2. Στην αριστερή πλευρά, κάντε κλικ στην επιλογή **Περιεχόμενα τοποθεσίας**. 
    
3. Στην αριστερή πλευρά της σελίδας περιεχομένων τοποθεσίας, κάντε κλικ στο κουμπί " **σελίδες**". 
    
4. Επιλέξτε τη σελίδα και, στη συνέχεια, κάντε κλικ στο κουμπί **αποσιωπητικά (...)** στα δεξιά της σελίδας. 
    
5. Κάντε κλικ στο κουμπί **Διαγραφή**.
