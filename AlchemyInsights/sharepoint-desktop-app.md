# <a name="desktop-app-for-sharepoint"></a>Εφαρμογή επιφάνειας εργασίας για το SharePoint

Ο οργανισμός σας εισάγεται στο πρόγραμμα ιδιωτικής προεπισκόπησης για την εφαρμογή επιφάνειας εργασίας για το SharePoint.

Η Microsoft δεν υποστηρίζει αυτήν τη στιγμή την εφαρμογή επιφάνειας εργασίας για το SharePoint (ιδιωτική προεπισκόπηση). Χρησιμοποιήστε τους συνεργάτες της Microsoft ιδιωτική προεπισκόπηση για να λάβετε βοήθεια με την εφαρμογή επιφάνειας εργασίας για το SharePoint.
