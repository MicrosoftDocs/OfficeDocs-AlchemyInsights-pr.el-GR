# <a name="desktop-app-for-sharepoint"></a>Εφαρμογή υπολογιστή για το SharePoint

Ο οργανισμός σας έχει εισαχθεί στο ιδιωτικό πρόγραμμα προεπισκόπησης για την εφαρμογή υπολογιστή για το SharePoint.

Η Microsoft δεν υποστηρίζει αυτήν τη στιγμή την εφαρμογή υπολογιστή για το SharePoint (ιδιωτική προεπισκόπηση). Χρησιμοποιήστε τους στρατευμένους συνεργάτες της Microsoft Private Preview για να λάβετε βοήθεια με την εφαρμογή υπολογιστή για το SharePoint.
