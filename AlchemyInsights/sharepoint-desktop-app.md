# <a name="desktop-app-for-sharepoint"></a>Στην επιφάνεια εργασίας εφαρμογών για το SharePoint

Ο οργανισμός σας εισάγεται στο πρόγραμμα προεπισκόπησης ιδιωτικό για το App επιφάνεια εργασίας για το SharePoint.

Η Microsoft δεν υποστηρίζει αυτήν τη στιγμή το App επιφάνεια εργασίας για το SharePoint (ιδιωτικό προεπισκόπηση). Χρησιμοποιήστε τους συνεργάτες σας καταχωρημένες προεπισκόπηση ιδιωτικού Microsoft για να λάβετε βοήθεια σχετικά με το App επιφάνεια εργασίας για το SharePoint.
