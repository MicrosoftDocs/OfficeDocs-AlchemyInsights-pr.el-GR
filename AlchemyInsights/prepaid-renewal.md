# <a name="prepaid-renewal"></a>Προπληρωμένο ανανέωσης

1. Στο Κέντρο διαχείρισης, μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**.
    
2. Επιλέξτε το προπληρωμένο συνδρομή που θα θέλατε να επεκτείνετε και στη συνέχεια επιλέξτε **επέκταση**.
    
3. Ενεργοποιήστε την επιλογή για τον **αριθμό-κλειδί προϊόντος**. (Εάν αυτό ήταν ένα αριθμό-κλειδί προϊόντος που αγοράσατε σε ένα σημείο λιανικής πώλησης, μπορεί να έχετε δυνατότητα επέκτασης με άλλους τρόπους πληρωμής.)
    
4. Εισαγάγετε τον αριθμό-κλειδί προϊόντος.
    
5. Βεβαιωθείτε ότι οι πληροφορίες είναι σωστές και, στη συνέχεια, επιλέξτε **Επόμενο**.
    
6. Επιλέξτε το κουμπί επιλογής για να επεκτείνετε την ημερομηνία λήξης από τις υπάρχουσες άδειες χρήσης και, στη συνέχεια, επιλέξτε **Επόμενο**.
