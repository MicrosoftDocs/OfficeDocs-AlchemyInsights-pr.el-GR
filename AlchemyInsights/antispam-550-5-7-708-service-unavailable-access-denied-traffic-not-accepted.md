# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>Δεν επιτρέπεται η πρόσβαση 550 5.7.708, κυκλοφορία δεν γίνονται δεκτές από IP

Αυτό το σφάλμα παρουσιάζεται κατά την αποστολή μηνύματος ηλεκτρονικού ταχυδρομείου από γνωστή, χαμηλή φήμη διευθύνσεις IP που χρησιμοποιούνται συνήθως από νέους πελάτες.

Αν πρέπει να στείλετε μήνυμα ηλεκτρονικού ταχυδρομείου από αυτές τις διευθύνσεις IP, πριν να αγοράσετε άδειες χρήσης, επικοινωνήστε με την υποστήριξη για να ζητήσετε μια εξαίρεση, μέχρι να είστε σε θέση να αγοράσετε άδειες χρήσης.
