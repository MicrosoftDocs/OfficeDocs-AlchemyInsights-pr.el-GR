# <a name="surface-xbox-and-microsoft-devices-safety-concerns"></a>Ζητήματα ασφάλειας των συσκευών Surface, Xbox και Microsoft

Εάν θέλετε να συζητήσετε για ένα πρόβλημα ασφάλειας προϊόντος (προσωπικός τραυματισμός, υλικές ζημιές, ορατός αιθάς), επιλέξτε "Επικοινωνία" στην κάτω αριστερή γωνία για να συνδεθείτε απευθείας με έναν συνήγορο υποστήριξης με βοήθεια.
