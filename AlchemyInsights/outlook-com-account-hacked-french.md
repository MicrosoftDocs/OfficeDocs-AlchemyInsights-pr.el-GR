# <a name="mon-compte-outlookcom-a-t-pirat"></a>ΔΕΥ υπόψη Outlook.com ένα piraté έχει

SI σας κάρτα ένα έχει piraté, cela signifie που quelqu'un s'est procuré σας mot de passe και ότι μπορεί να χρησιμοποιήσετε σας κάρτα για accéder indésirables ή envoyer des courriers στο vos πληροφορίες personnelles.

Σκυροδέτηση récupérer σας κάρτα, [suivez τις οδηγίες](https://support.office.com/fr-fr/article/mon-compte-outlook-com-a-été-piraté-35993ac5-ac2f-494e-aacb-5232dda453d8?ui=fr-FR&rs=fr-FR&ad=FR).
