# <a name="asms-test-solution"></a>Λύση δοκιμής ASMS

Οι πελάτες Azure με Σύμβαση Azure Enterprise (EA), Σύμβαση πελάτη της Microsoft (MCA) ή Σύμβαση συνεργάτη της Microsoft (MPA) μπορούν να προβάλουν και να κάνουν λήψη των τιμών τους στην πύλη Azure.
