# <a name="already-paid"></a>Ήδη καταβληθεί;

Εάν έχετε ήδη στείλει πληρωμής, μπορείτε να πείτε μας γνωρίζετε:
  
1. Μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**.

2. Επιλέξτε μια συνδρομή και στην ενότητα **το λογαριασμό μου** , επιλέξτε **Προβολή τιμολόγησης**.

3. Στη σελίδα **λεπτομέρειες τιμολόγησης** , επιλέξτε **ήδη καταβληθεί;** Θα σας ζητηθεί να εισαγάγετε τις πληροφορίες που σχετίζονται με την πληρωμή που σας έχει σταλεί.

4. Όταν καταχωρηθεί όλες τις απαιτούμενες πληροφορίες, επιλέξτε **Αποστολή**.
