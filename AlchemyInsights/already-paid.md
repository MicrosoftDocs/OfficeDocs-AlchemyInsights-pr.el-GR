# <a name="already-paid"></a>Έχετε ήδη πληρώσει;

Εάν έχετε ήδη στείλει την πληρωμή, μπορείτε να μας ενημερώσετε:
  
1. Στο κέντρο διαχείρισης, μεταβείτε στη **Billing** \> σελίδα [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) χρεώσεων.

2. Επιλέξτε μια συνδρομή και, στην ενότητα ο **λογαριασμός μου** , επιλέξτε **View Bill**.

3. Στη σελίδα **λεπτομέρειες χρέωσης** , επιλέξτε **ήδη πληρωμένο;** Θα σας ζητηθεί να καταχωρήσετε πληροφορίες που σχετίζονται με την πληρωμή που στάλθηκε.

4. Όταν έχουν εισαχθεί όλες οι απαιτούμενες πληροφορίες, επιλέξτε **Αποστολή**.
