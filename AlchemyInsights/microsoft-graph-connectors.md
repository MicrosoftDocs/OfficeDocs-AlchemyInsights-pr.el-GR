# <a name="microsoft-graph-connectors"></a>Συνδέσεις του Microsoft Graph

Με τις γραμμές σύνδεσης του Microsoft Graph, ο οργανισμός σας μπορεί να αναπροσαρμόζει τα δεδομένα τρίτων κατασκευαστών στα αποτελέσματα αναζήτησης της Microsoft.

- [Γραμμές σύνδεσης από τη Microsoft](https://docs.microsoft.com/microsoftsearch/connectors-gallery#Microsoft)
- [Γραμμές σύνδεσης από τους συνεργάτες μας](https://docs.microsoft.com/microsoftsearch/connectors-gallery#Partners)

Για περισσότερες πληροφορίες, ανατρέξτε  [στο θέμα Επισκόπηση των γραμμών σύνδεσης του Microsoft Graph](https://docs.microsoft.com/microsoftsearch/connectors-overview).
