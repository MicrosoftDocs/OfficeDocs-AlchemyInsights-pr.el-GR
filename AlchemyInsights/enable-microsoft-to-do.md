# <a name="how-to-enable-microsoft-to-do"></a>Πώς μπορείτε να ενεργοποιήσετε το Microsoft To-Do

Για να χρησιμοποιήσετε το Microsoft To-Do, ένας διαχειριστής πρέπει να ενεργοποιήσει την εφαρμογή στο λογαριασμό κάθε χρήστη. Μεταβείτε στη [ &amp; σελίδα πρόσθετα υπηρεσιών](https://portal.office.com/adminportal/home#/Settings/ServicesAndAddIns) της σελίδας ρυθμίσεις διαχείρισης του Microsoft 365 για να ενεργοποιήσετε τις εκκρεμείς εργασίες για κάθε χρήστη.
  
Το Microsoft To-Do χρησιμοποιεί δυνατότητες στο Exchange Online για τον συγχρονισμό και την αποθήκευση των στοιχείων εκκρεμών εργασιών των χρηστών. Για να χρησιμοποιήσετε το Microsoft To-Do, ένας χρήστης χρειάζεται ένα έγκυρο γραμματοκιβώτιο του Exchange Online.
  
Για περισσότερες πληροφορίες, ανατρέξτε [στο θέμα Διαχείριση προεπισκόπησης για το Microsoft To-Do](https://support.office.com/article/490c1a8c-2333-4952-8125-841afadb9620.aspx).
  
 **Σημείωση**: Εάν εμφανιστεί ένα σφάλμα 501 όταν προσπαθείτε να συνδεθείτε [https://todo.microsoft.com](https://todo.microsoft.com) , δοκιμάστε το [https://to-do.microsoft.com](https://to-do.microsoft.com) .
