# <a name="deploy-office-apps"></a>Ανάπτυξη εφαρμογών του Office

Για την ανάπτυξη εφαρμογών του Office σε πολλούς τελικούς χρήστες μέσω του εργαλείου ανάπτυξης του Office, Διαχείριση ρύθμισης παραμέτρων απόληξη της Microsoft, ως μέρος ενός ειδώλου δίσκου λειτουργικού συστήματος ή σε ένα σενάριο κοινόχρηστο υπολογιστή χρησιμοποιώντας το Windows Server με υπηρεσίες απομακρυσμένης επιφάνειας εργασίας (RDS) ή υπηρεσίες Terminal Services, εξετάστε το ενδεχόμενο να χρησιμοποιήσετε τον οδηγό βήμα προς βήμα στον **Σύμβουλο ανάπτυξης του Office 365 ProPlus**.
  
Για να ξεκινήσετε, βεβαιωθείτε ότι είστε συνδεδεμένοι ως διαχειριστής μισθωτών ή καθολικός διαχειριστής και, στη συνέχεια, κάντε κλικ εδώ [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance):.
