# <a name="microsoft-qualified-academic-institution-and-remote-teaching-and-learning"></a>Εγκεκριμένο Ακαδημαϊκό Ίδρυμα της Microsoft και απομακρυσμένη διδασκαλία και μάθηση

Για να ξεκινήσετε με τη Microsoft για εκπαιδευτικά ιδρύματα, επισκεφθείτε αυτή την [τοποθεσία ](https://www.microsoft.com/microsoft-365/academic/compare-office-365-education-plans) και εγγραφείτε.

Θα έχετε τη δοκιμαστική έκδοση του Office 365 A1 για διδακτικό προσωπικό και το Office 365 Α1 για μαθητές, ενώ περιμένετε να επανεξεταστούν τα κριτήρια καταλληλότητας του οργανισμού σας.  Φροντίστε να προσθέσετε έναν τομέα, καθώς αυτό μπορεί να σας βοηθήσει να επισπεύσετε τη διαδικασία έγκρισης.

Θα λάβετε ένα μήνυμα ηλεκτρονικού ταχυδρομείου που θα σας ενημερώσει εάν έχει εγκριθεί η καταλληλότητα του οργανισμού σας.  

Κατά την έγκρισή σας, μπορείτε να μεταβείτε στην καρτέλα [Αγορά υπηρεσιών](https://admin.microsoft.com/Adminportal/Home#/catalog) του κέντρου διαχείρισης της Microsoft για να αγοράσετε συνδρομές.

**Απομακρυσμένη διδασκαλία και μάθηση**

Στη συνέχεια, ανατρέξτε στο θέμα [Απομακρυσμένη διδασκαλία και μάθηση στο Office 365 για εκπαιδευτικά ιδρύματα](https://support.office.com/article/remote-teaching-and-learning-in-office-365-education-f651ccae-7b65-478b-8366-51bb884025c4) για να μάθετε τις πιο ενημερωμένες οδηγίες σχετικά με τη ρύθμιση του σχολείου σας, τον προγραμματισμό μαθημάτων, την εικονική σύσκεψη και την κοινή χρήση περιεχομένου με τους μαθητές.

Υπάρχει πρόσθετη καθοδήγηση ΙΤ στα θέματα [Γρήγορα αποτελέσματα με το Microsoft Teams για απομακρυσμένη εκμάθηση](https://docs.microsoft.com/MicrosoftTeams/remote-learning-edu) και [Μετάβαση στην απομακρυσμένη εκμάθηση](https://www.microsoft.com/education/remote-learning).
