# <a name="microsoft-viva-connections-availability"></a>Διαθεσιμότητα συνδέσεων Microsoft Viva

Το Microsoft Viva Connections θα είναι διαθέσιμο για υπολογιστή το πρώτο εξάμηνο του 2021 και για κινητές συσκευές το καλοκαίρι του 2021. Πρόσθετες δυνατότητες θα είναι διαθέσιμες στη διάρκεια του 2022.

Για περισσότερες πληροφορίες, δείτε πώς το [Microsoft Viva Connections βοηθά όλους τους χρήστες να παραμείνουν απασχολημένοι και ενημερωμένοι.](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-connections-helps-everyone-to-stay-engaged-and/ba-p/2107009)
