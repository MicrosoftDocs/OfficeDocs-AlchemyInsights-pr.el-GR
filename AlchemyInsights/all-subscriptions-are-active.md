# <a name="all-subscriptions-are-active"></a>Όλες οι εγγραφές είναι ενεργή

Οι εγγραφές σας είναι ενεργή. Αυτό μπορεί να επαληθευτεί από το **χρέωσης** \> σελίδα [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) . Συνδρομές σας ταξινομούνται κατά κατάσταση, με τις ενεργές εγγραφές στην κορυφή. Πρόσθετες πληροφορίες, όπως άδεια πλήθος και τη μέθοδο πληρωμής, μπορείτε να δείτε, επιλέγοντας τη συνδρομή.
  
Εάν αντιμετωπίζετε προβλήματα με την υπηρεσία σας, χρησιμοποιήστε την επιλογή "ας καλέσουν" για να συνομιλήσετε με έναν μηχανικό υποστήριξης.
