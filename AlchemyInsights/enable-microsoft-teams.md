# <a name="enable-and-use-microsoft-teams"></a>Ενεργοποίηση και χρήση ομάδες της Microsoft


Εάν δεν έχετε πρόσβαση στις ομάδες της Microsoft σε όλα, αναλάβετε τον έλεγχο [Πώς μπορώ να αποκτήσω πρόσβαση σε ομάδες της Microsoft](https://support.office.com/article/How-do-I-get-access-to-Microsoft-Teams-fc7f1634-abd3-4f26-a597-9df16e4ca65b.aspx)

Εάν έχετε ήδη ομάδες της Microsoft και πρέπει να γνωρίζουν πώς να ρυθμίσετε και να διαχειριστείτε, ανατρέξτε στην ενότητα [ρυθμίσεις διαχειριστή για ομάδες της Microsoft](https://support.office.com/article/Administrator-settings-for-Microsoft-Teams-3966a3f5-7e0f-4ea9-a402-41888f455ba2.aspx)
