# <a name="enable-and-use-microsoft-teams"></a>Ενεργοποίηση και χρήση του Microsoft teams

Εάν δεν έχετε καθόλου πρόσβαση στο Microsoft teams, ανατρέξτε στο θέμα [Πώς μπορώ να αποκτήσω πρόσβαση στο Microsoft teams](https://support.office.com/article/How-do-I-get-access-to-Microsoft-Teams-fc7f1634-abd3-4f26-a597-9df16e4ca65b.aspx).

Εάν έχετε ήδη το Microsoft teams και θέλετε να μάθετε πώς μπορείτε να τις ρυθμίσετε και να τη διαχειριστείτε, ανατρέξτε στο θέμα [Διαχείριση ρυθμίσεων του Microsoft teams για τον οργανισμό σας](https://docs.microsoft.com/MicrosoftTeams/enable-features-office-365).
