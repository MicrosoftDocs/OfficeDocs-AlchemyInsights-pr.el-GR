# <a name="enable-and-use-microsoft-teams"></a>Ενεργοποίηση και χρήση ομάδων της Microsoft

Εάν δεν έχετε πρόσβαση στις ομάδες της Microsoft, δείτε [Πώς μπορώ να έχω πρόσβαση στις ομάδες της Microsoft](https://support.office.com/article/How-do-I-get-access-to-Microsoft-Teams-fc7f1634-abd3-4f26-a597-9df16e4ca65b.aspx).

Εάν έχετε ήδη ομάδες της Microsoft και πρέπει να γνωρίζετε τον τρόπο ρύθμισης παραμέτρων και διαχείρισης, ανατρέξτε [στο θέμα Διαχείριση ρυθμίσεων ομάδων της Microsoft για τον οργανισμό σας](https://docs.microsoft.com/MicrosoftTeams/enable-features-office-365).
