# <a name="subscriptions-and-unsubscribing"></a>Συνδρομές και κατάργηση συνδρομής

Εάν θέλετε να διακόψετε τη λήψη ενημερωτικών δελτίων ή προωθητικών ενεργειών, υπάρχουν δύο τρόποι με τους οποίους μπορείτε να καταργήσετε την εγγραφή σας στο Outlook.com:

1. Στα Εισερχόμενά σας, επιλέξτε ένα ενημερωτικό δελτίο ή ένα διαφημιστικό μήνυμα ηλεκτρονικού ταχυδρομείου. Στο επάνω μέρος του παραθύρου ανάγνωσης, επιλέξτε Κατάργηση **εγγραφής**.
2. Ανοίξτε τις [Ρυθμίσεις συνδρομών του Outlook.com](https://outlook.live.com/mail/options/mail/brandsSubscriptions) (**Ρυθμίσεις**  >  **Προβολή όλων**των  >  συνδρομών**ηλεκτρονικού ταχυδρομείου**ρυθμίσεων του Outlook  >  **Subscriptions**) και επιλέξτε Κατάργηση **εγγραφής**.

Σημειώσεις

- Αυτή τη στιγμή δεν μπορούν όλοι οι πελάτες να επωφεληθούν από αυτήν τη δυνατότητα.
- Αυτή η δυνατότητα θα εμφανίζεται μόνο εάν έχετε λάβει ένα μήνυμα ηλεκτρονικού ταχυδρομείου από την εταιρεία που στέλνει το εγγεγραμμένο μήνυμα ηλεκτρονικού ταχυδρομείου.
