# <a name="subscription-information"></a>Πληροφορίες συνδρομής

Για πληροφορίες σχετικά με τις συνδρομές στην κατοχή, μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**.
  
Βασικές πληροφορίες σχετικά με κάθε εγγραφή:
  
- Κατάσταση για κάθε
    
- Άδειες χρήσης που αγοράσατε
    
- Άδειες που έχουν ανατεθεί
    
- Περιοδική κατάσταση χρέωσης
    
- Υπόλοιπο
    
- Μέθοδος πληρωμής
    
- Επόμενη ημερομηνία τιμολόγησης ή λήξης
