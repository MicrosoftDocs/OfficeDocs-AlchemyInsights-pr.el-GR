# <a name="subscription-information"></a>Πληροφορίες συνδρομής

Για πληροφορίες σχετικά με τις συνδρομές που σας ανήκουν, μεταβείτε στη σελίδα **Χρέωση** \> **[των προϊόντων σας.](https://go.microsoft.com/fwlink/p/?linkid=842054)**
  
Βασικές πληροφορίες για κάθε συνδρομή:
  
- Κατάσταση κάθε

- Άδειες χρήσης που αγοράστηκαν

- Εκχωρημένες άδειες χρήσης

- Κατάσταση περιοδικής χρέωσης

- Ισορροπία

- Μέθοδος πληρωμής

- Επόμενη ημερομηνία χρέωσης ή λήξης
