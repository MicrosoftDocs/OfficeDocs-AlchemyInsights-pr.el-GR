# <a name="access-denied-to-sharepoint-site"></a>Δεν επιτρέπεται η πρόσβαση στην τοποθεσία του SharePoint

Εάν λαμβάνετε μήνυμα πως δεν επιτρέπεται η πρόσβαση όταν προσπαθείτε να μεταβείτε σε μια τοποθεσία του Sharepoint Online, ανατρέξτε στο θέμα [Δεν επιτρέπεται η πρόσβαση στην τοποθεσία του SharePoint](https://docs.microsoft.com/sharepoint/troubleshoot/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-sharepoint-site).
