# <a name="cant-select-contacts-when-composing-email"></a>Δεν είναι δυνατό να επιλέξετε επαφές, κατά τη σύνθεση μηνύματος ηλεκτρονικού ταχυδρομείου

Κατά τη σύνθεση ενός νέου μηνύματος ηλεκτρονικού ταχυδρομείου, ορισμένοι πελάτες δεν βλέπετε τους λίστα επαφών, κάνοντας κλικ στα κουμπιά **προς** ή **Κοιν** . Για να επιλέξετε μια επαφή, αρχίστε να πληκτρολογείτε στο το **να** ή τα πεδία **"Κοιν."** και τις σχετικές επαφές θα εμφανιστούν.
