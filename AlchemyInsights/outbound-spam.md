# <a name="outbound-spam"></a>Εξερχόμενο ανεπιθύμητη αλληλογραφία

Εάν βλέπετε ψευδή θετικά στοιχεία για την εξαγωγή ανεπιθύμητης αλληλογραφίας (τα καλά εξερχόμενες μηνύματα επισημαίνονται ως ανεπιθύμητη αλληλογραφία), ανοίξτε ένα δελτίο υποστήριξης με παραδείγματα μηνυμάτων.
