# <a name="yammer---possible-service-issue"></a>Yammer-πιθανό ζήτημα υπηρεσίας

Ενδέχεται να υπάρχει ένα προσωρινό ζήτημα όπου οι χρήστες ενδέχεται να αντιμετωπίσετε σφάλματα κατά την πρόσβαση στο Yammer. Ελέγξτε τον [πίνακα εργαλείων εύρυθμης λειτουργίας υπηρεσίας](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) για να δείτε αν αυτό επηρεάζει το δίκτυό σας.
