# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Σε μια συσκευή Windows 10 στον Microsoft Defender Advanced Threat Protection (MDATP)

Το MDATP υποστηρίζει τα ακόλουθα εργαλεία και μεθόδους ανάπτυξης για τα Windows 10:

- Πολιτική ομάδας
- Microsoft Endpoint Configuration Manager
- Διαχείριση κινητών συσκευών, συμπεριλαμβανομένου του Microsoft Intune
- Τοπικές δέσμες ενεργειών

Για να μάθετε περισσότερα, ανατρέξτε στο θέμα Εργαλεία και μέθοδοι για [συσκευές Windows 10.](https://go.microsoft.com/fwlink/?linkid=2143460)
