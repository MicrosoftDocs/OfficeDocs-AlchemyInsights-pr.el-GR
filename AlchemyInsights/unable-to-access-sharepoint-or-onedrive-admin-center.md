# <a name="unable-to-access-sharepoint-or-onedrive-admin-center"></a>Δεν είναι δυνατή η πρόσβαση στο κέντρο διαχείρισης του SharePoint ή του OneDrive

- Εάν η τοποθεσία σας στο SharePoint ή το Κέντρο διαχείρισης του OneDrive δεν είναι προσβάσιμη ή δεν είναι διαθέσιμη, ενδέχεται να υπάρχει ένα προσωρινό ζήτημα υπηρεσίας όπου οι χρήστες αντιμετωπίζουν διαλείπουσες καθυστερήσεις ή σφάλματα περιήγησης κατά την πρόσβαση σε τοποθεσίες του SharePoint ή περιεχόμενο του OneDrive. Ελέγξτε τον [πίνακα εργαλείων εύρυθμης λειτουργίας υπηρεσίας](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) για να δείτε αν ο οργανισμός σας επηρεάζεται.

- Στους διαχειριστές καθολικού και SharePoint πρέπει να εκχωρηθεί μια άδεια χρήσης του SharePoint. Οι λογαριασμοί που μόλις δημιουργήθηκαν μόλις εκχωρούνται με μια άδεια χρήσης του SharePoint ή ρόλο διαχειριστή ενδέχεται να αντιμετωπίσετε ζητήματα πρόσβασης στο SharePoint, όπως "δεν επιτρέπεται η πρόσβαση" ή "ο χρήστης δεν βρέθηκε". Δώστε τουλάχιστον 24 ώρες για να ολοκληρωθεί ο συγχρονισμός σε όλα τα συστήματά μας. Καταλαβαίνουμε ότι 24 ώρες μπορεί να φαίνεται σαν ένα μεγάλο χρονικό διάστημα. Σε πολλές περιπτώσεις, εργαζόμαστε ήδη για μια λύση.
