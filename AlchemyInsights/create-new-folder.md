# <a name="create-a-new-folder-in-outlookcom"></a>Δημιουργία νέου φακέλου στο Outlook.com

1. Στην περιοχή **Φάκελοι** στο παράθυρο περιήγησης, επιλέξτε **νέο φάκελο**.

2. Πληκτρολογήστε ένα όνομα για το φάκελο και πατήστε ENTER.

Για να μάθετε περισσότερα, δείτε [τη δημιουργία ενός φακέλου στο Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
