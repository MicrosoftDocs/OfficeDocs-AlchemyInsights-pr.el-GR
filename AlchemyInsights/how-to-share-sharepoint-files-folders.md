# <a name="how-to-share-sharepoint-files-or-folders"></a>Τρόπος κοινής χρήσης αρχείων ή φακέλων του SharePoint

Τα αρχεία που αποθηκεύετε σε μια τοποθεσία του SharePoint είναι συνήθως διαθέσιμα σε όλους με δικαιώματα στην τοποθεσία, αλλά μπορεί να θέλετε να κάνετε κοινή χρήση συγκεκριμένων αρχείων ή φακέλων με άτομα που δεν έχουν διαφορετικά πρόσβαση στην τοποθεσία. Όταν μοιράζεστε αρχεία και φακέλους, μπορείτε να αποφασίσετε αν θα αφήσετε τους χρήστες να επεξεργαστούν ή απλά να τα προβάλουν.

[Κοινή χρήση αρχείων ή φακέλων του SharePoint](https://support.office.com/article/share-sharepoint-files-or-folders-1fe37332-0f9a-4719-970e-d2578da4941c?ui=en-US&rs=en-US&ad=US)
