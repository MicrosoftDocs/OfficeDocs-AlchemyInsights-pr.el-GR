# <a name="outlook-search-not-indexing"></a>Η αναζήτηση στο Outlook δεν έχει ευρετήριο

Γνωρίζουμε ότι τα προβλήματα αναζήτησης του Outlook σχετίζονται με τις πρόσφατες ενημερώσεις ασφαλείας για τα Windows.
  
Κατά την αναζήτηση στο Outlook, ενδέχεται να λάβετε ένα από αυτά τα σφάλματα: **Παρουσιάστηκε κάποιο πρόβλημα και δεν ήταν δυνατό να ολοκληρωθεί η αναζήτησή σας**ή τα **αποτελέσματα αναζήτησης ενδέχεται να μην είναι ολοκληρωμένα, επειδή τα στοιχεία εξακολουθούν να περιλαμβάνονται στο ευρετήριο**. Η επιδιόρθωση εκδόθηκε ως προαιρετική ενημέρωση των Windows. Εγκαταστήστε αυτό που αντιστοιχεί στην έκδοση των Windows στην οποία βλέπετε αυτό το πρόβλημα: 
  
> [Windows 7-27 Ιουνίου 2017 ενημέρωση](https://support.microsoft.com/kb/4022168.aspx)
    
> [Windows 8,1-27 Ιουνίου 2017 ενημέρωση](https://support.microsoft.com/kb/4022720.aspx)
    
> [Ενημέρωση των Windows 10 έκδοση 1703-27 Ιουνίου 2017](https://support.microsoft.com/kb/4022716.aspx)
    
Για τις πιο πρόσφατες πληροφορίες, ανατρέξτε στο άρθρο: τα [γνωστά θέματα του Outlook στο θέμα ενημερωμένες εκδόσεις ασφαλείας του ιουνίου 2017](https://support.office.com/article/Outlook-known-issues-in-the-June-2017-security-updates-3F6DBFFD-8505-492D-B19F-B3B89369ED9B.aspx) #5.
