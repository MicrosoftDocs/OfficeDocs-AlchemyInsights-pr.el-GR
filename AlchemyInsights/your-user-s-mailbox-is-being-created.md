# <a name="user-mailbox-creation-is-taking-a-long-time"></a>Η δημιουργία γραμματοκιβωτίου χρήστη διαρκεί πολύ

Μπορεί να διαρκέσει έως και 24 ώρες μέχρι ένα νέο γραμματοκιβώτιο να είναι έτοιμο για χρήση.
  
Εάν το γραμματοκιβώτιο του χρήστη "Προετοιμάζεται" για περισσότερο από μία ώρα, δοκιμάστε να καταργήσετε την άδεια χρήσης του Exchange από το χρήστη και να την προσθέσετε ξανά ή επικοινωνήστε [με την υποστήριξη.](/microsoft-365/admin/contact-support-for-business-products?tabs=online)
