# <a name="how-to-right-click-on-a-tablet"></a>Πώς μπορείτε να κάνετε δεξιό κλικ σε ένα Tablet PC

Κάντε δεξιό κλικ σε μια επιφάνεια ή Windows tablet, κτυπήστε και κρατήστε την επιλογή σας για μερικά δευτερόλεπτα. Όταν αφήσετε, θα εμφανιστεί το μενού δεξιού κλικ.
