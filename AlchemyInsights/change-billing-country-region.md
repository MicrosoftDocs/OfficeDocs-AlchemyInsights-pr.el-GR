# <a name="change-billing-country-or-region"></a>Αλλαγή χώρας ή περιοχής χρέωσης

Δεν είναι δυνατή η αλλαγή της χώρας/περιοχής από τους **λογαριασμούς**  >  **χρέωσης** στο Microsoft 365 διαχείρισης. Θα πρέπει να εγγραφείτε για έναν νέο λογαριασμό και να επιλέξετε την επιθυμητή χώρα/περιοχή.
