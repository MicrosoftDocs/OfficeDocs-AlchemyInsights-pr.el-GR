# <a name="create-a-shared-mailbox"></a>Δημιουργία κοινόχρηστου γραμματοκιβωτίου

Ένα κοινόχρηστο γραμματοκιβώτιο επιτρέπει σε περισσότερα από ένα άτομα να στέλνουν και να λαμβάνουν μηνύματα ηλεκτρονικού ταχυδρομείου για ένα ψευδώνυμο, όπως το sales@contoso.com, χωρίς να εκχωρήσουν άλλη άδεια χρήσης. 

Δείτε το άρθρο [Δημιουργήστε ένα κοινόχρηστο γραμματοκιβώτιο](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) για συμβουλές και περισσότερες πληροφορίες σχετικά με τη χρήση ενός κοινόχρηστου γραμματοκιβωτίου.
