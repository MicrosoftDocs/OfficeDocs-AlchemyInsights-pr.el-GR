# <a name="unable-to-print"></a>Δεν είναι δυνατή η εκτύπωση

Γνωρίζουμε ένα πρόβλημα κατά το οποίο ορισμένοι χρήστες δεν μπορούν να εκτυπώσουν όταν χρησιμοποιούν προγράμματα περιήγησης Safari ή Edge. Ενώ εργαζόμαστε σε μια ενημέρωση κώδικα για αυτό, χρησιμοποιήστε ένα άλλο πρόγραμμα περιήγησης. Ζητούμε συγγνώμη για την αναστάτωση.
