# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Χρήση κοινόχρηστου γραμματοκιβωτίου σε κινητή συσκευή

Για να χρησιμοποιήσετε ένα κοινόχρηστο γραμματοκιβώτιο στην κινητή συσκευή σας, θα πρέπει να [κάνετε λήψη του Outlook Mobile](https://products.office.com/outlook-mobile-for-android-and-ios) και, στη συνέχεια, [Προσθέστε ένα κοινόχρηστο γραμματοκιβώτιο στο Outlook Mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Για να λάβετε βοήθεια: [Αποκτήστε βοήθεια in-app για το Outlook για iOS και Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
