# <a name="restore-a-deleted-user"></a>Επαναφορά διαγραμμένου χρήστη

Επαναφορά χρήστη με τα παρακάτω βήματα:
  
1. Μεταβείτε στους [χρήστες που έχουν \> διαγράψει χρήστες](https://admin.microsoft.com/adminportal/home#/deletedusers).

2. Επιλέξτε το χρήστη και, στη συνέχεια, επιλέξτε **Επαναφορά**.

3. Ακολουθήστε τις οδηγίες για να καθορίσετε τον κωδικό πρόσβασης.

4. Κάντε κλικ στην επιλογή **Αποστολή μηνύματος ηλεκτρονικού ταχυδρομείου και κλείστε**το και είστε έτοιμοι!

Δεν ήταν τόσο εύκολο; Για περισσότερες λεπτομέρειες και βήματα με στιγμιότυπα οθόνης, ανατρέξτε σε αυτό το άρθρο: [Επαναφορά χρήστη](https://docs.microsoft.com/microsoft-365/admin/add-users/restore-user). Εάν αντιλαμβάνεστε ότι πρέπει να επαναφέρετε το γραμματοκιβώτιο του χρήστη, ανατρέξτε στο θέμα: [Διαγραφή ή επαναφορά γραμματοκιβωτίων χρηστών στο Exchange Online](https://docs.microsoft.com/exchange/recipients-in-exchange-online/delete-or-restore-mailboxes).
