# <a name="add-or-remove-a-custom-role"></a>Προσθήκη ή κατάργηση προσαρμοσμένου ρόλου

Για να διαχειριστείτε τους πόρους Azure, ανατρέξτε [στο θέμα Προσθήκη ή κατάργηση εκχωρήσεων ρόλων Azure χρησιμοποιώντας την πύλη Azure](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Για να διαχειριστείτε προσαρμοσμένους ρόλους για το Azure Active Directory (Azure AD), ανατρέξτε [στο θέμα προσαρμοσμένοι Ρόλοι διαχειριστή στο Azure Active Directory (Preview)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
