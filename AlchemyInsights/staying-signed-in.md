# <a name="staying-signed-in"></a>Παραμείνετε συνδεδεμένοι

Εάν δεν θέλετε να παραμείνετε συνδεδεμένοι, μπορείτε να επιλέξετε να **μην εμφανιστεί ξανά** την επόμενη φορά που θα πραγματοποιήσετε είσοδο.
