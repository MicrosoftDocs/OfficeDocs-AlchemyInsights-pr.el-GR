# <a name="increase-onedrive-storage"></a>Αύξηση χώρου αποθήκευσης στο OneDrive

Ως καθολικός διαχειριστής ή διαχειριστής του SharePoint στο Microsoft 365, μπορείτε να καθορίσετε το χώρο αποθήκευσης του OneDrive για έναν συγκεκριμένο χρήστη απευθείας από το κέντρο διαχείρισης της Microsoft.  

Για περισσότερες πληροφορίες, ανατρέξτε [στο θέμα Αλλαγή του χώρου αποθήκευσης στο OneDrive ενός συγκεκριμένου χρήστη](https://docs.microsoft.com/onedrive/change-user-storage).
