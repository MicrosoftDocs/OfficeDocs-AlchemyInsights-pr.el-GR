# <a name="hybrid-mdm"></a>Υβριδικά MDM

Η Microsoft αποσύρθηκε από την υβριδική υπηρεσία παροχής υπηρεσιών MDM από την 1η Σεπτεμβρίου 2019. Οι υπόλοιπες υβριδικές συσκευές MDM δεν θα λαμβάνουν πολιτικές, εφαρμογές ή ενημερώσεις ασφαλείας. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Τι συνέβη με τα υβριδικά MDM;](https://docs.microsoft.com/configmgr/mdm/understand/what-happened-to-hybrid)
