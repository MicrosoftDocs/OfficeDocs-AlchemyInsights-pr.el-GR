# <a name="recover-missing-or-deleted-site-collections"></a>Ανάκτηση συλλογών τοποθεσιών που λείπουν ή διαγράφηκαν

Ανατρέξτε στα παρακάτω άρθρα για πληροφορίες σχετικά με τον τρόπο αποκατάστασης συλλογών τοποθεσιών που λείπουν ή διαγράφηκαν στο SharePoint.

[Επαναφορά διαγραμμένων τοποθεσιών](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Επαναφορά με τη χρήση της λειτουργικής μονάδας SPO PowerShell](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Εάν δεν μπορείτε να βρείτε την τοποθεσία, μπορεί να θέλετε να [κάνετε αναζήτηση στο αρχείο καταγραφής ελέγχου στο &amp; Κέντρο συμμόρφωσης ασφαλείας του Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance).

**Σημείωση:** Οι "σύγχρονες" τοποθεσίες ομάδας δεν είναι ορατές στο **κλασικό** περιβάλλον εργασίας χρήστη διαχείρισης του SharePoint. Μπορείτε να αποκτήσετε πρόσβαση στη λίστα των "σύγχρονων" τοποθεσιών από το [νέο κέντρο διαχείρισης του SharePoint](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
