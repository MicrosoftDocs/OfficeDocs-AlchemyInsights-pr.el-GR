Για να ολοκληρώσετε τον έλεγχο μιας περιόδου λειτουργίας χρησιμοποιώντας τη συσκευή της ως συνθήκη, δημιουργήστε μια πολιτική πρόσβασης υπό όρους και μια πολιτική περιόδου λειτουργίας.
