# <a name="public-folder-access-from-mobile-devices"></a>Πρόσβαση σε δημόσιο φάκελο από κινητές συσκευές

Δεν είναι δυνατή η πρόσβαση σε δημόσιους φακέλους από κινητές συσκευές.
  
Εάν θέλετε να έχετε λειτουργίες παρόμοιες με τους δημόσιους φακέλους και να έχετε πρόσβαση σε αυτές από κινητές συσκευές, ανατρέξτε στο θέμα [Μάθετε σχετικά με τις Ομάδες microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) για μια καλύτερη εναλλακτική λύση.
