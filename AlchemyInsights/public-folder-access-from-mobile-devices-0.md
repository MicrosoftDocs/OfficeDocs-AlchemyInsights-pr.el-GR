# <a name="public-folder-access-from-mobile-devices"></a>Πρόσβαση σε δημόσιους φακέλους από φορητές συσκευές

Δημόσιοι φάκελοι δεν είναι προσβάσιμο από φορητές συσκευές.
  
Εάν θέλετε να έχετε λειτουργικότητα παρόμοια με δημόσιους φακέλους και την πρόσβαση σε αυτό από φορητές συσκευές, ανατρέξτε στο θέμα [πληροφορίες σχετικά με τις ομάδες του Office 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) για μια καλύτερη εναλλακτική λύση.
