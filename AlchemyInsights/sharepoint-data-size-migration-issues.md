# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Ζητήματα κατά τη μετεγκατάσταση δεδομένων στο SharePoint Online

Φαίνεται ότι πραγματοποιείτε μετεγκατάσταση πάνω από 100TB δεδομένων στο SharePoint Online. Ακολουθήστε τα παρακάτω βήματα έτσι σας μπορεί να σας βοηθήσει όσο το δυνατόν συντομότερα. 

1. Επιλέξτε **αίτηση νέας υπηρεσίας**και, στη συνέχεια, **αίτηση για νέα υπηρεσία**. 
2. Αφήστε τον τίτλο και την περιγραφή ως **μετεγκατάστασης του SharePoint πάνω από 100 TB**.
3. Από τη στιγμή που το εισιτήριο έχει υποβληθεί, ενημερώστε το με τις ακόλουθες πληροφορίες: 

    - Εκτιμώμενο μέγεθος της μετεγκατάστασης.
    - Εκτίμηση της όταν θέλετε να ξεκινήσετε και να ολοκληρώσετε τη μετεγκατάσταση.
    - Περιγραφή όπου πραγματοποιείτε μετεγκατάσταση περιεχομένου σας, όπως του SharePoint Server, πλαίσιο, GDrive, κοινόχρηστα αρχεία, κλπ...
