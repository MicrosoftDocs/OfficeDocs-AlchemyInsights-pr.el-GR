# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Ζητήματα κατά τη μετεγκατάσταση δεδομένων στο SharePoint Online

Φαίνεται ότι μπορείτε να μεταναστεύουν πάνω από 100TB των δεδομένων στο SharePoint Online. Ακολουθήστε τα παρακάτω βήματα, ώστε να μπορούμε να σας βοηθήσουμε το συντομότερο δυνατό. 

1. Επιλέξτε **νέα αίτηση υπηρεσίας**και, στη συνέχεια, **νέα αίτηση υπηρεσίας**. 
2. Αφήστε τον τίτλο και την περιγραφή ως **μετεγκατάσταση του SharePoint πάνω από 100TB**.
3. Μόλις υποβληθεί το εισιτήριο, ενημερώστε το με τις ακόλουθες πληροφορίες: 

    - Εκτιμώμενο μέγεθος της μετεγκατάστασης.
    - Μια εκτίμηση του πότε θα θέλατε να ξεκινήσετε και να ολοκληρώσετε τη μετεγκατάσταση.
    - Περιγράψτε από πού πραγματοποιείτε τη μετεγκατάσταση του περιεχομένου σας, όπως το SharePoint Server, το πλαίσιο, το GDrive, τα κοινόχρηστα στοιχεία αρχείων κ. λπ.
