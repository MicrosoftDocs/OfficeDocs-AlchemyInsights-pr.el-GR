# <a name="publishing-infrastructure-with-a-communication-site"></a>Υποδομή δημοσίευσης με τοποθεσία επικοινωνίας


Η ενεργοποίηση της δυνατότητας υποδομής δημοσίευσης του SharePoint Server **δεν υποστηρίζεται** με τοποθεσίες επικοινωνίας. 
  
Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Μετακίνηση από τοποθεσίες δημοσίευσης σε τοποθεσίες επικοινωνίας](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
