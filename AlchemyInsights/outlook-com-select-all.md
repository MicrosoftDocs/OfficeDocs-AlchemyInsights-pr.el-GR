# <a name="how-to-select-all-in-outlookcom"></a>Πώς να "επιλέξετε όλα" σε Outlook.com

Μπορείτε να επιλέξετε όλα μόνο εάν όλα τα μηνύματα είναι ορατά. Αυτό γίνεται για να αποτρέψετε τυχαίες ενέργειες σε μηνύματα που δεν μπορείτε να δείτε.

**Ctrl + κάντε κλικ στην επιλογή** πολλών μεμονωμένων μηνυμάτων.

**ΤΟ SHIFT + κλικ επιλέγει** όλα τα μηνύματα μεταξύ του πρώτου και του τελευταίου μηνύματος που επιλέγετε.
