# <a name="how-to-select-all-in-outlookcom"></a>Πώς να "επιλογή όλων" στο Outlook.com

Μπορείτε να επιλέξετε όλα μόνο εάν όλα τα μηνύματα είναι ορατά. Αυτό γίνεται για να αποτρέψετε τις τυχαίες ενέργειες σε μηνύματα που δεν μπορείτε να δείτε.

Το **συνδυασμό πλήκτρων CTRL + κλικ** επιλέγει πολλαπλά μεμονωμένα μηνύματα.

Το **συνδυασμό πλήκτρων Shift + κλικ** επιλέγει όλα τα μηνύματα μεταξύ του πρώτου και του τελευταίου μηνύματος που επιλέγετε.
