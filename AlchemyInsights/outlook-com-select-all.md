# <a name="how-to-select-all-in-outlookcom"></a>Πώς να "Επιλογή όλων" στο Outlook.com

Όλα μπορείτε να το κάνετε μόνο αν όλα τα μηνύματα που εμφανίζονται. Πρόκειται για να αποτρέψετε την ακούσια ενέργειες σε μηνύματα που δεν μπορείτε να δείτε.

**CTRL + κλικ** επιλέγει πολλά μεμονωμένα μηνύματα.

**SHIFT + κλικ** επιλέγει όλα τα μηνύματα μεταξύ το πρώτο και το τελευταίο μήνυμα που επιλέγετε.
