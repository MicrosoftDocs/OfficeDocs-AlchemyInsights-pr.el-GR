# <a name="seeing-a-false-alert"></a>Εμφάνιση εσφαλμένης ειδοποίησης

- Υποβάλετε ένα αρχείο στη Microsoft για ανάλυση. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Τι να κάνετε με τα ψευδώς θετικά/αρνητικά στο Microsoft Defender Antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Δημιουργήστε μια ένδειξη "Αποδοχή" για να αποτρέψετε μια ψευδή θετική ειδοποίηση. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Δημιουργία δεικτών για αρχεία](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Ρυθμίστε τις παραμέτρους των εξαιρέσεων στη συσκευή για να αποτρέψετε τη σάρωση ενός στοιχείου. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ορισμός εξαίρεσης σε μια μεμονωμένη συσκευή Windows για να αποτρέψετε τη σάρωση ενός στοιχείου](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
