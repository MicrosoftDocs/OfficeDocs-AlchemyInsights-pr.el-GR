# <a name="how-to-make-an-office-365-purchase"></a>Πώς να πραγματοποιήσετε μια αγορά του Office 365

Αγορές στο Office 365 μπορεί να γίνει στο Κέντρο διαχείρισης. Μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**.
  
- Για να αγοράσετε ένα Office 365 συνδρομή, μεταβείτε στη σελίδα [Υπηρεσιών αγοράς](https://go.microsoft.com/fwlink/p/?linkid=868433) ή επιλέξτε **χρέωσης** \> **Αγοράς υπηρεσιών**.
    
- Για να αγοράσετε πρόσθετες άδειες χρήσης, μεταβείτε στη σελίδα " [συνδρομές](https://go.microsoft.com/fwlink/p/?linkid=842054) " ή επιλέξτε **χρέωσης** \> **εγγραφές**. Στη σελίδα " **εγγραφές** ", επιλέξτε την εγγραφή που θέλετε να προσθέσετε άδειες χρήσης για την και, στη συνέχεια, επιλέξτε **Add/Remove άδειες χρήσης**.
    
- Για να μεταβείτε σε ένα διαφορετικό σχέδιο:
    
  - Επιλέξτε ** εναλλαγή σχέδια ** για να προβάλετε τη λίστα των νέων σχεδίων που είναι διαθέσιμες. 
    
  - Επιλέξτε ένα σχέδιο για να δείτε το νέο συνολικό κόστος ανά μήνα. Φροντίστε να διαβάσετε τις πληροφορίες του κλειδιού και, στη συνέχεια, επιλέξτε **Επόμενο** για να κάνετε ανάληψη ελέγχου.
