# <a name="use-microsoft-edge-extensions"></a>Χρήση επεκτάσεων του Microsoft Edge

[Οι επεκτάσεις είναι](https://go.microsoft.com/fwlink/?linkid=2135619) πρόσθετα που μπορείτε να εγκαταστήσετε στον Microsoft Edge για να αποκλείσετε διαφημίσεις, να διαχειριστείτε κωδικούς πρόσβασης, να μεταφράσετε κείμενο και να προσαρμόσετε την εμπειρία περιήγησής σας με πολλούς άλλους τρόπους.
