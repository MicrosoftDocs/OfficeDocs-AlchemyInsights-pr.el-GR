# <a name="delete-a-folder"></a>Διαγραφή φακέλου

1. Στο αριστερό τμήμα του παραθύρου, κάντε δεξιό κλικ στο φάκελο ή υποφάκελο που θέλετε να διαγράψετε και επιλέξτε **Διαγραφή**. 
    
2. Επιλέξτε **OK** για να επιβεβαιώσετε. 
    
Για να μάθετε περισσότερα, ανατρέξτε στο θέμα [Διαγραφή φακέλου στο Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
