# <a name="site-and-content-structure-removed"></a>Η δομή τοποθεσίας και περιεχομένου καταργήθηκε

Ξεκινώντας από τον Οκτώβριο του 2018, η σελίδα "περιεχόμενο και δομή τοποθεσίας" δεν θα υποστηρίζεται πλέον ή θα λαμβάνει ενημερώσεις δυνατοτήτων. Η σύνδεση ρυθμίσεων με τη σελίδα ιστοτόπου. aspx θα καταργηθεί από το SharePoint Online και άμεση πρόσβαση θα περιορίζονται σε διαχειριστές συλλογής τοποθεσιών μέχρι 2019 Μαρτίου, κατά την οποία η σελίδα περιεχομένου και δομής τοποθεσίας θα αποσυρθεί πλήρως. 

Σημείωση: ως εναλλακτικός τρόπος αντιμετώπισης μέχρι να αποσυρθεί πλήρως η δυνατότητα, οι διαχειριστές συλλογών τοποθεσιών μπορούν να περιηγηθούν στο κέντρο διαχείρισης Microsoft 365, να επιλέξουν το πλακίδιο του SharePoint και να προσαρτηθούν ή να επεξεργαστούν τη διεύθυνση URL αντικαθιστώντας το '/Sharetert.PIL ' με το '/Titaanagar.pip. 


Χρησιμοποιήστε το αρχείο μετακίνηση και αντιγραφή για να διαχειριστείτε το περιεχόμενο και τη δομή της τοποθεσίας. Για περισσότερες πληροφορίες, ανατρέξτε στο στοιχείο [Αντιγραφή αρχείων και φακέλων μεταξύ των τοποθεσιών OneDrive και SharePoint](https://support.office.com/article/copy-files-and-folders-between-onedrive-and-sharepoint-sites-67a6323e-7fd4-4254-99a8-35613492a82f). 

Για περισσότερες πληροφορίες, ανατρέξτε στην ενότητα [εργασία με περιεχόμενο και δομή τοποθεσίας](https://support.office.com/article/Work-with-site-content-and-structure-30fcaad9-02b1-4347-8b03-e1ccc5a4c19f).
