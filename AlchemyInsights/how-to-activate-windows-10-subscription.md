# <a name="activating-windows-10-subscriptions"></a>Ενεργοποίηση των συνδρομών των Windows 10

Το Microsoft 365 Business, Microsoft 365 εκπαίδευση, Microsoft 365 και Enterprise παρέχουν ψηφιακές άδειες για τα Windows 10 ως μέρος της συνδρομής. Για να ενεργοποιήσετε τη συνδρομή, η συσκευή σας πρέπει να έχει ήδη ενεργοποιήσει τα Windows 10 Pro (1703 ή νεότερη έκδοση). Για να δείτε μια πλήρη λίστα των απαιτήσεων για την ενεργοποίηση μιας συνδρομής των Windows 10, ανατρέξτε [στις απαιτήσεις ενεργοποίησης συνδρομής των Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
