# <a name="activating-windows-10-subscriptions"></a>Ενεργοποίηση συνδρομών στα Windows 10

Το Microsoft 365 Premium για Επιχειρήσεις, το Microsoft 365 για εκπαιδευτικά ιδρύματα, η Microsoft 365 και το Enterprise παρέχουν ψηφιακές άδειες χρήσης για τα Windows 10 ως μέρος της συνδρομής. Για να ενεργοποιήσετε τη συνδρομή, η συσκευή σας πρέπει να έχει ήδη ενεργοποιημένα τα Windows 10 Pro (1703 ή νεότερη έκδοση). Για να δείτε μια πλήρη λίστα απαιτήσεων για την ενεργοποίηση μιας συνδρομής με Windows 10, [ανατρέξτε στις Απαιτήσεις ενεργοποίησης συνδρομής των Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
