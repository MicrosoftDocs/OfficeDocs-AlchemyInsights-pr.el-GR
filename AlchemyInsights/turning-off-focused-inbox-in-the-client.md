# <a name="turn-off-focused-inbox-in-email-clients"></a>Απενεργοποίηση της "Εισερχόμενα" εστιάζει σε τα προγράμματα-πελάτες ηλεκτρονικού ταχυδρομείου

Για να απενεργοποιήσετε την εστιάζει στο φάκελο "Εισερχόμενα" στο Outlook 2016, Outlook στο web, Outlook.com/Hotmail.com, το Outlook 2016 για Mac ή αλληλογραφίας για τα Windows 10, δείτε [Απενεργοποίηση εστιάζει στο φάκελο Εισερχόμενα](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Για να απενεργοποιήσετε την εστίαση "Εισερχόμενα" για όλους τους χρήστες στον οργανισμό, ανατρέξτε στην ενότητα [Ρύθμιση παραμέτρων εστιάζει στο φάκελο "Εισερχόμενα" για όλους τους χρήστες στον οργανισμό σας](https://support.office.com/article/613a845c-4b71-41de-b331-acdcf5b6625d.aspx).
