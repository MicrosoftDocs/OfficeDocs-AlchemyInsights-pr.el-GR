# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Δεν είναι δυνατή η προσθήκη συνημμένου σε εξωτερική φόρμα

Εάν προσπαθείτε να προσθέσετε ένα συνημμένο σε μια εξωτερική έρευνα του Microsoft Forms, αυτή η δυνατότητα δεν είναι ακόμα διαθέσιμη. Για να παρακολουθήσετε την κατάσταση αυτής της αίτησης δυνατότητας, μεταβείτε σε [αυτήν την καταχώρηση UserVoice](https://go.microsoft.com/fwlink/?linkid=2133069)και προσθέστε την ψήφο σας.
