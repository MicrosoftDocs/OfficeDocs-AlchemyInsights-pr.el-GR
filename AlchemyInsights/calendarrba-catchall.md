# <a name="create-and-manage-room-mailboxes"></a>Δημιουργία και διαχείριση γραμματοκιβωτίων δωματίων

Για να δημιουργήσετε και να τροποποιήσετε γραμματοκιβώτια δωματίων στο κέντρο διαχείρισης του Exchange, ανατρέξτε στο θέμα [Δημιουργία και διαχείριση γραμματοκιβωτίων δωματίων](https://docs.microsoft.com/Exchange/recipients/room-mailboxes).

Για να μάθετε περισσότερα σχετικά με τα γραμματοκιβώτια δωματίου και εξοπλισμού (γνωστά και ως γραμματοκιβώτια πόρων), ανατρέξτε στο θέμα [Γραμματοκιβώτια δωματίων και εξοπλισμού](https://docs.microsoft.com/microsoft-365/admin/manage/room-and-equipment-mailboxes).
