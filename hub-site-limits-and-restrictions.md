# <a name="hub-site-limits-and-restrictions"></a>Όρια τοποθεσίας διανομέα και περιορισμοί


- Συνιστάται να επιλέξετε μια τοποθεσία επικοινωνίας ή μια τοποθεσία ομάδας που χρησιμοποιεί το νέο πρότυπο σύγχρονα. Εάν χρησιμοποιείτε μια τοποθεσία ομάδας classic, την περιήγηση διανομέας θα εμφανίζονται μόνο στη σύγχρονη σελίδων και ρυθμίσεις τοποθεσίας διανομέα θα εμφανίζονται μόνο σε σύγχρονα σελίδες.


- Τοποθεσίες που είναι ήδη συσχετισμένα με άλλο διανομέα **δεν είναι δυνατό** να μετατραπεί σε μια τοποθεσία του διανομέα.


- Μπορείτε να δημιουργήσετε έως 100 διανομέα τοποθεσίες για τον οργανισμό σας. Δεν υπάρχει όριο στον αριθμό των τοποθεσιών που μπορούν να συσχετιστούν με μια τοποθεσία του διανομέα.


- Όταν οι χρήστες συσχετίζουν τις τοποθεσίες τους με μια τοποθεσία του διανομέα, του δεν επηρεάζουν τα δικαιώματα στην τοποθεσία του διανομέα ή τις σχετικές τοποθεσίες. Είναι σημαντικό να βεβαιωθείτε ότι όλοι οι χρήστες που επιτρέπετε να συσχετίσετε τοποθεσίες στο διανομέα, έχετε δικαίωμα να το διανομέα.
