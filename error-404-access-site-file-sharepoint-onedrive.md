# <a name="error-404-when-accessing-files-in-sharepoint-or-onedrive"></a>Σφάλμα 404 κατά την πρόσβαση σε αρχεία του SharePoint ή OneDrive

Παραλήφθηκε <strong>σφάλμα 404 </strong>, όταν οι χρήστες προσπαθούν να αποκτήσετε πρόσβαση σε μια τοποθεσία ή ένα αρχείο στο SharePoint ή OneDrive. Αυτό συχνά προκαλείται από μια τοποθεσία ή αρχείο ή ομάδα γρήγορα μετονομαστεί, μετακινηθεί ή διαγραφεί.
Οι χρήστες θα αντιμετωπίσετε ένα σφάλμα 404 προσπαθεί να αποκτήσει πρόσβαση στη συλλογή τοποθεσιών ρίζας και έχει διαγραφεί.

Για να επιλύσετε το σφάλμα 404 για μια τοποθεσία που έχει μετονομαστεί, μετακινηθεί ή διαγραφεί: για κλασική τοποθεσίες που υπάρχουν στο Κέντρο διαχείρισης κλασικό, δείτε <a href="https://docs.microsoft.com/en-us/sharepoint/restore-deleted-site-collection">Επαναφορά μιας συλλογής τοποθεσιών διαγράφηκε</a>.&nbsp;</li> <li>Για σύγχρονη τοποθεσίες (επικοινωνίας, ομάδα συνδεδεμένος, ή άλλες τοποθεσίες) που υπάρχει στην νέα προεπισκόπηση Κέντρο διαχείρισης, ανατρέξτε στην ενότητα <a href="https://docs.microsoft.com/en-us/sharepoint/view-and-restore-deleted-sites-in-new-admin-center">προβολή και επαναφορά διαγραφή τοποθεσιών στο νέο κέντρο διαχείρισης του SharePoint. 

Για να επιλύσετε το αρχείο (ή άλλο στοιχείο) που έχει μετονομαστεί, μετακινηθεί ή διαγραφεί, επισκεφθείτε την τοποθεσία του SharePoint ή OneDrive και προβάλετε τον Κάδο Ανακύκλωσης από το περιεχόμενο της τοποθεσίας, δείτε [Επαναφορά στοιχείων από τον Κάδο Ανακύκλωσης μιας τοποθεσίας του SharePoint](https://support.office.com/en-us/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be).
