# <a name="sync-a-users-profile-picture"></a>Συγχρονισμός εικόνας προφίλ χρήστη

<p>Μπορείτε να προσθέσετε ή να αλλάξετε τη φωτογραφία προφίλ σας στο Office 365 από το Office.com ή όταν χρησιμοποιείτε εφαρμογές του Office 365 στον υπολογιστή σας ή σε ένα πρόγραμμα περιήγησης στο τηλέφωνό σας. Φωτογραφία σας θα εμφανιστούν οπουδήποτε υπάρχει ένας κύκλος φωτογραφία προφίλ στο Office 365.</p> <p>Για να προσθέσετε τη φωτογραφία προφίλ σας στο Office 365 για επιχειρήσεις</p> <ol> <li>Βρείτε μια φωτογραφία που θέλετε να χρησιμοποιήσετε και να το αποθηκεύσετε στον υπολογιστή σας, OneDrive ή άλλη θέση που μπορείτε να λάβετε για να</li> <li>Συνδεθείτε στο Office 365 με την εργασία σας ή σχολείο λογαριασμού στο office.com.</li> <li>Επιλέξτε τον κύκλο στην επάνω δεξιά γωνία της σελίδας που εμφανίζει τα αρχικά σας ή ένα εικονίδιο ενός ατόμου.</li> <li>Στο το παράθυρο μου λογαριασμών, επιλέξτε τον κύκλο που εμφανίζει τα αρχικά σας ή ένα εικονίδιο ενός ατόμου. Για να αλλάξετε τη φωτογραφία με το αναδυόμενο μενού που εμφανίζεται.</li> </ol> <p>Σημείωση: Εάν έχετε ήδη φορτώσει μια εικόνα μπορεί να θέλετε να δοκιμάσετε κάνοντας κλικ στο πλακίδιο Sharepoint ή OneDrive για να συγχρονίσετε ξανά αυτή την εικόνα.</p> <p>Για περισσότερες πληροφορίες, επισκεφθείτε το <a href="https://support.office.com/en-us/article/information-about-profile-picture-synchronization-in-office-365-20594d76-d054-4af4-a660-401133e3d48a?ui=en-US&amp;rs=en-US&amp;ad=US">πληροφορίες σχετικά με το συγχρονισμό της εικόνας προφίλ στο Office 365</a>.</p>
