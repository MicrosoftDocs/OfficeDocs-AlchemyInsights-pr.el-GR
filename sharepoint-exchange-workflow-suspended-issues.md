# <a name="workflows-in-sharepoint"></a>Ροές εργασίας στο SharePoint

<p>Εάν οι ροές εργασίας του SharePoint δεν στείλετε μηνύματα ηλεκτρονικού ταχυδρομείου, ο οργανισμός σας ενδεχομένως να Παρουσιάστηκε το Exchange Online όρια του αποστολέα.&nbsp;</p> <p>'Η ροή εργασίας έχει ανασταλεί' μήνυμα λάθους μπορεί να προκύψει, εάν έχετε ένα από τα ακόλουθα στοιχεία:</p> <ul> <li>Μπορείτε να έχετε μια ροή εργασίας σε ηλεκτρονική SharePoint που χρησιμοποιεί το SharePoint 2010 ή τύπος πλατφόρμας ροής εργασίας του SharePoint 2013.</li> <li>Η ροή εργασίας έχει ρυθμιστεί για να στείλετε ένα προσαρμοσμένο μήνυμα ηλεκτρονικού ταχυδρομείου σε περισσότερα από 200 χρήστες κάθε φορά, περισσότερες από 10.000 παραλήπτες ανά ημέρα είτε περισσότερα από 30 μηνύματα ανά λεπτό.</li> <li>Κατά την εκτέλεση της ροής εργασίας, το μήνυμα ηλεκτρονικού ταχυδρομείου δεν αποστέλλεται και παρατηρήστε το μήνυμα λάθους, εμφανίζεται <strong>εσωτερική κατάσταση έχει οριστεί σε αναστολής</strong> ή <strong>δεν είναι δυνατή η αποστολή σε παραλήπτη</strong> .</li> </ul> <p>Για περισσότερες πληροφορίες, ανατρέξτε στο ακόλουθο <a href="https://support.office.com/en-us/article/-daily-email-limit-has-exceeded-and-your-workflow-has-been-suspended-or-unable-to-send-to-a-recipient-error-in-a-sharepoint-online-workflow-89d02169-5fa6-4259-affc-73edb6ca9fb6?ui=en-US&amp;rs=en-US&amp;ad=US">άρθρο</a>.</p>
