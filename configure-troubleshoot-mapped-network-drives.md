# <a name="how-to-configure-and-troubleshoot-mapped-network-drives"></a>Τρόπος ρύθμισης παραμέτρων και αντιμετώπισης προβλημάτων αντιστοιχισμένων μονάδων δίσκου δικτύου

Η αντιστοίχιση μιας βιβλιοθήκης ως μονάδας δίσκου δικτύου είναι προσωρινή και υποστηρίζεται μόνο μέσω του Internet Explorer. Αντίθετα, [συγχρονίστε τα αρχεία του SharePoint με το νέο πρόγραμμα-πελάτη συγχρονισμού του OneDrive, το](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88) οποίο παρέχει [αρχεία κατ' απαίτηση](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e). Αποκτήστε πρόσβαση σε όλα τα αρχεία σας στο OneDrive χωρίς να χρησιμοποιείτε τοπικό χώρο αποθήκευσης.

* [Αντιμετώπιση προβλημάτων αντιστοιχισμένων μονάδων δίσκου δικτύου που συνδέονται στο SharePoint Online](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives)
